语义化就是*正确的标签做正确的事*。

# 语义化的好处在于：

- *对于开发团队而言，代码更加容易维护*
- *在css没有加载出来的情况下，也能很好的展示结构*
- *有利于SEO优化*
- *更好地支持各种终端，例如无障碍阅读和有声小说等*

# 常用的语义化标签有：

*header：定义页眉信息*
*nav：导航栏*
*section：页面的组成部分*
*footer：脚注信息*
*aside：侧边栏信息，比如菜单或者广告等*
