# 4、如何创建家族？
首先为何要创建家族呢？就类似写文章，要先创建一个文档，之后才能开始一系列操作。创建家谱后也才能进行后续操作，创建完后，才可以编辑谱系、谱文、谱书、卷轴这些内容。在首页创建完家族后，就会自动跳转到谱系界面。
# 5、如何录入谱系？
在谱系界面怎么录入族员呢？首先我们要知道什么是谱系，谱系就是整个家族的脉络，在记录家族血脉传承方面起到了很重要的作用。通过谱系，我们可以清晰地了解自己的亲戚关系，从而保持家族间的血脉传承性。谱系主要包括两方面的内容，一个是在家族中的位置，另一个是每个人具体的行传内容。

用一个宝塔图的形式，将家族的祖先后代的血统关系联系起来，并编排起来，然后我们点进来就可以看到每个人的详细信息。主要呢，有两种输入方式，第一种就是基础的录入，我们可以看到右边这会有一些图表空格。

那我们根据具体信息，把内容填写进来就好了，比如说姓氏、名字、常用名、别名等等。值得注意的是，有些关键字段必填，比如说名字、常用名、别名、简介等内容。比如曾国藩的名字、谱名，我们填的是曾传豫，然后常用名就填国藩，别名填‘字伯涵’。

然后我们可以填出生和去世的时间，这里填公元纪年或干支纪年都行，系统都会识别。比如曾国藩生于1811年11月26日，点击保存，并修改状态为去世，就会多出去世时间，殁（mo四）于1872年3月12日，然后点击保存。

然后我们可继续输入简介：‘道光进士，曾内内阁学士……’我们把整段内容复制粘贴进来，填写完行传内容会自动保存。那么还有另一种输入方式，是通过一个快捷录入的方式，专业人士可选择快捷录入，操作会更快。

我们继续点击‘添加配偶’，把原配夫人欧阳氏填写进来；如果不是原配，我们可以再添加一个，妾室陈氏，也可以修改配偶关系。然后点击添加儿女，比方说曾纪泽，而在这里可选择父母和孩子的关系，选择真正的父母。

我们继续填写曾纪泽的儿子曾广铨，还有曾国藩的另一个孩子曾纪第，就这样依次把整个家族的人都添加进来就行。那咋处理过继的情况呢？我们选中这个人，中间栏有个过继的操作按钮，点击后有个迁移本人及后代，、过继为嗣子或继子、过继为祧tiao子。

我们可以在这里选择，我们把曾广铨过继到曾纪第名下，勾选他然后点击确定。那么在这就可看到曾广铨在曾纪泽名下显示‘出嗣’，然后在曾纪第名下显示‘承嗣’。假设说是女儿，我们选中这个人修改性别后，就能看到前面的图标从蓝色变粉色。

依次填写完整个家族的世系脉络后，再填写具体每个人的一些行传内容和基本信息，那么就完成了整个谱系，整体呈现一个吊线图的形式。
# 6、如何录入谱文
那我们怎么进行谱文的编辑排版呢？首先，我们得明白谱文是啥，我们可以把家谱里，除了谱系外的所有文章，统称为谱文。谱文是指族谱当中的文字介绍部分，一般会有谱序、像赞、谱例、谱论、源流等等这些内容。

没有编辑谱文的家谱，只有单纯的一个谱系部分的家谱是不完整的。那么怎么编辑呢？谱文编辑界面的左侧有很多谱文类目可供参考，我们可以根据自己的内容填写进来。我们可以修改谱文标题，我现在要写的第一篇序文叫曾氏谱序，点击后能看到中间部分的标题也变了。

填写内容时，如果一页放不下，那可以新建页面，再输入剩下的内容。编辑完谱文后还有很重要的一步，就是点击一下‘保存’，那么就写完了一篇谱文。假设说，我们在序这个类目里面不只有一篇谱文，比方说新序、旧序、客序等等，

那么我们可以点击‘新建谱文’，输入标题，然后选择谱文布局，这里有现代图书的横向左开、古籍的竖向右开和竖向左开。然后选择纸张的版式、和类目名称后保存。另外，我们想要编辑像赞一类的谱文时，需要添加一些图片，上传完图片后勾选需要的图片。

除了谱序、像赞，还有家规族训、碑记、凡例、以及字辈派语。
# 7、如何生成谱书
主要给大家介绍两种谱书，一种是现代精装，一种是仿古线装。现代精装家谱是硬壳封面，非常典雅大气，内页选择的是道林纸，或者光泽度比较好的铜版纸。仿古线装家谱的封面，用的是凌绢四孔线装，中间还有纸钉固定，特点是古朴典雅，里面是宣纸。

单面印刷后再对折装订成一个筒子页。宣纸不容易掉色，而且保存时间较长，具有纸寿千年的说法。那要怎么设置这样的一本谱书呢？首先我们要编写完谱系和谱文的内容，然后点击进入到谱书界面，点击‘新建谱书’，输入谱书名字‘曾国藩家谱’

填写完卷号（‘卷一’）、页眉名称（曾国藩家谱）、郡jun望堂号（富厚堂）、编者、编修日期、起始页码后点击确定，然后就生成一本书。现在第一步要选择封面，选择仿古线装或硬壳精装，然后选择封面是否显示副标题、卷号、堂号、编修日期、以及扉页。

可以选两种扉页，第一种是双龙图案，第二种是四条边都有龙，根据我们具体的情况去选择。而且扉页设置里，还可以勾选是否显示副标题、卷号、堂号、编修日期、主编、责任编辑，确认完后点击确定，就完成了第一步。

第二步是编辑内容，我们点击进去，谱书包含四部分：卷首、正文目录、世系目录、还有附录，卷首一般放置前言、序、谱论、凡例、编修名录等内容，把这些谱文添加进来，卷首的内容就这样添加完了。

接下来，正文目录一般放置源流、碑文、恩荣录、祠堂坟茔ying、字辈等内容，我们继续添加这些谱文，点击确定。之后，是世系的内容。我们填写世系标题、起始族员，可以在右边选择起始族员，后代显示方面，可选择‘全部后代’、‘指定到结束人物’、或者是‘指定结束世系’。

比如说，我这本谱书，只想录入前五世。接下来，我们可以勾选是否显示检索表、世系图、世系表、追溯先祖等等。勾选完后我们继续添加附录，附录一般放置家规族训、名人传记、风俗礼仪、文献杂记、领谱字号、后记等内容。

比如，把像赞、家规字辈等等都添加进来后，点击确定。这样就完成了整本谱书，点击保存，返回第三步‘设置版式’。点击进去，可预览具体版式，牒记式、欧式、苏式、宝塔式等等，比方说，牒记式不用横竖线的连接，单纯用文字表述世代间关系。

每个人的名下都有相关的一些简介，比方说名字号、功名官爵、生辰年月日、葬地、功绩等等。牒记式之后还有欧式，是北宋文学家欧阳修创立的，就是以这么一个图表的形式来记录，五世为一表。第一表是一世到五世，第二表是五世到九世。

这样依此类推，欧阳修家谱的特点就是世代间非常分明，用起来也比较方便。另外，除了欧式还有苏式，苏洵创立的，特点就是世代间直行下垂，世代间没有一个横线连接。就是用一个竖线，用线串联起来的，然后图表的格式也是从右往左排列的。

还有宝塔式，顾名思义，世系人名像宝塔般排列。从一世祖开始，然后世系从上到下排列，形成一个宝塔形状。我们目前有好几种宝塔式，书籍式宝塔世系图结构比较紧密，比较符合现代人的阅读习惯。

还有一种就是卷轴式的宝塔大图，将宝塔式的世系图，做成一个卷轴。一方面摆脱篇幅的一个约制，使得入谱人数和世传的内容的展示空间得到全面拓展；另一方面也便于收纳存放和展示。然后，还有一个现代版式，采用的是从左到右的横版排版。

字体也采用的是简体字，不再用繁体字，这种格式的优点就是信息比较详细，而且比较节约纸张。但缺点就是上下世系联系比较难一目了然。预览后确定想要哪个版式的话，选中它，点击保存返回就行了，第三步就完成了。

第四步是制作谱书，耐心等待一会儿，谱书会生成好。生成完后点击‘阅读谱书’，就能在电脑上预览了。谱序、凡例、目录、碑文、字辈派语、检索表、吊线图，后面是世系录的内容、还有像赞、以及家规族训。

如果预览后没问题，可点击打印校对稿，打印好后宗亲就能校对了。如果选择校对打印服务的话，我们打印好给你寄过去。下载并打印的话就是你自己操作，如果打印出来宗亲们校对完没问题，那我们就点击印刷谱书。

可以选择装订类型，仿古线装和硬壳精装，还有内页材质、印刷套数，设置完后点击‘立即订购’，之后添加收货地址，选择省市区、输入详细地址，输入名字和手机号。设置好地址后我们再点击‘去支付’，按照您的消费习惯，选择一个支付方式。

最终我们会印刷完您的家谱，送到您手上时就是一套完善的纸质家谱了。至此，一套完整的家谱编修流程就是这样子。
# 8、如何生成挂图家谱？
挂图家谱是可以把一个家族所有人都展示在一张纸上的家谱。我们可以把成品家谱挂在祠堂等地方，上面有横批，两边有对联，中间就是家族的宝塔式家谱。

那么如何制作这样的挂图家谱呢？首先，我们要录入完整个家族的世系资料。录入完后，点击‘卷轴’，新建卷轴，选择挂图样式，输入卷轴标题，比方说‘曾国藩家谱’，点击确定。

之后，我们第一步是编辑内容，输入横批的内容，比如‘千秋功德’，输入上联‘祖功宗德流芳远’，点击确定，再输入下联‘子孝孙贤世泽长’，点击确定。标题内容是已经写好了的，那起始族员呢，就是指始祖，这里一般都是默认的。

后代显示的话，我们可选择所有后代还是说指定结束世系。然后返回到第二步，设置卷轴样式，这里有一些通用设置，我们可以输入规格尺寸，比如说我现在想要做一个，宽50cm高70cm的卷轴，选择阅读方向，系统关键字语言，繁简转换，族员姓名显示等等。

分别选择横批和对联的文字设置，包括字体、字号、颜色，还有对齐方式。还有标题位置选择顶部横排、侧边竖排、或者不显示，标题的文字设置和对齐方式。以及其他设置就不一一列举了。

第三步是制作挂图。点击进来，等几分钟就可以了。我们点击‘阅读挂图’，就可以在电脑上预览挂图的模样。然后，我们可以下载挂图，下载后是一个PDF文件。确认无误后，点击印刷挂图，扫码联系我们的专属客服，了解更多详情，关于价格、工艺等等信息。

我们是提供一条龙的家谱卷轴印刷服务的。
# 9、如何生成卷轴家谱
卷轴家谱就是可以把整个家族，所有人都展示在一张纸上，然后这张卷轴家谱可以横着卷起来。要怎么制作这样的一个卷轴呢？点击进入到卷轴页面，如果我们已经编辑完了家族内容，已经把所有族人录入进来了，那么我们点击卷轴，点击新建卷轴。

选择卷轴样式，设置卷轴标题‘曾国藩家谱’，点击确定。在这个地方，第一步‘编辑内容’，这里就是可以设置标题内容、起始族员，从谁开始的，然后怎么显示后代，设置完后点击保存。

第二步设置版式，我们可以在右边看到样式范例。比方说第一种是黄色底纹，第二种是橙色边框，第三种是纯白色的样式。然后是一些通用设置，包括装订规格、简繁设置、姓名设置、世系设置方面。

可修改规格尺寸、阅读方向、繁简转换、在世和过世成员是否显示谱名，过世成员后缀；是否显示世系数等等。设置完后点击保存，返回到第三步‘制作卷轴’。

过了一会儿就能阅读卷轴，是吊线图的样式，也能下载PDF格式的卷轴文件。如果您想要印刷，变成一个可以卷起来的好看的卷轴家谱，就点击‘印刷卷轴’，然后联系专属客服就行。
