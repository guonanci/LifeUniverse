# 其他动物：双线嗜粘液蛞蝓
**俗称鼻涕虫**，看起来就像是没有壳的蜗牛，身体柔软光滑，体表多具有粘液。全身灰白或淡黄褐色，背中央及两侧各有一条由黑色斑点组成的纵带，体前端较宽，后端狭长。触角2对，口内具有角质齿舌，用肺囊呼吸。
昼伏夜出、晴伏雨出，害怕强光和高温。雨后比较常见，走过的地方通常会留下一道亮晶晶的路径。主要以植物的茎叶果实和一些菌类为食。
# 狮纳蛞蝓
表面粗糙如皮革质，黏液很少，肉食性蛞蝓，捕食其他蜗牛。触角2对，口内具有角质齿舌，用肺囊呼吸。

# 灰尖巴蜗牛
蜗牛是一类有柔软的身体和坚硬的甲壳的陆生软体动物。体表光滑具有粘液，头部具有两对触角，大的一对顶端具有眼睛，小的为感觉器官。口内有齿舌，上面具有数万颗牙齿，用于刮取食物。喜欢阴暗湿润的环境，下雨天或者雨后较常见，主要以植物的茎叶、花果为食。

昼伏夜出、晴伏雨出，害怕强光和高温。雨后比较常见，走过的地方通常会留下一道亮晶晶的路径。主要以植物的茎叶果实和一些菌类为食。
# 饰纹姬蛙
小型蛙类，背部通常有2-4个”v”字标，对称着往身体两边生长。多数背部从嘴端一直到肛部上方都有一条中脊线，这条脊线上还长着小疣粒，显得凹凸不平，让线看起来不完整。

体型小但腿部肌肉发达，弹跳能力极佳；嘴巴虽小但叫声响亮，并通过叫声来求偶，狭小的口部也非常适合吃蚂蚁。在夏夜的水边，几乎都能听到它的叫声。
# 北方狭口蛙
合肥常见蛙类，体型中等。体型宽扁，看起来很胖，可能被误以为是蟾蜍。头宽大于头长，嘴部圆润狭长。四肢粗短，皮肤较厚而平滑，背面有少数小疣，呈浅棕色或橄榄棕色。
当遇到危险时他们会奋力吸气，让自己体积变大，使小型捕食者无从下口。别看腿短，爬树、挖洞却是人家的擅长。雄性声音虽不好听，但传的远，传播度高。
# 泽陆蛙
合肥常见蛙类，体型偏小。头部略尖，前肢短小，后肢较粗壮。背部皮肤粗糙，体背面有数行长短不一的褶皱。背面颜色及纹路差异很大，多为灰橄榄色或深灰色，杂有棕黑色斑纹，有的会有一条浅色中线。四肢背面各节有棕色横斑2～4条。

相对于其他蛙类，泽陆蛙可在陆地上活动更长时间。在合肥，远离水域的地区看到的蛙，大多数为此类蛙，昼夜均有可能看见。
# 环口螺
环口螺科种类丰富，贝壳形态多变，盘状或塔状，左旋或右旋，通常坚固而紧凑。厣石灰质或角质，外侧常具有突起。某些种类有呼吸管，当厣闭合时，仍能以呼吸管呼吸空气。
环口螺是成功进入陆地环境生活的一类中腹足目类群。其栖息环境湿润，并富于腐殖质和植物凋落物。植食性，雌雄异体。
# 中华蟾蜍
俗称“癞蛤蟆”，中国分布最广的蟾蜍。耳后腺和皮肤腺均可分泌毒液，毒性作用于心脏和神经系统，足以让人致命，民间说的“五毒”之一，但只有在非常危急时刻才会喷出毒液。
性情温顺，四肢粗短，行动迟缓，多在草丛、石下、土穴中栖息，天黑后出来活动捕食，捕食各种活的小动物，甚至有些小型蛇类都在它的食谱之内。
# 黑斑侧褶zhe蛙
传说中的“青蛙”、“田鸡”，是蛙类中体型相对较大的，广泛分布于我国除台湾岛、海南岛、新疆、西藏之外的各个地区，是我们最为熟悉的蛙类品种。
背绿或后端棕包，有许多黑斑，背侧褶较宽，雄性有一对颈侧外声囊。近年来由于过度捕捞和栖息地的生态环境质量下降，种群数量急剧减少。
# 金线侧褶蛙
简称金线蛙，身体呈青绿色（也有少数个体是黄褐色的），背部中央有一条浅绿色的背中线，身体两侧各有一条粗大隆起的背侧褶（颜色黄褐色、绿色或白色），这就是“金线侧褶”的由来。
虽然个头大，金线蛙的鸣声却并不响亮。雄性的金线蛙在繁殖季发出谨慎又短促的近似“叽咕……”的鸣声吸引雌性。
# 玉带蜻
虫体大小：腹长30mm，翅展70mm；褐色或黑色，翅基部具有黑褐色斑，腹部第三、四节白色，其中雌性白色腹节带有黄色。外表鉴别雄雌虫：雄虫腹部白色，前额红褐色。雌虫腹部黄色，前额黄色。
成虫飞行技术高超，较难捕捉。对自己的领地有占有现象，经常可以看到在水库边或者池塘边有雄虫在保卫自己的领地，一般早上下午没有下雨的时候都会在自己的领地来回的巡查。
# 锥zhui腹蜻
别名粗腰蜻蜓，外形较为特殊的小型种类，腹长约18mm；雄性体色淡蓝，胸部褐色斑纹非常特殊；雌性黄褐色或绿褐色，黑色斑文与雄性相同；该种腹部自中部后缩小成长锥状。由于无近似种类，因此容易识别。此种飞翔能力不强，生活在池塘、沼泽等环境中。
# 碧伟蜓
城市常见的大型蜻蜓，整体翠绿色，大而夸张的复眼在头顶紧密相连。胸部紧密而结实，看不出中后胸的明显分节（称为合胸）。
碧伟蜓的腹部整体呈褐色，雄性腹部基部有显眼的蓝色区域，雌性则有多型：绝大多数雌性的腹部基部是黄绿色的，但也有少部分个体如同雄性一样呈现蓝色。
# 晓褐蜻
晓褐蜻常被称作**紫红蜻蜓**，色彩艳丽，体中小型，腹长25mm。雄性全身及翅脉均为紫红色，额头有蓝黑色金属光泽，胸部具有黑色条状斑纹，翅基部橙红色；雌性体色为橙黄色，胸部及腹部黑色斑纹发达。
成虫出现于5月至11月，生活在海拔2000公尺以下地区的池塘、水库、山沟、沟壑、沼泽或缓流小溪等环境。
# 蓝额疏脉蜻
蓝额疏脉蜻，又称橙斑蜻蜓。腹长27.0-30.0 mm，后翅长29.0-29.5 mm。头顶蓝色，有金属光泽。胸部黑色，侧面黄色，有黑色条纹。翅透明，翅脉黑色，翅痣黄褐色；前后翅基部具不明显的红黄色斑。足黑色，具刺。腹部黑色，大部分腹节具黄斑纹，前部斑纹形成环形宽带。广泛分布于沼泽和野草丛生的池塘等地。 
# 黑丽翅蜻
成蜻7月出现，生活在水草丰富的水库或者森林里有水的地方。这种酷似蝴蝶的小精灵背部闪着蓝色的金属光泽，像是盖着了一层绚丽的铠甲。它们羽翼轻薄，前翅前端1/3为透明状态，其余部分是混杂着蓝绿黑色的金属色。后翅宽大如蝶翅，展开可达70mm，飞行时又似蝴蝶一般扇动翅膀，因此又有“**蝶形蜻蜓**”之称。
# 翠胸黄蟌
**蟌，俗称豆娘。**翠胸黄蟌识别要点是：成虫腹长30-35mm，后翅长20-25mm。复眼绿色，胸部绿色，雄虫腹部橙红色，雌性为褐色至橙红色。5月起可见成虫，活动于各类水体附近。
# 业足扇蟌
在豆娘家族，扇蟌科是长得比较有特色的一类。它们的雄虫中足和后足的胫节像加厚了一层，扩大成了扁平薄片状，看起来像几片树叶。雌性扇蟌就没有这个结构。叶足扇蟌的雄虫的中足和后足是白色膨大的，身体颜色是青灰色，雌性颜色偏黄。
# 杯斑小蟌
体长20毫米，翅展22毫米。腹部长15-18毫米，雄虫胸部布满白粉，腹部黑褐色，末端橙红色。雌虫胸部灰绿色；部分个体近似雄虫，胸部具白粉、黑色斑。老龄雄虫胸部会覆盖粉霜。体形十分细小，较难察觉其出没。生活在水田、沼泽等静水环境内。
# 黑狭扇蟌
相对少见。体长约36mm，雄性胸部背面黑色，腹末白色，足胫节明显双色区别于雌性，雌性与叶足扇蟌雌性相似，但本种体型明显较大。5月起可见成虫，活动于植被较好的水体附近。
# 褐斑异痣蟌cong
褐斑异痣蟌也叫青纹细蟌，常见的中等体型豆娘，体长30mm以内；雄性胸部青绿色（部分老熟雄性为蓝绿色或蓝色）并具黑色条纹；腹部背侧黑色，腹侧黄色，末端具水蓝色斑；雌性部分个体与雄性相同，部分个体体色较淡且无水蓝色斑。水田池塘边的常见种类，以小虫，蜘蛛，同类为食。
# 长叶异痣蟌
成虫腹长22-25mm，后翅长18-22mm。体小至中型。雄虫下唇白色，额黑色。复眼上部分为黑色，下部分为天蓝色。雌虫体色与雄虫相差较大，全身以淡绿色为主，腹端没有斑点。刚羽化的个体全身橙红色，随着成熟度的提高，渐渐由橙红色变为淡绿色。

栖息于挺水植物生长茂盛的池塘、湖泊、水渠附近。成虫发生期6-9月。
# 云斑白条天牛
体长32-65毫米，是天牛中较大的一种。体黑褐至黑色，密被灰白色至灰褐色绒毛。雄虫触角超过体长1/3，雌虫略长于体，每节下沿都有许多细齿。前胸背板中央有一对肾形白色或浅黄色毛斑，小盾片被白毛。鞘翅上具不规则的白色或浅黄色绒毛组成的云片状斑纹，一般列成2～3纵行，以外面一行数量居多，并延至翅端部。身体两侧由复眼后方至腹部末节有一条由白色绒毛组成的纵带。
# 华星天牛
体长21-37mm，体黑色，具光泽，触角自第3节起基部均具蓝白色毛环，足披蓝白色绒毛，每鞘翅具数个白色毛斑。前胸两侧具刺凸，鞘翅基部具若干小瘤凸，寄主为多种阔叶树，成虫于6-7月发生。
# 桑黄星天牛
体长15-30mm，黑色，通体披灰绿色绒毛，前胸背板两侧有一对纵向黄毛斑，鞘翅缀有若干黄色小斑。雄性触角长，约为体长2.5倍，雌性稍短，前胸侧缘瘤凸小。
寄主为构树、桑树、无花果等，幼虫蛀食韧皮部。成虫发生于5-10月，趋光。
# 裸角天牛
体长30-55mm，褐色，体背披黄色短绒毛，前胸近侧缘黄色稍浓密，雄性触角及足粗壮，触角超过体长，第1-5节粗糙，雌性触角及足纤细，腹末具细长产卵管。寄主众多，幼虫主要钻蛀朽木。成虫于春末至夏季发生，具趋光性。
# 姬蠊lian
一类体型较小的蟑螂，身体较宽，且为扁平状。触角丝状，较长。头部较小，前胸背板多会有深色斑纹。足较为细长，步行速度很快。有两对发达的翅膀，但不擅飞行，仅在遇到危险时进行短距离飞行。
分布广泛，在树林下杂草、灌木丛、垃圾堆、枯枝落叶层中经常可以发现，白天隐匿，多在夜晚活动。食性非常繁杂且生命力极强，合肥家居中常见的蟑螂多为此类中的德国小蠊。
# 黑胸大蠊
身体黑褐色，有光泽，体长约25-30mm，翅脉明显，前胸背板无斑纹。食性广泛，喜欢甜食。主要生存于下水道、厕所、菜市场等阴暗潮湿的环境。
# 中华大刀螳
成虫体长5-10cm，多以绿色和褐色为主。脑袋倒三角形，胸部短小，腹部修长。两对翅膀，外侧的革质前翅保护内侧的膜质后翅。胸口的前肢特化为两把“大刀”，下方具有许多锯齿，平时常折叠起来，攻击或者警戒时便展开大刀示威。

平时以其他昆虫或蜘蛛为食，保护色非常强，常常模拟成周围植物，等待猎物靠近，用大刀攻击猎物后直接啃咬进食。
# 广斧螳
顾名思义，广斧螳有着宽阔的腹部，尤其是要孕育后代的雌性，有着比雄性更大的体格和腹部。也叫广腹螳、宽腹螳螂，是国内最常见的大型螳螂。

头部三角形，身体绿色或褐色，长脖子连接着宽阔的腹部，腹部上面覆盖着革质的前翅和膜质的后翅，前翅上通常有两个明显的白斑。捕捉足可以灵活折叠，基部的腿节上长有一些米黄色的突起。
# 棕静螳
又名棕污斑螳。体型中等，大约长4.5～6厘米，身体大多为棕色，较为细瘦。性格与其它螳螂相比稍显胆小，主要栖息于近地面的草丛中。
肉食性，生性安静，可长期守候猎物，常在靠近地面的草丛、草地上伺机捕食小型昆虫，如蚱蜢、蟋蟀、蝇类等。
# 竹节虫
身材纤细修长，颜色多为绿色或褐色，形似树枝或竹竿。头部比较短，具有两根细长的触角，胸腹部非常修长，具有同样修长的6只步足，多具有两对翅膀，外侧的革质前翅保护内侧的膜质后翅。
竹节虫主要以树叶为食，具有极强的保护色，能够模拟树枝或者竹竿。虽然有6条大长腿，但是行动缓慢，多夜间活动，遇到危险常先后退，然后从植物上掉落远离敌人。
# 红蜻
常见蜻蜓之一，腹长约3厘米，后翅长约4厘米。成熟雄蜻蜓体色为朱红色，翅膀透明；雌虫则为黄色，分布于中低海拔地区。主要出现在4-12月份，常在水域附近的草丛附近活动。
这种蜻蜓雌性成虫和未成熟的雄性都是黄色的，但是雄性在成熟过程中就会慢慢“变色”，从黄色变成了红色。
# 黄蜻
体长32-40毫米，身体赤黄至红色；雌雄体征近似，不易区分，但一些老熟的雄性黄蜻的身体会稍带红色。迁徙能力出色，是**世上分布最广的一种蜻蜓**。
**白天集群的黄蜻在空中捕食飞虫，有时还会和家燕配合，上下夹击剿灭成团飞舞的蚊虫**。黄昏则寻找低矮的植物枝条停落，谨慎地选择一根树枝吊挂；一旦落实，逐渐入睡的黄蜻便对周围的扰动愈加迟钝。
# 碧蛾蜡蝉（合肥常见夜观生物）
体长约7mm，翅展约21mm，体浅绿色，前胸背板具两条浅棕色纵条带，中胸背板具两对浅棕色纵条带；前翅外缘平截，外缘后1/3处、前缘1/3处、外缘及后缘边缘浅褐色。寄主广泛，常见的寄主为桑、柑橘、柿及紫叶李等。
# 螳蝎（xie一）蝽
俗称**水螳螂**。体长43mm左右；体色为黄褐色，体及各足均细长；镰刀状的捕捉足十分发达、灵活，外形酷似螳螂；腹部末端带有细长的呼吸管。
多在静水水域的水草间觅食。主要以守株待兔的方式捕捉小鱼、小虾、蝌蚪、**孑（jie二）孓（jue二）**等水中小动物，再以刺吸式口器注入消化液溶解猎物内脏，吸食猎物的体液。


螳蝎蝽（学名：*Ranatra chinensis*），又称中华螳蝎蝽、水螳螂、螳蛉蝽，是一种半翅目蝎蝽科的水生昆虫。以下是综合多篇资料的详细介绍：

### 🦗 **一、形态特征**
1. **外形酷似螳螂**
   - **体型细长**：成虫体长40–45毫米，体宽仅2–3毫米，形如“水中的竹竿”或“枯树枝”。
   - **镰刀状前足**：前足特化为镰刀状的捕捉足，尖锐有力，用于伏击猎物。
   - **呼吸管**：腹部末端有细长的呼吸管（长约20毫米），可伸出水面直接呼吸空气。
   - **体色**：通体黄褐色，若虫初蜕皮时呈红色，24小时内渐变为黑褐色。

2. **运动能力**
   - **水面弹跳**：后足肌肉发达，小腿中空，身体轻盈，可在水面快速弹跳移动，被称为“天然快艇”。
   - **飞行能力**：成虫具翅，可短距离飞行，飞行时露出鲜红色后背（故俗称“红娘华”）。

---

### 🌿 **二、生态习性**
1. **栖息环境**
   - **静水水域**：主要栖息于池塘、湖泊、稻田等水流缓慢或静止的水域，藏身水草丛或底层泥沙中。
   - **耐污能力**：因呼吸管可直接呼吸空气，可在低氧死水中生存，但水生污染导致其数量减少。

2. **捕食行为**
   - **守株待兔式猎食**：潜伏水草中，拟态枯叶，以镰刀足瞬间夹住游经的小鱼、小虾、蝌蚪、孑孓（蚊幼虫）等猎物。
   - **刺吸式进食**：用口器注入消化液溶解猎物内脏，吸食体液。

3. **生命周期（不完全变态）**
   | **阶段** | **特点**                                                                 | **时长**         |
   |----------|--------------------------------------------------------------------------|------------------|
   | **卵**   | 米白色，直径4–6毫米，具8条白色须状构造；产在水生植物茎秆或泥沼中（水面2–3厘米高处）。 | 约7–10天孵化     |
   | **若虫** | 共5–6龄，初龄红色→渐变为黑褐色；捕食水蚤、蚊幼，脱皮6次后羽化。           | 若虫期约1–2个月 |
   | **成虫** | 3月出蛰，5月交配产卵；寿命约1年，冬季潜入底层石缝或泥土越冬。              | 成虫存活6–8个月  |

---

### ⚖️ **三、生态作用与经济影响**
1. **益处**
   - **蚊虫控制**：大量捕食孑孓，抑制蚊虫繁殖，是天然的“灭蚊能手”。
   - **生态平衡**：作为水生食物链的消费者，调控小型水生动物种群数量。

2. **危害**
   - **渔业威胁**：捕食鱼苗、虾苗和蝌蚪，对淡水养殖业造成损失。
   - **种内竞争**：食物短缺时自相残杀，吞食同类若虫或卵。

---

### 🗺️ **四、分布与保护**
1. **地理范围**
   - **中国**：除西藏及少数省份外，全国均有分布，常见于辽宁、江苏、浙江、湖北、湖南等地。
   - **全球**：东亚（朝鲜、日本、缅甸）、部分欧洲洞穴及澳大利亚流动水域。

2. **生存现状**
   - **数量下降**：因水体污染（农药、富营养化）及栖息地破坏，种群数量锐减。
   - **保护建议**：保护净水生态系统，减少农业污染物排放，维持水草多样性。

---

### 📜 **五、文化趣闻**
- **古诗中的形象**：宋代杨万里《水螳螂》以“前怒两臂秋竹竿，后拖一腹春渔船”描述其外形，“柳上螳螂工捕蝉，水上螳螂工捕鳣”点明其食性。
- **民间俗称**：北方称“酱油油”“卖糖的”，因其静伏时似糖渍树枝；飞行时红背显露得名“红娘华”。

---

### 💎 **总结**
螳蝎蝽是水生生态系统的“双刃剑”——既是蚊虫克星，又是养殖业潜在威胁。其镰刀足伏击策略与呼吸管适应性，展现了水生昆虫的演化智慧。保护其栖息环境，可最大化发挥其生态价值，维系水域生物多样性平衡。



孑孓（jié jué）是蚊子的幼虫阶段，由蚊卵在水中孵化而成，属于双翅目蚊科昆虫。以下是关于孑孓的全面解析：

---

### 🦟 **一、名称与定义**
- **别称**：俗称“跟头虫”“方学虫”，因游泳时身体一屈一伸似翻跟头得名。
- **词源**：
  “孑孓”为象形字，源于《说文解字》——“孑”意为缺少右臂，“孓”意为缺少左臂，形容幼虫无翅无爪的形态。

---

### 🔬 **二、形态特征**
1. **外形**：
   - 身体细长，深褐色，胸部宽于头部和腹部。
   - 头部有咀嚼式口器，两侧具发达口刷（用于滤食）。
2. **呼吸结构**（不同蚊种差异）：
   | **蚊种** | **呼吸方式**                     | **静态姿态**             |
   |----------|----------------------------------|--------------------------|
   | **库蚊** | 尾端具细长呼吸管                 | 头下垂，与水面成角度 |
   | **伊蚊** | 尾端具短粗呼吸管                 | 同库蚊           |
   | **按蚊** | 无呼吸管，靠腹节气门呼吸         | 身体与水面平行   |

---

### 🌊 **三、生活习性**
1. **栖息环境**：
   - 静水或缓流水域（稻田、池塘、积水容器、污水沟等），夏季高发。
   - 耐低氧环境，可直接通过呼吸管/气门摄取空气。
2. **食性与行为**：
   - **食物**：细菌、单细胞藻类、有机颗粒，部分种类捕食其他孑孓。
   - **活动**：垂直上下游动，遇惊扰迅速下潜。

---

### 🔄 **四、生命周期**
1. **发育阶段**：
   **卵 → 孑孓（幼虫） → 蛹 → 成虫**
   - **卵期**：雌蚊产卵于水面，1-2天孵化（环境适宜时仅数小时）。
   - **幼虫期**：孑孓经**4次蜕皮**（约5-7天），体长可达5毫米。
   - **蛹期**：2-3天，不进食，羽化前浮至水面。
2. **繁殖力**：
   - 雌蚊一生产卵1000–3000枚，10–12天完成一代，年繁殖7–8代。

---

### ⚖️ **五、生态作用与危害**
1. **生态价值**：
   - 净化水体（摄食细菌和藻类），作为鱼类（如柳条鱼、斗鱼）的重要饵料。
2. **健康威胁**：
   - 羽化成蚊后传播 **疟疾、登革热、乙型脑炎、黄热病** 等疾病。
   - 伊蚊（如白纹伊蚊）是登革热主要媒介，常见于家庭积水容器。

---

### 🛡️ **六、防治方法**
#### **物理防治**
- **清除孳生地**：翻盆倒罐、清理积水容器，水生植物每5–7天换水洗根。
- **阻断呼吸**：水面滴煤油或植物油（形成油膜使其窒息）。

#### **生物防治**
- **引入天敌**：放养食蚊鱼（1条/天可食2000只孑孓）或蜻蜓幼虫。

#### **化学防治**
- **安全用药**：使用微生物制剂（如苏云金杆菌）或低毒杀虫剂处理水体。

> 💡 **家庭小贴士**：肥皂水碱性环境可抑制孑孓生长，定期更换有效。

---

### 💎 **总结**
孑孓作为蚊子幼虫，是疾病传播链的起点。其存在提示周边有积水环境，需及时清理。通过“清积水、养天敌、控油膜”三管齐下，可有效阻断蚊虫滋生，减少疾病风险。若家中发现孑孓，优先选用物理或生物防治法，安全环保。
# 角蝉
体小型至中型的蝉类，形状奇特，颜色各异，少数种类还具有艳丽的色彩。前胸背板发达，会出现向前或向后延伸形成突起的角状结构，有的品种突起直至腹部，形态多变。

成虫及若虫均停留在枝条或叶背面，吸食植物的汁液。具有很强的弹跳能力，是其逃生的绝技。若虫身上常长有刺，会分泌蜜露，所以经常会吸引蚂蚁与其共生。

小型至中型的蝉类，体长约14mm；头及前胸背板蓝黑色，带有光泽；小盾片及前翅基部血红色；前翅端部黑色，中间黄白色，有六个大型斑点排成两列。

常聚集在灌木上。若虫因常隐藏在自身分泌的泡沫团中而得名，既可以保持湿润，也可以躲避天敌。
# 黑尾大叶蝉
成虫体长12-13.5毫米，身体呈黄色，并常有变异。头部的两颊较为宽大，触角短小像刚毛的样子。头部、前胸背板及小盾片深黄色。

多生活在植株上，取食植物的叶子。雄虫的前翅端部黑色，当翅覆于体背时，黑色部分在尾端，因此得名。
# 斑衣蜡蝉
不完全变态，不同龄期体色变化很大。小龄若虫体黑色，上面具有许多小白点。大龄若虫身体通红，体背有黑色和白色斑纹。成虫后翅基部红色，飞翔时可见。翅膀颜色偏蓝为雄性，翅膀颜色偏米色为雌性。

成虫、若虫均会跳跃，成虫飞行能力弱。在多种植物上取食活动，吸食植物汁液，最喜臭椿。俗称“花姑娘”、“花大姐”等。
# 日本纺织娘

头至翅末约25mm，头胸翅背侧具褐色纵纹，纵纹两侧具黑色纵纹，该黑纹在头部过复眼，后足股节端部和胫节基部具黑斑。
栖息于草丛，鸣声为连续的“啧，啧，啧，啧”声。

# 蟪蛄gu
体长约2.5cm，是一种比较小型的蝉,绿色的和黄褐色的蟪蛄在南方较常见。分布广泛，5-8月鸣，从早到晚，鸣声作“哧——哧”,叫声不如黑蚱蝉般等大型蝉的大声，比较脆弱。成虫一般喜欢栖息在树干上，一边用中空的管状物插入树枝来吸吮汁液，一边鸣叫。
# 蒙古寒蝉
体长28~35mm，前翅长约40mm，体绿色，具灰褐色花纹，腹部具有白蜡粉；翅透明，雄性腹瓣极发达，达第5腹节后缘；雌虫腹末产卵鞘长。5月下旬至秋季10月，可见成虫。鸣声为“知了，知了”的双音节重复。
# 黑蚱蝉
体长36-40mm，前翅长50mm，体黑色，新鲜羽化个体体表具金黄色短毛；中胸背板宽大，具“X”形隆起。发声期从6月持续到8月，7月为发声高峰。鸣声极其洪亮且连续。
# 肥螋sou
体长14-23mm，体型较粗壮，体黑色，足浅色，前后翅均退化，触角18-20节。活动于地表，多见于石块下，捕食性。
# 茶翅蝽
茶翅蝽，俗称“**臭板虫”“梨蝽象**”，属半翅目蝽科茶翅蝽属。体长10-16mm，体茶褐色至黄褐色，具黑色斑点，前胸背板具4个黄褐色横列斑，小盾片基部具5个黄褐色横列斑。
茶翅蝽寄主范围广泛，主要有苹果、梨、桃、樱桃、杏、海棠、山楂、葡萄等果树，大豆、菜豆、和甜菜等作物，以及榆树、梧桐、泡桐、蔷薇科乔木、和枸杞等树木。
# 大鳖土蝽
又名大田鳖、鳖蝽。体长12-20mm；体黑色，刻点显著，具光泽。触角褐，第4、5节色较浅。前胸背板侧缘缺刚毛，前翅革片密布细小刻点。土栖，取食较大植物种子，如无患子。
# 短翅迅足长蝽
体长5.5-7.5mm；体褐色，具棕色纹。触角第4节基部及端部白色；前胸背板具“M”纹；前翅革片具白色三角斑；足褐色，腿节及胫节基部黄色。夜间常在地面或墙面上爬行，具趋光性。
# 红脊长蝽
体长约10mm，橘红色，并具黑色大斑，被金黄色短毛。头黑、光滑，触角黑色。前胸背板具1对近三角形黑斑。小盾片黑色，革片红色，中部具不规则的大黑斑，足黑色。

成虫和若虫具群聚性，以萝藦、刺槐、花椒、洋槐、小麦等植物为食。
# 瓦同缘蝽
体长约17mm，体狭长，两侧缘几乎平行，鲜黄绿色。头、前胸背板和前翅的绝大部分褐色。触角4节，第一至第三节紫褐色，第四节最短，基半部黄绿或黄色，端半部褐色或黑褐色。前胸背侧角呈三角形，稍向上翘。前翅前缘有1条黄绿色的带纹，此纹在革片近端1/3处向内扩展成半圆形斑。
# 环斑猛猎蝽
体长16-18mm，体黑色光亮，具黄色或暗黄花斑，体被淡色毛。各足具淡色环斑。
本种色泽及色斑有变异，尤其是前胸背板色泽变化大，由淡黄到黑色；若干个体前胸背板后叶为暗黄色；头部及各足淡色斑变异不甚明显。
# 水黾（min三）
俗称**水蜘蛛**。体色黑褐色，体长约22mm。身体细长，非常轻盈；前脚短，可以用来捕捉猎物；中脚和后脚很细长，长着具有油质的细毛，具有防水作用。
多在静水水域的水草间觅食。主要以守株待兔的方式捕捉小鱼、小虾、蝌蚪、孑孓（蚊幼虫）等水中小动物，再以刺吸式口器吸食猎物的体液。

栖息于静水面或溪流缓流水面上，以落入水中的小虫体液、死鱼体、或昆虫为食。
# 草蛉ling
身体多为绿色，身体柔软，复眼有金属光泽。口器为咀嚼式，上颚较为发达，触角细长呈线状，前翅呈现绿色有明显的翅脉，宽大且透明，后翅较窄。
属于完全变态昆虫，草蛉的卵上有细长的丝柄，很是奇特。幼虫称为“蚜狮”，以蚜虫为食，有的种类会把吸食后的蚜虫空壳背在身上作伪装。成虫的飞行能力不强，多在夜晚活动，经常可以在植物的茎叶上发现它们。
# 暗黑鳃sai金龟
成虫长椭圆形，体长17—22mm。初羽化成虫为红棕色，以后逐渐变为红褐色或黑色，体被淡蓝灰色粉状闪光薄层，腹部闪光更显著。触角10节，红褐色。前胸背板侧缘中央呈锐角状外突，刻点大而深，前缘密生黄褐色毛。腹部圆筒形，腹面微有光泽，尾节光泽性强。雄虫臀板后端浑圆，雌虫则尖削。
# 双叉犀金龟
俗称**独角仙**，雄虫体长45-90毫米，雌性30-50毫米。体色红褐色至黑褐色，被茸毛。头较小，唇基前缘侧端具齿突。小盾片三角形，具中纵沟。
雌雄异型。雄虫头部具1粗壮角突端部分叉，前胸背板中部具1端部分叉的角突。雌虫头部粗糙无角突，头顶横列3个小丘突。
# 扁锹
体长雄性大多在27-72mm，体色黑褐色，具光泽，体型稍扁，大型雄虫大颚发达，具齿状排列，小型则无，雌虫体型较小，翅鞘有光泽，头部具凹凸的刻点。
成虫常集体出现吸食树液或熟透的果实，具趋光性，夜晚或清晨于路灯下容易发现，有时会见两只雄虫在树上互斗，将对方挑落地面，但却很少造成严重伤亡。
# 疑步甲
体长27-48mm，颜色多变，红绿相间或略带褐色，腹面及足黑色；头部狭长；前胸宽阔，密布刻点；鞘翅愈合,呈卵圆形,具6行大瘤突,后翅退化。
4月末至10月出没于林下落叶层或草地，捕食蜗牛等。受惊时尾部会喷射酸臭防御液。
# 屁步甲
体长10-19mm，体宽5-7.5mm，头、前胸背板棕黄色；头顶有心形黑斑；小盾片和鞘翅黑色，各鞘翅肩部和中部有1黄色斑；体腹面及腿节端部黑色，足的其余部分棕黄色。
常见于潮湿的石块及林下，杂食性。受惊和捕食时会有肛门放出高温防御液，能烫伤皮肤。
# 黄足黑守瓜
体长5-6mm的叶甲，鞘翅黑褐色，其余部分棕黄色。前胸背板横宽，中部之前侧缘弧形，之后平行，盘区中部具1浅沟。
初春至秋末可见成虫。成虫访瓜类花花朵，取食花粉、花朵及叶片。幼虫钻蛀瓜类作物地下部分。
# 食蚜蝇
体色较为鲜艳，有很多鲜亮的环节，外形很像蜜蜂或者胡蜂，但是身体较为光滑。前翅明显，后翅特化成平衡棒，足也较蜜蜂更细。
由于后翅特化成为平衡棒，其飞行技巧较高，可以在空中悬停或突然起停。常在花丛中活动，采食花粉、花蜜、植物汁液。经常会在蚜虫群体中产卵，幼虫出生后可以更方便的捕食蚜虫等等其他小昆虫。
# 日本弓背蚁
大型工蚁，体长12mm，中小型工蚁体长10mm，蚁后体长1.7cm左右；头大，近三角形，上颚粗壮；前、中胸背板较平；并胸腹节急剧侧扁；头、并腹胸及结节具细密网状刻纹，有一定光泽；后腹部刻点更细密。体黑色。
由于是在日本首先发现并命名，所以称为“日本弓背蚁”。在中国大部分地区广泛分布。
# 燕山蛩（qiong二）
体长约12厘米的倍足纲动物，全身由50多对圆形体节组成，身体多为黑色，体节中间为黄色（或橙色），触角很短，身体第2-4体节每节具有一对步足，第5体节后每节具有2对步足。
喜欢阴暗潮湿的环境，主要以落叶等腐殖质为食，是生态系统中重要的分解者。擅长爬树，虽然有很多脚但是走路速度非常缓慢，受到刺激后，会散发出特殊的臭气驱逐敌人。
# 多疣壁虎
体较扁平，尾与身体长度基本相当。背面具深褐色横斑，色斑差异较大，颜色呈灰褐色，常与墙壁融为一体。
夜间活动，喜欢生活在石缝、屋檐或干燥的管道中。善爬墙，捕食小型昆虫。遇到危险时，会断尾自保，而断的尾巴还会重新长出来。
# 宁波滑蜥
体型小而细长，尾长是头体长的1-1.4倍。躯体古铜色，散布零星的黑褐色点斑或细纹，体侧各有一黑色纵纹，鳞片光滑。
主要以小型无脊椎动物为食。合肥地区并不常见，主要栖息在丘陵地带的优质森林中，数量正在逐渐变少。
# 少棘蜈蚣
**少棘蜈蚣别名金头蜈蚣、红头蜈蚣，身体扁平的唇足纲动物**。每个体节一对足，一共21对足；头部红色、步足黄色，最末端的步足一般是赤褐色；虽没有毒牙，但身体最前端有一对特化且红黑分明的短足称作颚肢，锋利且与毒液腺相连。
常在夜间出没于阴暗潮湿的地方，白天则潜藏在石头缝中休息。一般以昆虫、蜘蛛等动物为食，颚足可注射麻痹猎物的“神经毒液”。
# 蝎蝽
**酷似蝎子**。体长40mm左右；体色为棕色，身体长筒形，**前足为镰刀状的捕捉足，十分发达与灵活**。其他各足均细长，**腹部末端带有细长的呼吸管**。

**生活在静水水域，不善游泳，多在水草间觅食，主要捕捉小鱼、小虾及其他水生昆虫**。

# 盲蛛
**具有8条细长腿的一类蛛形纲节肢动物，形似蜘蛛但不是蜘蛛。头胸部和腹部几乎融为一体，非常小。喜欢生活在潮湿的地方，杂食。蜘蛛只能吸流食，而盲蛛则是一口一口吞下食物**。

**遇到危险时，盲蛛可以主动断腿，断下的腿像可以自行跳动，吸引敌人的注意力，盲蛛则自行逃走，但盲蛛断腿后不能再生**。

# 园蛛
小型至大型蜘蛛（2-60毫米），种类繁多，颜色、体型差异很大，甚至同种都会出现较大的体色差异。8个眼睛分成两排，中部的眼睛较大。大部分园蛛的雄蛛明显小于雌蛛。
园蛛是结网蜘蛛的典型代表，也是大众普遍理解的蜘蛛类型。在树林、灌丛、墙角等地均易发现它们的身影。
# 络新妇
络新妇多属于超大型蜘蛛（12-50毫米），腹部比较修长，但这些体型大的都是雌蛛，雄蛛体型很小，只有2-8毫米。有些络新妇的头胸部长得很像人脸，所以又被称为“**人面蜘蛛**”。

雌蛛结非常大的圆网，有的可以分成3层，看上去像一个立体的建筑。雄蛛不结网，通常寄生在雌蛛的网上，且往往一个雌蛛的网上寄生着不止一个雄蛛。
# 猫蛛
中型蜘蛛（5-23毫米），身体颜色多变，暗淡的棕黑色和鲜艳的黄绿色皆有。8个眼睛以2-2-2-2的方式排列，但间距不同，8个眼睛形成一个圆环的形状。猫蛛的腿比较细长，通常具有黑色的刺。
游猎型日行性蜘蛛，主要在植物上生活，在树叶、树枝等地狩猎。视力较好，反应迅速，跳跃能力佳。
# 王锦蛇
俗称菜花蛇、大王蛇，大型无毒蛇，体型粗壮，成体全长2米以上。额头上多有形如“大王”二字的斑纹，头部颜色与颈部区分明显。成体背面颜色较为多变，多见黄色、黄绿色、橄榄绿色等。
在我国广泛分布，捕食小型哺乳动物、鸟类、鸟蛋、蜥蜴、甚至蛇类。遇到危险时会分泌浓烈臭液保护自己。但由于其体型大、行动迟缓且无毒，所以几乎是国内食用市场最常见、消费量最大的蛇类，受到野外捕捉的威胁明显。
# 赤练蛇
俗称火赤链、红斑蛇，成体全长0.8-1.3米。背面黑色，伴有等距排列的红色窄横斑，体侧有红黑相杂的斑纹。
广泛分布，多傍晚出没于水源地附近。食性极广，捕食鱼类、蛙类、蜥蜴、鸟类、小型哺乳动物、甚至蛇类。全身会有腐鱼的腥臭味，不会主动攻击人，但性情较为凶猛，有一定毒性，如遇危险会反击。
# 乌梢蛇
俗称过山刀、乌风蛇，大型无毒蛇，成体全长2米左右，头部椭圆形，眼睛较大。幼年时背面黄绿色，身体两侧各有两条黑色纵线由颈后一直延伸到尾末端。随着年龄的增长，体色愈来愈暗，转为黄褐色或灰褐色，甚至纯黑色。身体前部黑色纵纹可见，后面纵线模糊不清甚至完全消失。
分布广泛，常出没于水源地附近，以蛙类和小型哺乳动物为食。行动迅速，反应敏捷，善于逃跑，性较温顺，快速移动或游泳时头部常常昂起。
# 短尾蝮
俗称土公蛇、七寸子、草上飞，中等大小管牙类毒蛇，身体短粗，成体全长0.5-0.7米。头呈三角形，眼后有一道宽大的黑褐色眉纹，在其上缘有一道白色细纹。背面颜色与泥土融为一体。尾巴很短，黄色或黄褐色。
分布广泛，多夜晚活动，常埋伏在水边，捕食蛙类、鱼类，有可能躲藏在树下休息。虽然该类蛇毒性大，但并不主动攻击人，通常发现它们时，它们会保持不动，继续伪装。只要不随意捕捉，通常没有危险。
# 盗蛛
游猎型夜行性蜘蛛，体型一般较大，体色较深，以褐色、灰褐色、黑褐色为主，8个眼睛多为3列，以4-2-2的方式排列。
常见的盗蛛以狡蛛类为主，狡蛛喜欢在水边活动，甚至有很多可以在水面上行走以及潜水，它们通常在水边长时间蹲守捕食，有的狡蛛可以捕食鱼类甚至蛙类。在繁殖季节，雌性盗蛛常常会把卵囊抱在自己的腹部。
# 蟹蛛
小型或中型蜘蛛（2-23毫米），颜色多样，前两对足较长，威吓时会举起，就像螃蟹举起大钳子一样。8个眼睛以4-4的方式排列。
游猎型蜘蛛，行动时也类似于螃蟹横向行走，行动不是很敏捷，但多数蟹蛛都具有很好的伪装，比如伪装成鸟粪、花朵、蚂蚁等，它们的伪装常常能够骗过猎物，所以即便它们行动不快，但捕食的效率还是很高的。
# 漏斗蛛
漏斗蛛因其所结的漏斗状网而得名，它的网由比较平坦的网面和后端的网管构成。通常在灌丛、洞穴、落叶层以及各种缝隙间结网，平时藏匿于网管中等待猎物，感到来自外界的威胁时也会经网管后端开口逃生。
漏斗蛛体色一般较暗，许多种类的腹部背面有数量不等的“人”字形斑纹。行动迅速，但受限于生活环境和结网的习性。
# 福寿螺
水生螺类，入侵物种。原产南美，最初是作为餐桌上的美食引入中国的，但是因为口感不佳被抛弃到野外，然后凭借自己食性杂、繁殖快等特点很快就泛滥成灾。一般夜晚活动，喜欢啃食植物的茎叶，在稻田和池塘里都能很好地适应生存。
我们经常可以在池塘边的水草上能见到福寿螺的卵，粉粉嫩嫩的还有些可爱。但如果遇到了，建议尽量将其敲落至水中，**福寿螺的卵在水内无法成活**。
# 蜀葵
二年生直立草本，高达2米，茎枝密被刺毛。叶近圆心形，掌状5-7浅裂或波状棱角，两面都被毛。花腋生，单生或近簇生，排列成总状花序式，花大，直径6-10厘米，有红、紫、白、粉红、黄和黑紫等色，单瓣或重瓣，花瓣倒卵状三角形，先端凹缺，基部狭，爪被长髯毛。果盘状，具纵槽。花期3-8月。
# 球蛛
小到中型蜘蛛，多数种类腹部较大，呈圆球状，因此得名。球蛛的网，甚至有些种类的卵袋也是近球形的。这种球型网似天罗地网，让粘网的小型昆虫难以逃脱，越挣扎越可能被粘得更牢，最后在气力用尽时成为球蛛的美食。

合肥常见的球蛛科蜘蛛如：日本拟肥腹蛛，它的网中常会有落叶等自然凋落物，借用天然材料建造伪装帐，让天敌没办法直接攻击到它。
# 直行马路
一种扁平的倍足纲动物，全身由20对左右扁扁的体节组成，每个体节两边各有一鲜黄色斑块。头部具有一触角，大多数体节具有2对步足。

喜欢阴暗潮湿的环境，主要以落叶等腐殖质为食，是生态系统中重要的分解者，虽然有很多脚但是走路速度非常缓慢，受到刺激后，会散发出特殊的臭气驱逐敌人。
# 爱国者腹马陆
一种扁平的倍足纲动物，全身由20对左右扁扁的体节组成。背板前部黑色，后部及背侧板红色或白色。触角黑褐色，足淡黄色。

喜欢阴暗潮湿的环境，主要以落叶等腐殖质为食，是生态系统中重要的分解者，虽然有很多脚但是走路速度非常缓慢，受到刺激后，会散发出特殊的臭气驱逐敌人。
# 球鼠妇
又名西瓜虫，全身呈椭圆形，身体扁平，背上具有比较坚硬的背甲，身体下方具有14只脚，用于爬行。受惊后会蜷缩成球状，保护自己。

食性较杂，主要以枯草、枯叶、真菌等为食，有时也会啃食其他动物的尸体。主要在潮湿、腐殖质丰富的地方活动，如潮湿的石头下、木头中、草丛等。
# 木槿
落叶灌木，花期7-10月。高3-4米，小枝密被黄色星状绒毛。叶菱形至三角状卵形，具深浅不同的3裂或不裂。

花单生于枝端叶腋间，被星状短绒毛；小苞片6-8，线形，密被星状疏绒毛；花钟形，淡紫色，花瓣倒卵形，长3.5-4.5厘米，外面疏被纤毛和星状长柔毛。蒴果卵圆形，直径约12毫米，密被黄色星状绒毛；种子肾形，背部被黄白色长柔毛。栽培品种多重瓣。
# 木芙蓉
落叶灌木或小乔木，高2-5米。花期8-11月，果期10-3月。叶宽卵形至圆卵形或心形，常5-7裂，裂片三角形，上面密被星状细毛和点，下面密被星状细绒毛。

花单生于枝端叶腋间，初开时白色或淡红色，后变深红色。直径约8厘米，花瓣近圆形。蒴果扁球形，被淡黄色刚毛和绵毛，种子肾形，背面被长柔毛。
# 再力花
多年生挺水草本植物，花果期8-11月。叶卵状披针形，浅灰蓝色，边缘紫色，长50厘米，宽25厘米。复总状花序，花小，紫色。全株附有白粉。细长的花茎可高达3米，茎端开出紫色花朵，像系在钓竿上的鱼饵，形状非常特殊。蒴果近圆球形，果皮浅绿色，成熟时顶端开裂，种子棕褐色。
# 东亚伏翼
体长在4厘米左右的小型蝙蝠。前肢和后肢之间具有薄薄的翼膜，背部毛发多为黑灰色，有些个体为灰褐色，腹部毛色比背部略淡。耳朵呈三角形，但是耳尖比较圆钝。
合肥地区最常见的蝙蝠，主要以小型昆虫为食如蛾子、蚊子等。白天基本在城市建筑和居民区的建筑缝隙中休息，夜间出没捕食，飞行轻快而多变。
# 东北刺猬
体长约22厘米到28厘米的刺猬。身体比较圆胖，头尖而长，耳朵、四肢和尾巴均比较短小。背部褐色，腹部灰色，背上有许多浅色的硬刺。
东北刺猬主要以昆虫、蚯蚓等动物为食，有时也会吃一些果实、草根等植物，但不会将果实扎在自己背后的刺上背着。主要在黄昏后开始出没，遇到危险会将自己蜷缩成球状，用尖刺御敌。
# 隐翅虫
体型较小，身体大多十分细长，触角为丝状。翅膀非常短小，一般盖不住腹部，而腹部经常可以做卷腹运动。有的种类颜色搭配非常鲜亮。
常生活在植物上或者一些动物的巢穴内，喜欢湿润环境。可以吃植物也可以捕食小昆虫或者软体动物。一些毒隐翅虫体液具有强酸性，可能会引起皮炎。如果发现它在你身上，吹走即可，不要用手去拍打。
# 栾树
俗称**灯笼树**，落叶乔木或灌木；花期6-8月，果期9-10月。树皮厚，灰褐色至灰黑色，老时纵裂。羽状复叶，小叶纸质，卵形、阔卵形至卵状披针形。
聚伞圆锥花序，花淡黄色，稍芬芳；花瓣4，开花时向外反折，线状长圆形。蒴果圆锥形，具3棱，入秋后渐渐由黄绿色变为红色；种子近球形。
# 乌蔹莓
草质藤本，花期3-8月，果期8-11月。小枝圆柱形，有纵棱纹；卷须2-3叉分枝，相隔2节间断与叶对生。叶为鸟足状5小叶，中央小叶长椭圆形或椭圆披针形。花序腋生，复二歧聚伞花序；花序梗长1-13厘米；花瓣4，三角状卵圆形，雄蕊4；果实近球形。
聚伞圆锥花序，花淡黄色，稍芬芳；花瓣4，开花时向外反折，线状长圆形。蒴果圆锥形，具3棱，入秋后渐渐由黄绿色变为红色；种子近球形。
# 鸡矢藤
别名鸡屎藤，花期5-9月，果期9-11月。茎长3-5米，无毛或近无毛。叶对生，纸质或近革质，形状变化很大，卵形、卵状长圆形至披针形，圆锥花序式的聚伞花序腋生和顶生，扩展，分枝对生，花冠浅紫色，管长7-10毫米，外面被粉末状柔毛，里面被绒毛，顶部5裂。果球形，成熟时近黄色。
# 一年蓬
一年生或二年生草本，花期5-9月，果期6-10月。茎粗壮，高30-100厘米，上部有分枝，下部被开展的长硬毛。基部叶花期枯萎，最上部叶线形，全部叶边缘被短硬毛，两面被疏短硬毛，或有时几近无毛。
头状花序数个或多个，排列成疏圆锥花序，总苞片背面密被腺毛和疏长节毛；外围的雌花舌状，2层，白色，或有时淡天蓝色；中央的两性花管状，黄色。
# 蓼（liao三）
常见草本植物，茎直立、平卧或上升，通常节部膨大。叶互生，具有膜质托叶鞘，筒状。花序穗状、总状、头状或圆锥状，花小。蓼的种类很多，叶面常有人字形暗紫色斑纹。常见的有红蓼、水蓼、愉悦蓼等，广泛生长在河沟、山谷、路边、田边等地。

**秋天是蓼科植物的天下，俗话说“十分秋色五分蓼”**。

红蓼的花穗红艳艳，形如狗尾巴，高大招摇。诗经里提到红蓼，“山有桥松，隰有游龙。”红蓼的枝叶和红色的花穗在水边随风摇曳，好似红色的龙在游动。
辣蓼长在水边，又叫水蓼，开花低调一些，是过去美食中调辣的食材之一。
愉悦蓼花如其名，一小片一小片的淡粉色看了让人心情愉悦，田野间最能引人注目的就是它了。

一年生或多年生草本，茎直立，平卧或上升，通常节部膨大。叶互生，多为披针形、椭圆形、箭形、或戟形；托叶鞘膜质或草质，筒状。花序穗状、总状、头状、或圆锥状，顶生或腋生。

花被5深裂稀4裂，瘦果卵形，具3棱或双凸镜状。广泛生长在河沟、山谷、路边、田边等地，常见的有酸模叶蓼、红蓼、水廖。
# 梭鱼草
常见水生植物，据说北美一种叫梭鱼的幼鱼喜欢藏匿在叶丛和根茎之间嬉戏，故名。叶片翠绿丛生，穗状花序紫色，高于叶面直立顶生。蓝紫色的花朵像飞舞的小鸟，上方花瓣有黄绿色的斑点。也有开白花的，数量相对少。
# 黄花狸藻
常见水生草本。茎有许多分枝（叶器），每个叶器又有着复杂的深裂，细如毛发的裂片上，长着很多芝麻粒大的扁圆形“气球”，这是它们的捕虫囊。
囊口有一些毛，可以感受振动。如果有水蚤等小型水生动物触碰到了这些毛，囊口就会打开，因为内外的压力差，虫和水都会被吸进捕虫囊，然后被囊内的消化液所消化。
# 菱ling
一年生浮水水生草本，“水八仙”之一。根二型：着泥根细铁丝状，着生水底；同化根，羽状细裂，裂片丝状。茎柔弱分枝。叶二型：浮水叶呈旋叠状镶嵌排列在水面成莲座状的菱盘，叶片菱圆形或三角状菱圆形，表面深亮绿色，背面灰褐色或绿色；沉水叶小，早落。
花小，单生于叶腋，花瓣4,白色。果三角状菱形，2肩角直伸或斜举，，内具1白种子。花期5-10月，果期7-11月。
# 芡实
睡莲科芡属水生植物，“水八仙”之一。**漂在水面的圆形叶片和蓝紫色花朵都与睡莲极为相似，区别是芡实上面长满了刺。种子就是俗称的“鸡头米”，淀粉含量丰富，可用来勾芡。**
# 慈姑
多年生水生或沼生草本，“水八仙”之一。根状茎横走，较粗壮，末端膨大或否。挺水叶箭形。花葶直立，挺水，花序总状或圆锥状，具花多轮，每轮2-3花；花被片反折，外轮花被片椭圆形或广卵形；内轮花被片白色或淡黄色。瘦果两侧压扁，倒卵形，具翅。种子褐色。花果期，5-10月。
# 泽泻
多年生水生或沼生草本。叶通常多数；沉水叶条形或披针形；挺水叶宽披针形、椭圆形至卵形，先端渐尖，基部宽楔形、浅心形，叶脉通常5条。花葶高78-100厘米，或更高；花白色，粉红色或浅紫色。瘦果椭圆形，种子紫褐色，具凸起。花果期5-10月。
# 美人蕉
多年生草本植物，高可达1.5米，全株绿色无毛，被蜡质白粉。具块状根茎。地上枝丛生。单叶互生；具鞘状的叶柄；叶片卵状长圆形。总状花序，花单生或对生；萼片3，绿白色，先端带红色；花冠大多红色，外轮退化雄蕊2-3枚，鲜红色；唇瓣披针形，弯曲；蒴果绿色，长卵形，有软刺。花果期3-12月。
# 芦苇
多年水生或湿生的高大禾草，根状茎十分发达。杆直立，高1-3米，具20多节。叶鞘长于节间；叶舌边缘密生短纤毛，易脱落；叶片披针状线形。圆锥花序大型，分枝多数；在开花季节特别漂亮，可供观赏。全株具有通气组织，茎秆坚韧，纤维含量高可用于造纸。
# 水烛
多年生沼生草本，又名**香蒲、水蜡烛**。花期6-9月，果期7-10月。地上茎直立，粗壮，叶片呈海绵状。雌雄花序远离，雄花序在上，雌花序在下。叶片用于编织、造纸等；根状茎先端可作蔬食；雌花序可作枕芯和坐垫的填充物，是重要的水生经济植物之一。
# 芦竹
多年生水生植物，具发达根状茎。秆粗大直立，坚韧，具多数节，常生分枝。叶鞘长于节间，无毛或颈部具长柔毛；叶舌截平，先端具短纤毛；叶片扁平，上面与边缘微粗糙，基部白色，抱茎。圆锥花序极大型，分枝稠密，斜升；背面中部以下密生长柔毛，两侧上部具短柔毛，颖果细小黑色。花果期9-12月。
# 合欢
落叶乔木，合肥常见行道树。花期6-7月，果期8-10月。二回羽状复叶；头状花序于枝顶排成圆锥花序；花粉红色，有香味。荚果带状。
合欢开花是粉柔柔的，像扇子挂满了枝头，还带着一股淡淡的香气，叶子也是随着花开花谢而晨展暮合。
# 乌桕
落叶大乔木，高可达15米，树皮暗灰色，有纵裂纹。叶互生，纸质，叶片菱形、菱状卵形或稀有菱状倒卵形，长蒴果梨状球形，成熟时黑色，直径1-1.5厘米，白色的部分是蜡质假种皮，富含油脂，可做肥皂，里面的种子黑色，可以榨油。
主要分布于黄河以南各省区，北达陕西、甘肃。生于旷野、塘边或疏林中。合肥常见园林树木，秋季树叶变黄或是变红，甚是美丽。
# 无患子
落叶大乔木，高可达20余米。花序顶生，圆锥形；花小，辐射对称，花梗常很短；花瓣5，披针形，有长爪，长约2.5毫米，雄蕊8，果近球形，橙黄色，干时变黑。果子稍微弄烂一点，放入水中，用力摇晃，可以起泡泡。
合肥常见园林物种，基本每个小区都有，可以仔细留意哦。
# 元宝槭qi
合肥常见园林物种。落叶乔木，花期4月，果期8-11月。高8-10米，树皮灰褐色或深褐色，深纵裂。叶纸质，长5-10厘米，宽8-12厘米，常5裂，稀7裂，边缘全缘。
花黄绿色，杂性，雄花与两性花同株，常成无毛的伞房花序，花瓣5，淡黄色或淡白色。翅果嫩时淡绿色，成熟时淡黄色或淡褐色。
# 鹅掌楸qiu
**又称马褂木，乔木，高达40米**。花期4月，果期9-10月。**叶马褂状**，长4-12厘米。花杯状，花被片9，外轮3片绿色，萼片状，向外弯垂，内两轮6片，长3-4厘米，绿色，具黄色纵条纹，花期时雌蕊群超出花被之上。聚合果长7-9厘米，具翅的小坚果长约6毫米。常见本种与北美鹅掌楸的杂交种。
# 三角槭
合肥常见园林物种，**别名三角枫**。落叶乔木，花期4月，果期8-11月。高5-10米，稀达20米。树皮褐色或深褐色，粗糙。叶纸质，通常浅3裂，裂片向前延伸，稀全缘，中央裂片三角卵形，急尖、锐尖或短渐尖。
花多数常成顶生被短柔毛的伞房花序，开花在叶长大以后，花瓣5 ，黄绿色；翅果黄褐色，中部最宽，基部狭窄，张开成锐角或近于直立。
# 蚯蚓
**身体柔软，由一圈圈的环节构成，体表具有刚毛和黏液，辅助爬行**。**成年的蚯蚓在身体中间多具有一不同颜色的环带，里面是蚯蚓的生殖器官**。

**主要生活在潮湿泥土中，以腐烂的植物为食，是生态系统重要的分解者。蚯蚓活动能够有效疏松土壤，其粪便是许多植物的优质肥料。同时蚯蚓还是许多动物重要的食物**。

# 青葙
一年生草本，高0.3-1米；花期5-8月，果期6-10月。茎直立，有分枝，绿色或红色，具显明条纹。叶片披针形，绿色常带红色。
**花多数，密生，形成红白相间的宝塔状花穗**；花被片初为白色顶端带红色，或全部粉红色，后成白色。**常见的鸡冠花即为青葙的栽培变种**。
# 合萌
一年生草本或亚灌木状，花期7-8月，果期8-10月。茎直立，高0.3-1米。叶具20-30对小叶或更多；小叶近无柄，薄纸质，线状长圆形，上面密布腺点，下面稍带白粉。

花冠蝶形，淡黄色，具紫色的纵脉纹，易脱落。荚果线状长圆形，背缝多少呈波状；荚节4-8（-10），成熟时逐节脱落；种子黑棕色，肾形。
# 翅果菊
秋日常见小菊花，植株高大，甚至可以长到一人多高。**苞片四层，覆瓦状排列**，边缘有紫色描边。**花序一元硬币大小，全是舌状花，淡黄色。“花瓣”顶端分成5个小齿，每瓣“花瓣”对应中间的一个“花蕊”**。种子也有特点，不像蒲公英那些菊，瘦果细长，翅果菊的瘦果扁平，近似圆形。
# 三裂叶薯
别名小花假番薯，一年生草本，外来入侵物种；茎缠绕或有时平卧，叶型多变，宽卵形至圆形，全缘或有粗齿或深3裂，基部心形。花序腋生，花冠漏斗状，长约1.5厘米，淡红色或淡紫红色，喉部深紫色。蒴果近球形，成熟时裂开，种子4或较少。
# 盒子草
葫芦科缠绕草本，其攀援工具是由茎变态而来的卷须，卷须着生在叶腋。花序腋生，花朵细密，白色，花瓣上下两层，每层5瓣，乍一看就像只章鱼。
# 钻叶紫菀wan
菊科一年生草本，高可达150厘米。茎单一，直立，茎和分枝具粗棱，光滑无毛，基生叶在花期凋落；茎生叶多数，叶片披针状线形，头状花序极多数。雌花花冠舌状，舌片淡红色、红色、紫红色或紫色，线形，两性花花冠管状，冠管细，瘦果线状长圆形，稍扁，6-10月开花结果。

**果实水滴型，表面有小突起和绒毛，腰部有缝**，等到果实成熟，稍稍施加外力，盒子就会沿着中缝崩裂开。湖边、水塘边的许多植物上都有盒子草密集缠绕攀附。
# 悬铃木
落叶大乔木，树皮灰绿或灰白色，不规则片状剥落。叶片3～5掌状分裂，边缘有不规则锯齿。花期4～5月，头状花序球形；球果下垂，通常1-3球一串，9-10月果熟。

**世界著名的优良庭荫树和行道树，有“行道树之王”之称。中国引入栽培的有3种，又被称为"法桐"、“法国梧桐”，但悬铃木与梧桐是两种不同的植物**。

# 梧桐
又称**青桐**，落叶乔木，高达16米；**树皮青绿色，平滑**。叶心形，掌状3-5裂，叶柄与叶片等长。圆锥花序顶生，花淡黄绿色。花期6月。

蓇葖果膜质，有柄，成熟前开裂成叶状，每蓇葖果有种子2-4个；种子圆球形，表面有绉纹，直径约7毫米。
# 喜树
合肥常见园林物种，落叶乔木。树皮灰色，纵裂成浅沟状。叶互生，纸质，矩圆状卵形，全缘。头状花序近球形，通常上部为雌花序，下部为雄花序。花杂性，同株；花瓣5枚，淡绿色。翅果矩圆形，两侧具窄翅，幼时绿色，干燥后黄褐色。花期5-7月，果期9月。
# 枫香
合肥各个公园常见园林物种。落叶乔木，花期4-5月，果期8-10月。叶薄革质，阔卵形，掌状3裂，搓揉后可闻到香味。雄性花序短穗状。雌性头状果序圆球形；种子多数，褐色，多角形或有窄翅。
# 枫杨
大乔木；幼树树皮平滑，浅灰色，老时则深纵裂。叶多为偶数或稀奇数羽状复叶，叶柄长2-5厘米，叶轴具翅；小叶对生或稀近对生，长椭圆形。
雄性葇荑花序单独生于去年生枝条上叶痕腋内，雌性葇荑花序顶生。果序长20-45厘米，果实长椭圆形；果翅狭，条形或阔条形，具近于平行的脉。花期4-5月，果熟期8-9月。
# 女贞
灌木或乔木，高可达25米；树皮灰褐色。枝黄褐色、灰色或紫红色，圆柱形，疏生圆形或长圆形皮孔。叶片常绿，革质，椭圆形，先端锐尖至渐尖或钝，叶缘平坦。
圆锥花序顶生，果肾形或近肾形，深蓝黑色，成熟时呈红黑色，被白粉。花期5-7月，果期7月至翌年5月。
# 重阳木
落叶乔木，高达15米；树皮褐色，纵裂；树冠伞形状，大枝斜展。三出复叶；叶柄长9-13.5厘米；顶生小叶通常较两侧的大。花雌雄异株，春季与叶同时开放，组成总状花序；花序通常着生于新枝的下部。果实浆果状，圆球形，直径5-7毫米，成熟时褐红色。花期在4-5月，果期10-11月。
# 黄连木
落叶乔木，高达20余米；树干扭曲，树皮暗褐色，呈鳞片状剥落。羽状复叶互生；小叶对生或近对生，纸质，卵状披针形。花单性异株，先花后叶，圆锥花序腋生。核果倒卵状球形，略压扁，成熟时紫红色。
木材鲜黄色，可提黄色染料，材质坚硬致密，可供家具和细工用材。种子榨油可作润滑油或制皂。幼叶可充蔬菜，并可代茶。
# 盐肤木
又名五倍子树，落叶小乔木或灌木，高2-10米；小枝棕褐色，被锈色柔毛，具圆形小皮孔。奇数羽状复叶有小叶(2-) 3-6对，叶轴具宽的叶状翅，小叶自下而上逐渐增大，无柄。圆锥花序宽大，多分枝，花白色。核果球形，略压扁，成熟时红色，味酸、咸。花期8-9月，果期10月。
# 水杉
高大乔木，树姿优美，为著名的庭园树种。树干基部常膨大；树皮灰色、灰褐色或暗灰色，幼树裂成薄片脱落，大树裂成长条状脱落；枝斜展，小枝下垂。叶条形，在侧生小枝上列成二列，羽状，冬季与枝一同脱落。球果下垂，近四棱状球形或矩圆状球形，成熟前绿色，熟时深褐色; 种子扁平，倒卵形，周围有翅。花期2月下旬，球果11月成熟。
# 构树
合肥常见落叶乔木，高10-20米。叶似桑叶而粗糙被绒毛，先端渐尖，基部心形，边缘具粗锯齿，不分裂或3-5裂，小树之叶常有明显分裂。 
雌雄异株，雄花序为柔荑花序，雌花序球形头状。果圆球形，成熟时红色，肉质多汁，常有鸟儿、小型哺乳动物取食。花期4-5月，果期6-7月。
# 池杉
乔木，树干基部膨大，通常有屈膝状的呼吸根（低湿地生长尤为显著）；树皮褐色，纵裂，成长条片脱落；枝条向上伸展，树冠较窄，呈尖塔形。叶钻形，微内曲，在枝上螺旋状伸展。球果有短梗，向下斜垂，熟时褐黄色；种子不规则三角形，微扁，红褐色。花期3-4月，球果10月成熟。
# 落羽杉
落叶乔木，树干尖削度大，树干基通常膨大，常有屈膝状的呼吸根；树皮棕色，裂成长条片脱落。叶条形，扁平，基部扭转在小枝上列成二列，羽状，凋落前变成暗红褐色。球果有短梗，向下斜垂，熟时淡褐黄色，有白粉，径约2.5厘米；种子不规则三角形，有锐棱，褐色。球果10月成熟。
# 雪松
常见栽培树木，终年常绿，树形美观，有树脂，具香气。树皮深灰色，裂成不规则的鳞状块片；枝平展、微斜展或微下垂。
叶在长枝上辐射伸展，短枝之叶成簇生状，针形，坚硬，淡绿色或深绿色，常成三棱形，叶之腹面两侧各有2-3条气孔线。球果成熟前淡绿色，微有白粉，熟时红褐色；种子近三角状，种翅宽大。
# 五针松
松科松属的植物，针叶5针一束，针叶紧密秀丽，又因长期嫁接高生长受到抑制而成灌丛状的小乔木，树枝优美，为珍贵的观赏树种，花园、居住区、宾馆的广场露地栽种较多。

球果卵圆形或卵状椭圆形，几无梗，熟时种鳞张开；种子为不规则倒卵圆形，近褐色，具黑色斑纹。
# 八角金盘
常绿灌木或小乔木，茎光滑无刺。叶片大，革质，近圆形，掌状7-9深裂。上表面暗亮绿色，下面色较浅，有粒状突起，边缘有时呈金黄色。

圆锥花序顶生，长20-40cm；伞形花序直径3-5cm；花瓣5，黄白色。果实近球形，熟时黑色。花期10-11月，果熟期翌年4月。
# 杠板归
一年生草本，花期6-8月，果期7-10月。茎攀援，多分枝，长1-2米，具纵棱，沿棱具稀疏的倒生皮刺。叶三角形，薄纸质，上面无毛，下面沿叶脉疏生皮刺；叶柄具倒生皮刺，盾状着生于叶片的近基部。

总状花序呈短穗状，花被5深裂，白色或淡红色，花被片椭圆形，果时增大，呈肉质，深蓝色。瘦果球形，黑色，有光泽，包于宿存花被内。
# 大吴风草
多年生葶状草本，常作为地被植物而被人们栽培。根茎粗壮，花葶高达70厘米。叶全部基生，莲座状，有长柄，柄基部扩大，呈短鞘，抱茎；叶子酷似莲叶，所以也叫“活血莲”与“一叶莲”。
头状花序辐射状，2-7，排列成伞房状花序；舌状花8-12，黄色；管状花多数。瘦果圆柱形，花果期8月至翌年3月。 
# 南天竹
常绿小灌木。茎常丛生而少分枝，高1-3米，光滑无毛，幼枝常为红色，老后呈灰色。叶互生，集生于茎的上部；二至三回羽片对生；小叶薄革质，全缘，上面深绿色，冬季变红色。
圆锥花序直立，花小，白色，具芳香。浆果球形，熟时鲜红色，稀橙红色。种子扁圆形。花期3-6月，果期5-11月。
# 香樟
常绿大乔木，高可达30米，树冠广卵形；枝叶及木材均有樟脑气味；树皮黄褐色，有不规则的纵裂。叶互生，卵状椭圆形，先端急尖，边缘全缘，软骨质，有时呈微波状，上面绿色有光泽，下面灰绿色较暗，具离基三出脉。幼时树皮绿色平滑，老时渐变为黄褐色或灰褐色纵裂。

圆锥花序腋生，花绿白或带黄色。果卵球形，紫黑色；果托杯状。花期4～ 5月，果期8～11月。
# 闻啼鸟的迎春传统：寻踪老鸦瓣——合肥常见野花｜早春使者：老鸦瓣

二月，郊野大地的山坡草丛中，有一种植物的花苞迫不及待地探出头来，轻声唤醒了春天。它的名字叫做老鸦瓣，花期早，紧追春天的脚步，堪称早春的使者。


“老鸦瓣生田野中，湖北谓之棉花包，固始呼为老鸦头。春初即生，长叶铺地，如查草而屈曲索结，长至尺余。抽攀开五瓣尖白花，似海桅子而狭，背淡紫，绿心黄蕊，入夏即枯，根如独颗蒜。”——清·吴其濬《植物名实图考》

这是古人通过观察总结出的老鸦瓣每年冬天抽叶、早春开花、入夏前以地下球茎的方式休眠的生命旅程。如果你观察过老鸦瓣，就能发现其中一些有误的记录。

趁大多数植物还沉睡梦中尚未苏醒，老鸦瓣选择在早春难得晴朗的日子里完成开花结果的“花”生使命，这是早春开花植物的生存智慧。

所以天气晴朗的春日要出去走一走，找到属于自己的第一朵老鸦瓣。如果它开了，陪它静待传粉精灵的到来。如果它还没开，那就坐下来等一等，和它一起绽放。

老鸦瓣是百合科老鸦瓣属植物，在形态和气质上都很像山野里的小百合。
两枚（初生仅一枚）叶片狭长似蒜叶，但更为纤细小巧；

小花顶生，花瓣6枚，分为里外两层，白色，背面饰以紫红色条纹，配色精致；

果实未成熟时是个有三条棱的绿色小球，顶端有长长的“天线”；

地下鳞茎似蒜头，因其富含淀粉曾在饥荒年代被当作野菜，但其实有毒，没事不要挖来吃啊。

不开花的时候老鸦瓣与杂草无异，所以想在野草地上一眼找到它也并没有那么容易。

你得弯腰弓背，甚至蹲下来、轻轻拨开枯枝落叶，才有可能发现它的身影。开了花的老鸦瓣则出尘脱俗、清新别致，除了把你美出一连串“哇塞”，还能增加你的惊喜感和满足感。

花园里美丽的郁金香是它的远房亲戚，但是老鸦瓣身形娇小，在草丛里不太起眼。所以如果你想去草地上寻找它，一定要小心翼翼不要踩到它哦。

但是这么好看的花仙子却有个土里土气的名字，难道仅仅是因为它是野生的？想知道答案的话，花开之后过段时间再去看看它的果实吧。

猜测一下，老鸦瓣名字的由来跟它的果实形状有关，果实有喙，也就是顶端那根“天线”。
仔细看，像不像某些鸟儿的嘴？

每年春天，在少有人迹的草地上寻找老鸦瓣，是闻啼鸟er的迎春传统。
# 春日盛宴，舌尖上的野花野草
在人类文明发展的漫长过程中，我们一直从大自然中获得能量，狩猎、采集、驯化、培育......帮助人类发展进步，当然这也体现了人类强大的生存智慧。 

而在现今科技高度发达的社会中，很多人会怀念田间地头的味道。尤其是在万物复苏的春季很多人都会去寻觅野菜。而草地上的这些小野菜们，或许承载着人们对家乡的思念，对生命的思考，对大自然的感恩与敬畏。

你吃过野菜吗？你挖过野菜吗？
今天介绍的几种野花野草它们都曾经出现在我们人类的餐桌上，让我们一起去认识一下吧。


01荠菜 国民认知度最广的小野菜

荠菜味道的鲜美想必不用我多言，上一篇文章（《春日序章，小野花也有春天！》）中也有介绍到它，只是因为它在春天野味中的重要性，在这里不得不再把它提出来。

它的花形是典型的十字花形，而它的果实也很是有趣，有人说是爱心形状，有人说像吉他的拨片，更有小朋友说它像海底捞门口送的虎牙脆。

也由于人们对它味道的痴迷，现代农业的种植技术将其予以改良，所以现在的菜市场中已经可以比较方便地买到它，猪肉荠菜馅的饺子也是很多饺子馄饨店的必备菜品。

02蒿子 最有传统文化气息的小野菜

说到蒿子，这其实是个大家族，端午节常用的艾草其实也是一种蒿子。 我们这里提到的蒿子又称野艾蒿，叶片正面是绿色，背面有白色绒毛，茎干的高度也不过十多公分，远没有艾草那么高大。很多老人家摘取蒿子，都有自己的独特的识别技巧，却又好像不尽相同。

蒿子被用来做成食物，也是由来已久了，蒿子粑粑也是合肥及周边很多地区春天必吃的一样春味。

经过焯水，它身上原本略显刺鼻的气味变得柔和。配合面粉、腊肉一起做出来的蒿子粑粑，香味浓郁，是很多人的心头所好。很多地区也用蒿子来制作青团。

03鼠麴（qu二）草 最有文艺气质的小野菜

看它的模样你也许不会想到这是菊科大家族下的物种，它通体碧绿，全身都长有白色的绒毛，摸上去手感特别好，要是你没有摸过，那这个春天推荐你找到它，去尝试一下。

也正因为如此，在雨后的清晨才有了下面这张挂满水珠的照片，晶莹剔透，很是可爱。

它的花朵是金黄色的，一簇一簇长在一起，却又特别的小巧，更显出它的别致。鼠麴草也是很多地方制作青团的主要原料，看到它，你一定对青团的颜色由来再无疑问了吧。
 
04黄鹌an菜 最具视觉效果的小野菜

有一个说法是只要这些野菜的名字中带有“菜”这个字的，那基本都是上过餐桌没跑了。黄鹌菜同样来自菊科大家族，旧时多选用它的嫩叶进行食用。 

它的茎高高直立，在顶端开放着成百上千朵舌状小花，非常容易在树林中形成花海。荒野一些的公园山林中在每年4月份时会迎来它的盛开期，它鲜嫩的黄色与新生树叶清新的绿色交相呼应，很有春天的感觉。

05救荒野豌豆 最朴实无华的小野菜

单单从它的植物名就可以看出它曾经对人类的生存发挥了多大的作用。在早春时，它们便钻出泥土，用叶子顶端的蔓须向上缠绕用力去争取阳光，所以一旦气候适宜，它们会生长地飞快。 

人们对于豌豆的喜欢也是非常明显的，不管是火锅店里的豌豆尖还是凉菜摊上的水煮豌豆，给人们留下最经典的记忆是鲜嫩。

而作为其亲戚的救荒野豌豆不管是花形还是果实都与豌豆有着非常多的相像之处，食用之法也颇为相似。

除此之外，我还特别喜欢将救荒野豌豆成熟的果荚做成口哨，可以发出婉转的声响。这个声响也是我儿时上学路上的陪伴，现在每当玩起这个，还能想起儿时在田野中奔跑的景象。

所以在闻啼鸟之前的课程中，如果遇到了成熟的救荒野豌豆，我也会带孩子们一起玩起来。

06南苜mu蓿su 最不为人知的小野菜

南苜蓿又叫草头、金花菜，它的茎叶平铺在地面上向四周生长，叶片似典型的三叶草，每片小叶片又犹如爱心形。金黄色的花朵簇拥在一起，仔细观察每一朵的结构又像蝴蝶一般。 

相传苜蓿是在汉朝时随着汗血宝马一起传入我国的，因为它是汗血宝马的重要饲料，所以它的味道应该也不会差吧。旧时的吃法大多选取鲜嫩的茎叶焯水后炒食或者凉拌。


07薤白 最易制作的小野菜

对于孩子们来说最有吸引力，也是食用操作起来最方便的应该就是属它了。它的独特气味与叶形也让它非常容易识别，很多地方又叫它为野葱。在合肥山林的落叶堆或是田间地头都容易见到它的身影。 

它的味道就如同我们菜市场中买到的小葱，食用方面主要是它来炒鸡蛋或是做为菜肴的调料，那味道想必隔着屏幕，你都能感受的到吧。

# 春日序章，小野花也有春天！
早春的野花，最能代表春日万物生长的态度——它们在乍暖还寒的微风中摇曳，看似平庸渺小，却有目标、有决心，代表着一种顽强的生命力。

这种生命力可以让它们在贫瘠的土地中盛放，在苍白中五光十色。它们未经雕琢，却生机勃勃，用各种各样的生长形式打开你对春天的认识。

合肥的春日序章已经开始，闻啼鸟将陆续给大家介绍合肥春日里的自然。
本篇，我们一起去找找这些不起眼又了不起的小花小草吧。

01老鸦瓣

早春开花植物，花期3-4月，常见于山坡草地路旁，野生动物园附近有较多分布 。
叶2枚，长条形，像蒜叶，远比花长。花通常单朵顶生，形似百合，花瓣6枚，白色，背面有紫红色纵条纹。果实近球形，有三条棱，有长喙。入夏即以地下鳞茎的方式休眠。

02阿拉伯婆婆纳
草地常见小野花，又名波斯婆婆纳。
花期3-5月，更早也能看到，多铺散成片生长。花冠蓝紫色，裂成4片，有深色条纹，基部相连，花梗较长。花蕊高于花瓣，形似芭蕾舞女。
果实肾形，形似婆婆的针线包。

03繁缕
草地常见小野花，草如其名，“茎蔓甚繁”，铺地生长。茎的一侧有很多细绒毛，且“中有一缕”。这一缕指的是，茎折断后，中间有一根有弹性的“筋”。
白色小花，看似10个花瓣，其实是5个，每个花瓣深裂成2片长椭圆形，像兔子的耳朵。

04球序卷耳
“采采卷耳，不盈顷筐。”——先秦·佚名《卷耳》
诗经里的卷耳在今天是什么样子呢？
是个全株毛茸茸的草本植物；茎丛生，直立，带点紫红色；叶片椭圆形，对生；花期为3-5月，花瓣有5、白色，先端具有2浅裂，与繁缕的花很像，就是兔耳朵短一点。

05荠菜
“城中桃李愁风雨，春在溪头荠莱花。”
荠菜是春天里最有代表性的一种野菜，也被称为春菜，因为它对气温要求比较低，回春比较早。荠菜的味道鲜美，中国人采食荠菜的历史由来已久，《诗经》里就有“甘之如荠”的诗句。
这种常常出现在饺子馅里的野菜有白色小花，4瓣，细如米粒，还有爱心形状的果实。不过荠菜等到开花结果就不好吃了。

06附地菜
附地菜，可以字面理解成一种贴附地面生长的野菜吧，不过一般也没人吃它。
花朵五瓣，特别小，如小米粒般。花冠是浅浅的蓝色，花心有一圈浅黄色。
叶片椭圆形，加上较长的叶柄，看起来像个勺子，据说叶子撕碎了有黄瓜的清香哦。

蛇 莓
蛇莓是草莓的远房亲戚，鲜红色的“果实”很像草莓，所以很多时候也被称作野草莓。
花黄色，5瓣，花下面还有5个萼片和5个副萼片，妥妥的护花使者。小叶3枚，有圆锯齿。
虽然长辈总是跟孩子说，蛇莓表面被蛇爬过或者蛇莓附近有蛇出没，不让吃。可能是为了让孩子们不要随意采食野果吧。但其实蛇莓没啥毒，当然，也不咋好吃。

08蒲公英
草地常见小野花，花期3-9月，黄色的头状花序其实是上百多小花聚集而成，果实成熟后形似透明的灯笼，风一吹，长“羽毛”的种子就开始飞行啦。
叶片铺地生长，羽状深裂，顶端裂片较大，呈三角形，整个叶片形似鱼骨。
基于以上特征，大概就能区分蒲公英和那些同样开黄花、果实有毛的菊科亲戚了。

09宝盖草
叶片半圆形，对生于直立茎两侧，围成一圈，像宝塔一样层层叠高，又形似古代帝王车顶的华盖，因而得名。
花期3-5月，唇形花朵，粉红色，花冠细长，像一个穿红裙跳舞的小姑娘。
但在早春气温较低时，宝盖草不怎么开花，这是它们利用自体花粉完成受精，也就是闭花授粉。

10泽漆
比较少见的开绿花的植物，泽漆有一些有趣的别名：五朵云、猫眼草，都是根据它的形态而来的。
名字里的“泽”体现其生境，“漆”是指植株有白色汁液，有毒。
茎顶有5片轮生的叶状苞，每片叶状苞上又生3个小伞梗。远远看去，如一朵翠绿色的云朵。

想要不错过春日野花的美，一定要离草地近一点、近一点、再近一点……
弯腰、蹲下、趴在草地上……

# 合肥秋日常见种子——快来打开这份秋日种子盲盒吧！ —— 合肥秋日常见种子30种（一）
（注：本文提到的“种子”泛指果实及其包含的种子。）
“一年好景君须记，最是橙黄橘绿时。”
秋意深浓，瓜果飘香，与此同时，大自然里也有一大波成熟的种子前来报到，快来打开这份属于你的秋日种子盲盒吧。

无患子
可以做洗手液的植物，是不是很有用？没准你家小区或者附近公园就有。

长椭圆形的叶子羽毛状排列，这个时候枝头已经挂满了一粒粒金黄色的果实，这就是无患子树了。果实长得跟龙眼有点像，也确实是“亲戚”，但闻起来却有点菠萝的芬芳，可惜不能吃。 

虽然不能吃，但无患子依然称得上是大自然赠予人类的宝藏玩具，玩法多样。除了制作天然洗手液，里面黑色的种子还可以用来串佛珠。与宗教有关联添了一丝神秘感，想必这也是它能得到一个有吉祥无患的寓意的名字的由来吧。

乌桕
“清秋霜未降，乌桕叶先红。”
青叶/摄 乌桕是常见的彩叶树种，入秋后满树彩虹色的叶子好看极了，菱形水滴状的叶片容易识别。果实也很有特色，成熟后由青绿色变成黑褐色，果皮会裂开脱落，露出里面白色的种子。  

乍一看，“一个黑屋子，里面住着三个白胖子”，是不是很形象？到了冬天叶子和果壳都掉光了，留在枝头的那些种子如点点梅花绽放，也是很多鸟儿喜欢吃的“棉花糖”！这样一棵棉花糖树说不定就在你家楼下哦！ 

栾树
长在树上的灯笼，而且是粉红色的！

秋日里能欣赏到这副光景，想想就很美丽。大自然也就是这么美丽！ 

这满树的灯笼就是栾树的果实，一般由3片看起来像树叶一样的果瓣围合而成，里面是空心的，所以也有人把它叫做泡泡果。只有为数不多的几粒种子藏在里面，像是住在一个巨大的帐篷里。 如果你打开过这顶帐篷，说不定还会遇到有些其他小动物来拜访呢。 

喜树
喜树是中国所特有的一种高大落叶乔木，是国家二级重点保护野生植物。

喜树

它花开成秋，远远看像一个个毛绒绒的小球，结成的果实像一粒粒小香蕉组成的球串，很是可爱~ 喜树因为它颇为特别的名字，受到很多人们的喜爱，还经常与合欢树搭配种植，有着“欢天喜地”的寓意。

玉兰
合肥也有广泛种植好几种玉兰，如广玉兰、望春玉兰、二乔玉兰等，它们有着红色醒目的种子，这种红色M豆自然也是鸟儿的美食。



广玉兰
广玉兰的正式名称叫做荷花玉兰，顾名思义，琼花玉貌，出水芙蓉，也难怪被选为合肥的市树。但你留意过这么好看的市树，它的种子是什么样的吗？

怎么说呢，就像变魔术一样，硕大如碗的花朵谢掉之后，里面长出了粉红色玉米棒形状的果实，到了秋季果实会裂出一个个的口，吐出一粒粒红色的种子。 

而同样被称为玉兰的其他几种玉兰，合肥也有广泛种植，如望春玉兰、二乔玉兰等，它们也是这种红色醒目的种子，这种红色M豆自然也是鸟儿的美食。  

构树
名字很陌生却很常见的一种植物，说它是桑树的“亲戚”，是不是感觉亲切了几分？ 

成熟的果实跟桑葚一样甜美多汁，而且是红色圆球形，像是有人特意在树上系了很多毛线球，给季节增色。这美味的果实也是很多鸟儿、松鼠等小动物的水果大餐！  

槭树
“枫树”本树，我们俗称的枫树大多属于槭树这一类，包括很多种，如三角槭、鸡爪槭、元宝槭等。它们都符合“枫树秋天叶儿红”的特征，也符合种子有翅膀会飞的特征。  
 
我们把**这类有翅膀的种子统称为翅果**，所以如果你在户外看到槭树的翅果，不必纠结是哪一种，捡起它、让它飞一会儿才是正确的玩法！   

枫杨
不是枫树，也不是杨树，但起这样的名字，总归是有相似之处吧。 

你看，它的种子长了一对小翅膀，是不是跟“枫树”的种子很像？果实一长串从枝头垂下来，随风摇曳，是不是也很符合“杨柳依依”的意境？ 

女贞
秋冬季节，如果你抬头看见茂密的绿叶枝头藏着一大串紫黑色的迷你小葡萄，上面还披着一层浅浅白霜，那很有可能你站在一棵女贞树下。  

这些紫黑色的小葡萄晒干了就是女贞子，一味中药。女贞的果实跟桂花的果实（桂籽）很像，遇到了可以仔细对比一下哦~ 

 香樟
说它是长江流域最常见的行道树应该不为过，也是最多被选为市树的树种（没有之一）！
  
一到秋天，香樟就变成了葫芦娃树。至于葫芦里装的是什么样的种子，你打开盖子看看就知道了。这种葫芦样的果实备受鸟儿青睐，但是，也备受广大车主嫌弃。

青桐
《诗经·大雅·生民之什·卷阿》有"凤凰鸣矣，于彼高岗。梧桐生矣，于彼朝阳"之句，是凤栖梧桐的最早记载。梧桐是我国有诗文记载的最早的著名树种之一，即“中国梧桐”，别名青桐、桐麻，又称国桐。 

树干粗壮挺拔，树皮青色，相当清雅。 青桐树上结的桐子，很香，可以食用，和悬铃木的圆球形果实不一样的是，它的种子点缀在一个个小船一样的船沿边。

枫香
枫香果实为聚合头状蒴果，像个小刺球。

“停车坐爱枫林晚，霜叶红于二月花。”这其中的“枫”指的就是枫香，合肥有很多行道树栽种的是它。秋季到来，在色素的催化下，它的树叶就变成了红色、黄色、橙色等等，色彩丰富，甚是好看。

枫香果实为聚合头状蒴果，像个小刺球，很多小朋友把它当做小刺猬，用来玩游戏。再到深秋，它的种子从刺球中掉落，也有一个小小的翅膀。落完种子原有的刺球上就多了很多小孔，将它的果柄插入小孔，便有了个活脱脱的大戒指，很多小朋友都喜欢这样玩耍。

卫矛
为毛卫矛这么红？为毛卫矛这么好看？是呀，好看就行！

 栎li树
秋天了，大蜀山的栎树已经给它的孩子们戴了一顶厚实的帽子。

通常我们所说的栎树，或者橡树，泛指那些壳斗科的栎属植物，它们的种子就是我们俗称的橡子，喜欢戴一顶多刺的帽子，这也是它们被统称为壳斗科的原因。 

既然是俗称，也可以把橡子叫做栎子，虽然栎子≠栗子，但它确实跟我们吃的那些“栗子”是同科不同属的亲戚。如果你在野外遇到它们，请忽略这些科属种的分类，开心地把它们称为好玩的[lì zi] 就可以啦！ 

杉树
 池杉常栽种于水边，叶子细长，种子呈球形，表面粗面有褶皱。成熟后会裂成碎片掉落，像核桃碎。种子里有类似松脂的油，黏黏的有香味。

楝
楝树也是常见的绿化植物，果实成熟后由青色变金黄，常驻枝头，经冬不落，到了明年春天还能看到它与花与叶同在。  

楝树又称苦楝，据说是因为它的果实是苦的。（楝树有毒，不建议尝试哦。）苦楝不仅苦，果实腐烂的时候还有一股刺鼻的臭味。虽然苦，但鸟儿吃起来还是蛮开心的。  

柘（zhe四）
柘（这)是什么呢？  柘是好吃的！ 青叶/摄 柘树常见于山林，果实外形有点像荔枝，也是一种清甜美味的野果，遇到了可以浅尝一下~  

松树
松果是松科植物的球果，成熟后内有种子为松子，是小松鼠喜爱的食物。松果形状像峰塔，没裂开时外表看起来有点像鱼鳞。

木芙蓉
木芙蓉的花很漂亮，是常见的观赏植物。全株都有毛，尤其果实更是毛绒绒的，看起来像棉桃，种子很小。

悬铃木
著名行道树，俗称的“法国梧桐”，但它不是梧桐，跟法国的关系也不大，大名叫做悬铃木，顾名思义，果实成熟后黄褐色圆球形，像一个个小铃铛一样挂在树上。  

它其实是很多毛绒绒的种子密密麻麻挤压形成，到了第二年春天它们就蓬松裂开飞向城市的各个角落，剩下一个像月球表面一样坑坑洼洼的秃头（原来的花托）。  

银杏
银杏树扇形的叶子很特别，到了秋天还是一道亮丽的风景线。它的果实加工处理后就是我们吃的白果。银杏果落下来腐烂后也是别有一番风味，谁闻谁知道。  

南天竹
常见绿化植物，矮小灌丛，果实红色，有毒，但是不妨碍它成为美丽的观赏植物。
 
 麦冬与山麦冬
这个季节，如果你愿意弯下腰身在绿化带那些长得绿油油、像韭菜一样的茂密草丛里细细寻觅，说不定会得到一串蓝宝石或者黑宝石。
它们就是麦冬与山麦冬这对“表兄弟”，前者低调地隐匿在叶丛里，后者则高高立于叶片之上。 

杠板归
见过了蓝宝石和黑宝石，我们再去野草丛旁边转转，说不定还能发现另外一种红宝石与蓝宝石。只不过这两种宝石来自同一种植物——杠板归。

需要小心的是，这种好看的宝石草可不像麦冬兄弟那么柔软温顺，它是带刺的，甚至连三角形的叶片背面都长了刺，看这架势，怪不得它还有“蛇倒退”的别名了。  

香蒲
常见水生植物，果序像烤肠，也叫水蜡烛。成熟后会炸开变成松软毛絮状，可以做填充物。

蒲公英
谁的童年没有吹过蒲公英呢？
就算是成年人遇到了一朵蒲公英，也会忍不住吹一下，感受这种随时随地可见的小草带来的快乐，仿佛自己也背上了降落伞，和小伞兵们一起飞向远方。

盒子草
盒子草常见于水边，果实是水滴形的，非常特别。果如其名，是个可以打开的盒子，盒子里藏着的种子有时候是黑巧克力，有时候是白巧克力，是大自然的盲盒。

翅果菊
秋日常见野菊花，花浅黄色，乍一看有点像蒲公英，但植株比蒲公英高，气质上更加清新秀丽。果实也跟蒲公英很像，更显轻薄，自带“降落伞”飞行。

以上只是部分合肥比较常见的植物种子（果实），还有很多有趣好玩的种子也在秋天成熟，等待着大家去认识它们、探索它们，发现一个不一样的硕果累累的秋天。

# 秋天必做的15件小事，你做了几件？
秋天到了，你会因为遇到什么事情发出“秋天真好”的感叹呢？ 你会在秋天来临的时候做什么事情呢？

看一看闻啼鸟推荐的“秋天必做15件小事”，用一个个小事享受秋天吧！

观察昆虫最后的狂欢秋日的昆虫数量如同白昼时间一般，与日俱减。
在自然中寻找今年昆虫们最后的身影成为了秋日的寻宝游戏，夏日随处可见的蜻蜓、蝴蝶，现在都成为了秋日难得一见的空中宝石，一起来寻找最后飞舞的昆虫，与他们来告别吧。

拾秋果、打板栗、摘柿子
秋天最能带给人喜悦感的是它硕果累累的样子。

栾树枝头早早挂满了红彤彤的小灯笼，风一来，红灯笼吹落满地，它们“精巧得令人爱惜，成年人也不免捡了一个还要捡一个。”
还要去摘几个庭前屋后的柿子、打几颗山林里的板栗，总要把它们捧在自己的手心里，才算收获了属于自己的秋天。

体验一场秋收

春华秋实，秋天是收获的季节。 金黄的稻谷，挂在屋檐下成串的玉米，枝头红彤彤的柿子……现在的孩子们少有机会去田间劳作，在这个秋天，找机会下稻田看看，去菜地转转，爬上一棵果树亲手采摘，不仅可以体验劳作的辛苦与乐趣，更可感念收获的意义。

跟种子一起去旅行
种子成熟后还有很长的一段路要走，它们会用怎样的方式完成这段旅程呢？ 和种子一起来一场说走就走的秋日旅行吧，或许你也会长出“羽毛”，拥有翅膀，蹭个免费的车，又或者经历一场奇幻漂流记……

来一场叶子大战
落叶是秋天的印记，一场金黄灿烂的落叶雨，是大自然的美好馈赠。 一定要挑个晴朗的日子，在洋洋洒洒的落叶雨里尽情撒欢、肆意奔跑，开启一场告别秋天的落叶大战。

用自然创作把这一瞬的秋天留下

合肥的秋天很长，它从不经意间开始；合肥的秋天很短，秋天浓烈的色彩转瞬即逝。 在公园里，收集秋天的颜色和形状，把这一瞬的秋天留下。敲拓染、石膏植物拓染，都是可以自己尝试的自然手作的方式哦！

走进秋天的自然，就像走进一幅油画一样，色彩斑斓。一眼望过去，绿色、黄色、橘色、红色，互相交织在一起，非常的美丽。拿起油画笔，涂涂抹抹吧。

登高望远，自古便有的文娱活动。而徒步行走更是人与自然与自己相处的绝佳机会。

# 合肥秋日常见彩叶——切莫辜负五彩斑斓的大自然呀！  合肥常见 | 秋冬彩叶14种，看庐州的斑斓秋色
早在第一场略带寒意的秋风吹来时，植物们就早早收到了讯号，纷纷闻风而动、施展起各自的变身术。 一片又一片的叶子或橙或黄、亦红亦紫，于是，绿意盎然的合肥朝夕之间就变成了秋色迷人的庐州。

银杏
想必很多人脑海里出现的秋天的第一片黄叶当属银杏叶了吧。秋天的风，没准就是那一把把在枝头摇曳的小扇子招来的。 要是风再大一点，它们又变成穿着黄裙子在空中跳舞的小姑娘，跳累了就躺到大地的怀抱，层层叠叠铺成金地毯。

乌桕
乌桕带小尾巴的菱形叶子，凭一己之力集齐了秋天的红橙黄绿紫各色，是一棵老远就能引人注目、近点还能让人驻足的彩虹树，也是秋天最好看的树，没有之一。

槭（qi四）树
“枫树秋天叶儿红”，**通常我们把那些像手掌一样分裂、入秋叶色变红的叶子都称为枫叶**。 合肥的枫树主要分为两大类，其中担当主力军的那一类大名叫做槭树。槭树区别于其他枫树的最明显特征是翅果。

合肥常见的槭树有三角槭、鸡爪槭及其变种红枫，以及难以区分的元宝槭和五角枫。这些槭树名字里面的“槭”字通常也可以替换为“枫”。 于是在合肥的秋冬时节你能看到以下各种枫叶。

生产人类快乐源泉的糖槭，据说是加拿大国旗上的那片“枫叶”。

枫香
枫香树可能是最接近中国古代诗歌作品里“枫”这个意象的。 

不同于槭树的“枫叶”有各种各样的的分裂方式，枫香的叶子只有3个角，而且名副其实有香味。秋天变红之后足以与槭树家族的三角槭媲美，除了揉碎叶子闻一闻，看果实也是一种直观的区分方式。

木瓜
先说明一下，这里的木瓜树指的是我们中国本土的木瓜，也是《诗经》里的木瓜。而我们平常吃的水果木瓜其实叫番木瓜（外来的），是两个完全不同的物种。
木瓜树有迷彩色的光滑树皮，叶子在秋天也会变成暖暖的南瓜色。

杉树
杉树常见于水边，是合肥各个公园常见的水生植物。常见种类有水杉、池杉、落羽杉。 

通常看到的杉树都是成排或成林种植的，树高且直，秋季叶色变成深深浅浅的黄色或红褐色，还有悬在枝头的球果，景色蔚为壮观。

池杉的叶子，看气质略显凌乱；水杉的小叶在侧生小枝上呈羽毛状排列，小枝（近）对生；落羽杉的叶子看起来跟水杉很像（你能发现它们的区别吗？），但其实跟池杉的亲缘关系更近。

黄连木
黄连木是一种高大挺拔的树木，树叶羽毛状排列，秋季远望树冠，红彤彤一团，近看叶色更加丰富多彩。

如此五光十色的黄连木，可不是哑巴吃了心里苦的那个黄连。

悬铃木
就是马路边那个悬挂着“小铃铛”的树，俗称法国梧桐，但其实跟梧桐没啥关系，只是因为叶子长得像而已。

著名行道树的秋色是各种各样的黄，明黄、棕黄、大地棕……

鹅掌楸
鹅掌楸在秋天还有另外一个名字——马褂木，是个很会穿搭的树。经历了春生夏长，鹅掌楸把清新的绿T恤换成了温暖的黄马褂，有的黄马褂还有时髦的“耸肩”，这是大自然别出心裁的服装设计。

重阳木
重阳木的三小叶到了秋天像是被染色了一样，至于染的是什么颜色，全看大自然的心情。

无患木
无患子是合肥常见的栽培树种，西扩公园有大片的无患子林。 

它的叶子像一片大羽毛，经常一棵树的叶子还是翠绿欲滴，旁边一棵就已经变黄，或者一棵树上的叶子半绿半黄。在秋冬季节的一众红黄叶里，能在大蜀山脚下欣赏到这样清新亮丽的景色，也非常悦目。

以上是合肥秋日常见的一些彩叶树种，有的常见于园林栽培，有的广布于山野乡村，它们一起编织着秋天的五光十色。天气晴好的日子，记得要出门去欣赏这五彩缤纷的庐州秋色呀！

蜻蜓泛指昆虫纲蜻蜓目的所有昆虫，是迄今最古老的昆虫类群之一。古代蜻蜓有的种类体型巨大，翅展能达到1m。
现生的蜻蜓可分成三类：第一类俗称蜻蜓，属差翅亚目；第二类俗称豆娘，属束翅亚目，又称均翅亚目；第三类俗称昔蜓，属间翅亚目，十分罕见，目前仅发现三种，身似蜻蜓，翅似豆娘，集合了其他两类的特征。

《淮南子·说林训》记载“水虿为蟌，孑孓为蟁，兔啮为螚。”指水虿变成了蜻蜓，孑孓变成了蚊子，兔啮变成了苍蝇。说明古人很早就观察到了昆虫的变态现象。
蟌如今被昆虫学家作为豆娘的正式名称使用，有别于蜻蜓。而水虿则作为蜻蜓目下所有种类的稚虫的俗名被保留下来。

正如古人所述，蜻蜓是一种半变态的昆虫，稚虫的生活方式和成虫差异较大。

水虿生活在各种淡水环境中，潜伏于泥沙和水草之中，是凶猛的水下杀手。小到蚊子的稚虫孑孓，大到各种小鱼小虾，都可以成为它们的食物。
水虿的一生中要经历多次蜕皮（9-15次），才能彻底羽化为展翅飞翔的成虫。

羽化是它最脆弱的阶段，若无法顺利脱离稚虫的外壳，新生的蜻蜓将困死在自己的旧衣之中。刚羽化的蜻蜓身体还十分柔软，色彩白淡，还需要时间经历身体干燥和外骨骼硬化，逐渐变得色彩艳丽。


蜻蜓的心形交尾法则

水虿会在水中蛰伏长达一到几年之久，而成熟的蜻蜓寿命仅有几周至数月。它们唯一的任务就是完成繁殖，开启生命的下一次轮回。
那么，蜻蜓的繁殖有哪些特点呢？
雄性蜻蜓的外生殖器位于腹部第2、3节下方，腹部末端具有肛附器；雌性蜻蜓的腹部末端则为产卵器。

这样独特的身体结构，也决定了交尾时奇特的动作：
据说雄性蜻蜓的控制欲很强，首先它会用自己的肛附器钳住雌性的后头或前胸；而雌性需要将自己的腹部末端弯到身体前方，与雄性的外生殖接触，完成受精。
整个环连结起来就像一个爱心形。

交尾可以在空中进行，也可以停歇时进行。交尾完成后，雌性会将卵产在水中、泥沙、水藻中等地方，产卵方式除了常说的“蜻蜓点水”，也有插秧、空投、潜水等形式。雄性通常会护卫在雌性身旁，或是交尾结束保持和雌性连结，以此保证产卵的顺利。

# 合肥常见鸟类 | 我们身边的鸟儿邻居
如果有人问你：我们生活的周边有哪些鸟儿？你的脑海里会闪过哪些答案或者画面呢？
大部分人脱口而出的答案可能是“麻雀”，或者再多个“喜鹊”、“鸽子”……
很多人都不曾留意过我们身边的鸟类，认为想看到更多的鸟就得到动物园，或者离开城市，到山里到更野的地方去。

其实，在我们的生活周边，就有数量众多且多种多样的鸟儿与我们共同生活在这个环境中，今天，就来给大家介绍几种，在咱们合肥小区、校园、路边绿化带等地方，都非常常见的几种鸟类邻居。

第一个要介绍的就是这种蓝灰相间的鸟儿，它的体型不小（身长33-38厘米），数量很多，而且不太怕人，所以只要你对身边的鸟儿稍有留意，就很有可能见过它们。

它身上淡淡的蓝色很像这几年流行的“雾霾蓝”，给人带来一种高级感，非常漂亮。
它的名字叫“灰喜鹊”，听起来挺普通，但它跟我们安徽关系密切。

因为在安徽比较常见，又长得好看，再加上名字里的第一个字的发音与咱们“安徽”的“徽”字同音，所以被选定为安徽省省鸟。
它们常常成群活动，胆子比较大，树林、草地、路边都有可能看到它们，是咱们在合肥观察鸟类入门时的理想观察对象。

# 给喜欢摸鱼捉虾捞蝌蚪的大小朋友们的几条建议
先把建议写在开头，其实是几个问自己的问题，这些问题不仅仅是在培养我们/孩子对于大自然关心和保护的能力，也是在让我们自己/孩子去练习对环境、对生命、对自己的行为更负责的过程。详细内容在文章的最后。
第一问：抄网是非常好的自然观察工具，捞出水生生物做观察也是常用的自然观察方式。但在观察的时候 ，你可以尽量减少对它的伤害吗？
第二问：如果很想带走，那么先问一问自己：我为什么想要带走？是否只有带走这一种方式才能满足我的目的？还有没有其他的方法？
第三问：如果你确定要通过带走的方式才能满足自己的需要，那再问问自己：我是否了解它的生活习性，我是否有一定的把握能够让它们在我的照顾下，健康生长？
第四问：如果你也确定自己可以好好照顾它们，那再问问自己需要带走多少？请尽量少地带走。
第五问：最后再问问自己，是准备一直养下去吗？还是到了某个阶段就可以放回了？

今天一早去南艳湖，在公园里几乎呆了一整天。早上的人虽不算多，但也明显比往日要热闹，预示着今天一天南艳湖的人流量。

我照例前往每日观察小蝌蚪的那片小池塘，自从4月1日蝌蚪出生（形成了蝌蚪的形状并且可以自主活动）以来，这几天它们都没有什么太大的变化。

3月28日 我第一次发现它们；4月1日，小蝌蚪出来了。

然而，我观察完准备离开时，看到一位妈妈带着孩子来到这里，带着大大长长的抄网，说准备在这里钓龙虾。
临走时，我心里隐约有些忐忑，“今天来南艳湖的各个小池塘里捞东西的孩子估计不少，这些小蝌蚪会不会都被带走呀？”但心里还是觉得这个小池塘挺偏的，估计不会有几个人到这里玩。

接下来说两件小事。
第一件事：我路过一个池塘，遇到一个10岁左右的孩子大声问他的同伴：“这个网你们还要吗？” 

“不要了，你就让它沉入池塘底部吧。”远处的一个孩子回应道。
“啥？就这样高调地要把废弃的东西扔进池塘吗？”我心想孩子们不至于觉得乱扔垃圾这件事可以这么高调地在公共场合说出来吧。
可接下来，那个拿着抄网的孩子真的把抄网往水塘里一丢，然后说了句：“我丢进去了。” 

周围有很多人，我心里犹豫了一下这地方也不归我管，我要不要干预这件事。可脑海里快速闪现一个声音“如果我作为一名自然教育老师，都不能坚定地去制止这样的行为，还指望谁出来说话？”

“嘿，就这样随便把东西扔到水里吗？”我在远处大声对那个孩子喊道，然后走上前。孩子显然被我吓到了，站在那里一动不动，我走到他面前，跟他说把抄网捡回来。但抄网扔得有些远，他显然够不到。我说“我拉着你，你来试试。”还是够不到，他说他去找找树枝，用树枝给捞回来。

这时我才回头看刚刚给他出主意的同伴，吓得在往远处跑，我冲他喊了一句，但具体喊得啥我也不记得了，大概意思是“怎么能给朋友出这样的主意？”

这个小男孩在周围没找到树枝，然后跟我说：“叔叔，要不用你的抄网把它够回来吧？”这时我才想到我的背包里有个抄网。于是我把相机放在一边，放下背包，取出抄网，伸长杆子，一点一点把他的抄网够了回来。此时，这男孩的几个朋友也都围在我旁边了。

“这个抄网应该怎么处理呢？”男孩问我。

“如果你真的不想要了，就找个垃圾桶丢到垃圾桶里，别把人类制造的东西留在自然界。”说完，我就开始收拾东西，几个孩子也就拿着抄网走了。

第二件事：我在公园看了一下午书，然后又去找了找刚刚出生的小鸭子，没想到一到湖边就遇到了它们，十几只小鸭子在落日的余晖下，太可爱了。不过湖边游客这时候还是不少，一些兴奋的家长和孩子显然吓到了这群宝宝，它们快速地游到可以遮挡的水丛中，甚至超到了妈妈前面。

此时我又想起了那群蝌蚪，心想这一天了，也不知道它们到底怎么样，还是去看看吧。

我来到那群蝌蚪的小水塘边，远处就看到还有两三家正围在水塘边，正在收拾东西准备离开。走近点，我看到水塘周围全都是水，水草破败很明显，估计今天一天这个小水塘周围都有人在打捞，我的心里咯噔一下：“蝌蚪不会全被捞走了吧？”

我走到自己前几天连续观察的几处有卵的地方，无一例外，都没了。旁边一位爸爸对女儿说：“你拿好你自己的战利品。”这个水塘边人们活动的痕迹很大，甚至还留下了钓龙虾用的鸡内脏，但小蝌蚪几乎没了。

我试图在水面上仔细找找，这个水面上的几处产卵点，加起来可能有几千只小蝌蚪呢，这一天的功夫，全都没了吗？我捡起一根水草杆，在水面上搅动搅动，几只小蝌蚪游动了起来，然后向水塘中间游去。我的心里稍微有了一些安慰。

希望大部分蝌蚪是在受到惊扰的时候逃走了，而不是都被孩子们带回家了。可我也清楚，从1号到3号，它们聚集在一起，几乎一动不动，想一窝端，实在太容易了。

这些小蝌蚪是黑斑侧褶蛙的宝宝，是我们国家的三有保护动物，保护级别并不高。从对生态影响的角度而言，没有可靠数据能说明这一群蝌蚪被带走，到底会有多大影响，以及在蝌蚪长大过程中，它们能顺利成年的概率也是很低的。也就是说，这片池塘里的大部分蝌蚪都可能在变成青蛙前死亡。

但是黑斑蛙一年只会产卵一次，今天这一天，几只蛙妈妈的宝宝就几乎全都被人类带走了。

我知道带一些小生物回家养，是很多小朋友喜欢的事，这也确实是观察了解一类生物的一种很棒的方式，但这样“一窝端”的行为，实在令我有些伤心无奈。

在此，我想向喜欢在野外打捞、观察、养殖水生生物的家长和小朋友们提些建议：

建议一： 抄网是非常好的自然观察工具，捞出水生生物做观察也是常用的自然观察方式。但在观察时，注意不要伤害你喜爱的这些小生物。

例如鱼类、蝌蚪离水时间不要太长，在拿起小昆虫观察时注意力气别太大。观察完后，尽量放归回它们原先生活的环境。

建议二：如果很想带走，那么先问问自己：我为啥想带走？是否只有带走这一种方式才能满足我的目的？还有无其他方法？

最开始发现蛙卵的时候，我也考虑着想带些回去观察，我带走是为了能观察到它的生长过程，这是一次难得的机会。但转念一想，我来南艳湖也很方便，我完全可以保证每天都来一趟记录它们的变化，这样也能满足我观察它们生长过程的需要，而且是在更自然的环境下，那我还是不带走了吧。

而我跟同事说了这群小蝌蚪的事之后，同事表示如果带回来，可以做更高频率更细致的观察和记录。我便再次考虑带回来，不过因为这群蝌蚪当时都已经变成蝌蚪的模样了，没有找到刚生出的卵，所以也就再次没有选择带回来。

如果你只是一时兴起，甚至只是为了满足自己的占有欲，就像“战利品”一样对待这些生命，那希望你能对这些小生命多些尊重，让它健健康康的在大自然里长大，是对它们而言最好的方式，请你一只都别带走。

建议三：如果你确定要通过带走的方式才能满足自己的需要，那再问问自己：我是否了解它的生活习性，我是否有一定的把握能够让它们在我的照顾下，健康生长？如果你不知道该如何照顾它们，那你可以抓紧在野外的这段时间好好观察，在临走时全部放回。

就像之前我考虑带蝌蚪回来时，也同时我们考虑了办公室的水缸是否有条件满足我们养蝌蚪这件事。

建议四：如果你也确定自己可以好好照顾它们，那再问问自己需要带走多少？请尽量少地带走。

也许你有足够的理由带走它们，但你一定要明白，带走的越多，对这个小生态的影响就越大。如果我们是为了观察，加上可能自然死亡的概率，是不是带走几只也就可以了？

前两天我也是在这片小池塘捞出几只自己之前没见过的虾，于是想带回办公室观察，当时一共有5只，我选择了3只体型差距比较大的虾带回了办公室，想看看体型不同的这几种虾，会不会在生活上有什么差异，是不是成长的不同阶段。

建议五：最后再问问自己，是准备一直养下去吗？还是到了某个阶段就可以放回了？例如这次的小蝌蚪，如果自己的主要目的是观察它的生长过程的话，那么等它变成青蛙后，当然就可以选择放回自然界。

以上几个建议，其实都是问自己的问题。这些问题不仅仅是在培养我们/孩子对大自然关心和保护的能力，也是在让我们自己/孩子去练习对环境、对生命、对自己的行为更负责的过程。

以上建议对于小龙虾、巴西龟、牛蛙、福寿螺等外来入侵物种除外，这些物种在合肥遇到，可以放心抓回。


晚上想想今天在南艳湖遇到的这两件事，感慨其实大家是多么喜爱在大自然里探索、玩耍呀，但我们又是多么地不了解大自然、以及该如何跟大自然相处。

两件事反映的其实都是多年来的普遍现象：我们走进自然，喜欢留下自己带来的东西，喜欢拿走原本属于自然的东西。我们依然对大自然、对生命缺乏敬畏。

作为一名自然教育从业者，看到这样的现象依然如此频繁，确实会深感无奈。我们要做的工作还有很多。

清明假期还有两天，合肥的各个小水塘一定还会围满了人，欢迎大家把这些建议分享给喜欢在小池塘捞鱼捉虾抓蝌蚪的大小朋友们，希望在享受自然乐趣的同时，尽量降低对自然的伤害。

# 成为一名优秀自然教育导师并不容易，你准备加入么？
这两年，常会有人问到我们——闻啼鸟还需要导师吗？成为闻啼鸟的自然导师需要具有哪些能力呀？要学习哪些东西才能成为一名自然教育导师？要学习多久才能带课呢？

自然教育是一个美好的行业。从外界看来，作为一名自然教育导师，既可以到处游山玩水，还可以传递保护自然的理念，更是在帮助孩子拓展成长的可能性、促进儿童的综合全面成长。这实在是一份既有趣又有意义的职业，所以也不断有人问闻啼鸟是否还会招老师。

我们当然缺老师，虽然只是一个看似很小的自然教育领域，但闻啼鸟想到的可以做的方向也有很多，仅限目前的几个课程模式，我们现有的老师能够覆盖的人群也非常低，所以如果有更多有能力做这件事的人加入，我们当然开心。

但对于闻啼鸟而言，遇见或培养一名优秀的自然教育导师需要面对几个层面的hard，而对一个想要从事这个职业方向的个体而言，这几层hard也恰恰是需要去持续成长突破的。

有一定的自然知识和真实的自然经验积累是第一层hard。

作为一名自然教育导师，具有一定的自然知识和真实的自然经验积累非常重要。自然教育发生的场景在真实的大自然中，教学的内容主要是关于真实的大自然，从传统的教学元素来看，相当于自然教育的教室和教材都是大自然。有些人可能会建议招聘生物科学、地理科学相关专业的学生，全国培养这些专业的学生每年也有很多，他们的就业现状好像也不怎么样，来做自然教育不是正好？

这些专业确实是跟自然教育强关联的，但是这些专业出来的大学生，未必就真的了解真实大自然。

我自己本人大学的专业与地理相关，自己当时选择这个专业也是因为兴趣，但是大学的教学是大教室，老师在台上枯燥的讲授，实际的野外实践只有一次，去的就是我现在的工作中常带孩子去的巢湖，当时人数众多、时间很短、老师能够教授的内容也极其有限，虽然我印象深刻，但实际上也并没啥收获，而我在大学时期，已经算是一个非常积极主动学习本学科知识的学生了。

现在的大部分学生也面临同样的问题，大学里学到的知识依然以理论为主，即便学习的是自然科学，但学生与真实的大自然也很远。所以即使学习了生物专业，他们可能对生物分类的基础理论、生态学的基本概念有过接触甚至是考过背过，但身边常见的花花草草小鸟虫子，也很可能从没观察过。

对于在大自然中的生活和工作状态很可能是排斥的，在野外的各种情况可能是毫无经验判断和不会处理的。

如果研究生阶段的研究与跟动物行为、生态变化、濒危物种等方向相关，可能会多一些一线实践的野外经验和真实的观察积累，但在整个自然科学领域，这个方向依然小众。更多学习生物专业的学生，可能会选择微生物、细胞、基因等方向，毕竟这些方向会更“有用”、有更多资源支持、未来的就业也更好，但这些方向的学习研究可能更多会在实验室发生。

不否认现有的社会培养路径，确实培养了一些自然功底扎实、野外经验也丰富的人成为了自然教育导师，很多都非常优秀。但整体而言，这个路径培养出的具有一定的自然知识和真实的自然经验积累的人并没有那么多。

而从“非专业”人士的成长路径来看，无论因为童年还是工作，有在乡村、野外成长生活经验的人通常具有更丰富的自然经验，但即便是当下，自然科普在社会上也依然稀缺，所以他们在成长过程中通常很难接触到现代自然科学，整体的知识体系是欠缺的。

有教育同行曾问我，既然了解自然知识本身并不是闻啼鸟自然教育的目的，那没有自然积累的老师就不可以成为你们的老师吗？确实，一位教育实践经验丰富的老师，在大自然这个场域中也完全可以创造一个与自然相关的优质学习体验，但这位老师也一定与自然有深度连接，这个连接本身就非常难得，就是大部分人所缺失的，这样的连接不是凭空就有的，一定是在长期跟自然的互动中产生的。

缺失了自然知识和自然经验积累的自然教育老师，就像一位不会画画的美术老师想通过教授绘画的方式展示孩子的创意提升孩子的视觉表达、一位不会小提琴的音乐老师想通过教授小提琴的方式让孩子感受音乐的魅力，会失去最重要的抓手，不可能做出有持续性的、好的自然教育课程，也不可能真正把自然的魅力传授给学生。

闻啼鸟的自然教育，是建立在对真实大自然观察的基础上的自然教育。


有对自然和生命的尊重敬畏，有符合生态伦理的自然观（或叫生态观）是第二层hard。

具有自然知识或自然经验的人，未必能有对自然和生命最起码的敬畏。例如以野生动物违法犯罪行为为例，做这些事的人，很多都具有丰富的野外经验或与野生动物打交道的经验。盗猎者对某类濒危野生动物及其生境的熟悉程度，可能不亚于保护这类物种的巡护员；

贩卖某些非法野生动物的人对于动物的养殖救助经验，可能不亚于专业救助机构的工作人员…… 大自然在他们眼里，可能只是能够生财的商品而已。

再例如一些户外探险爱好者。他们可能走过很多荒野、探索过很多少有人至的自然环境，在野外的生存技能也是经验丰富。但探索的过程却给自然留下很多伤痕，甚至以捕捉野生动物这样的野外生存技能为荣，大自然在他们眼里可能也只是寻求刺激的平台而已。

而在我们的很多专业学科人士的教学中，就算真的有机会到大自然中实践，一株植物、一只昆虫，也是随意采摘、捕捉、带回的，自然中的这些生命，不过是研究的对象而已。我不是反对教学和科研中的采集，事实上采集确实也对我们了解一个物种起到了很重要的作用。

但这些行为，在我们教育中有可能太随意了，在行为发生前，可能没有基本的生态伦理教学，如果简单粗暴的开展这样的行为，对学习者而言，是不可能建立起对一个生命应有的尊重的。

事实上，整个人文关怀和通识教育在我们的教育中都很缺失，我们正是在这样的环境中成长起来。而专业教育里所谓的专业划分，所谓的自然科学，是让学习者只能通过所谓理科的思维去看待大自然。但其实，大自然中是发生生命教育的重要场所，然而很多学习自然专业的人，在学习过程中却可能接受不到任何对生命的尊重。

这就导致在这个领域掌握的知识很深入的人，却也同样因自己的“热爱”在不断伤害大自然。例如打着擦边球在野外采集大量未被法律保护甚至已经在保护范围内的动植物。我们曾见过自然经验非常丰富的初中生，到了野外会抓蛇带走，见到蝴蝶就是一网，如果发现比较“珍贵”就决定带走作为自己的收藏。

如果带着这样的态度成为一名自然教育导师，在博学方面，他可能会收获大量粉丝，有很多家长孩子都很崇拜他，但他在教学中的行为，他的自然观也一定会在影响每个孩子。我曾遇见一位非常博学的老师，发现一条蛇后，使劲把蛇从躲藏的石头缝中拔出，然后用夹子把这条小蛇当猎物一样介绍它的生物学特性，学员在旁吓得哇哇大叫他反而更兴奋。

虽然最后放归了，但蛇本身在抓捕的过程中受了严重的伤，且过程中，完全感受不到对这个生命一点点的尊重。同样，讲解一株植物时，是选择毫无顾忌地连根拔起讲解，还是引导孩子蹲下甚至趴下仔细观察；在带孩子观察鸟类时是觉得种类数量越多越好，还是建立在减少干扰的前提下，更加随缘、更重视持续耐心的观察，可能都能体现出一个导师的自然观。

闻啼鸟与孩子在大蜀山夜间观察 

闻啼鸟也同样会有自然物的采集，也同样会有昆虫或植物标本的制作。但采集的前提、采集的目的、采集的限度、标本的来源等等跟生态伦理相关话题的探讨或强调，是我们课程中极其重要的一部分，甚至在一定程度上，比采集、制作本身更重要。

相比于自然知识，了解自然物之间的连接以及与人类的关系，在大自然懂得尊重生命，欣赏自然甚至改变自己的行为，走向更加自然友好的生活方式，是我们更看重的。


在教学的过程中会跟人互动，并能看到的人的成长，是第三层hard。

以上两层hard都与大自然相关，在这两个层面上有所积累、持续思考的人就比较稀有了。但做自然教育，毕竟是一个教育行为，是需要设计课程，需要与人互动的。就像我们常说的例子，数学学得好的人不一定能教得好，打球好的运动员不一定能成为一个教练员。自然教育也面临这样的问题。

单从科普的角度讲，现在的社会，很多领域其实都不缺专家，尤其是面向孩子所需要了解的基础层面，知识储备够、足够专业的人就更多了，但他们却不一定会做科普。自然教育中这样的案例也很多，也许从某个高校请的一位古生物学家来讲古生物科普，孩子们听两句就没兴趣了，而换做一个自然教育导师来讲，孩子们却可能聚精会神充满兴趣。

我几年前做讲师培训时也曾邀请过一个高校很有名的鸟类专家来代课，因为挺认可他的专业知识和生态观。可结果却是，专家只会自己找鸟拍鸟，讲解教学都说不出几个字，一群学员在旁边懵懵得不知道专家在干嘛。

所以会教学这个门槛，对于以上两个层面hard筛选出来的人，又会拦住很多人。而就像我在第一个层面就有提到的，我们的自然教育活动不仅仅在传授自然知识、自然技能这个层面。同时，也不仅仅是单向地“保护自然、爱护自然”这个价值观的输出。

闻啼鸟非常看重这个过程中人本身的成长，期待在大自然中培养孩子成为独立且自由的人（推荐阅读：《为什么大自然是培养自由独立个体的绝佳场所？》），而这就需要导师在教育学、心理学等层面上，也同样有兴趣，并且有自己的觉察和积累，要在对人的好奇层面上，不亚于对大自然的好奇。（推荐阅读：《做自然教育，不仅要对大自然好奇，还要对人好奇》）

这个要求在很多其它的教育领域也很稀缺。其实我觉得做自然教育的人，相对于很多传统教育领域，对人的关注和好奇比例就是比较高的，这可能也跟大自然本身的疗愈性相关。但基数本就不大，稀缺也是同样的。

能够面对、接受并运用好自然的不确定性，是第四层hard。

这一层面的对比，主要是跟其它教育领域相对比而言的，同时也跟第一层面的hard相呼应，有足够自然知识积累和真实自然经验的人，接纳和处理这一层hard就会容易些。

自然教育的吸引力，很大层面来自于真实的大自然太丰富了，这其中有太多有趣的、可以玩的、可以学习的元素，而且时时刻刻都在变化。但这也同样带来了自然教育的复制性弱、需要面对的不确定性因素大大增加。

相对于很多在室内或网络上进行的教学，整个场域的大部分元素都是人为制造的，虽然并不一定是为了课程而刻意制造，但其丰富度和不可控性大大小于户外。但在一个开放的户外空间，尤其是我们更强调一个丰富的人为干扰尽量小一些的大自然中，其不可预知的可能会对教学产生的影响因素就多了很多。

温度是多少、会不会下雨、风有多大、有没有雾霾……这些人不可控的因素都会有影响，而除了这些课前也许可以预知，提前做准备的因素，也许在讲解某个植物后，突然有一只鸟儿飞得很近在旁边歌唱吸引了孩子的注意力、也许在课程要开始的时候发现今天课程上准备玩的落叶，公园的工作人员正在清扫、也许想带孩子寻找的鸟儿今天就是不愿意出现……

这都让教学的过程，变得更需要随机应变。而与此同时，闻啼鸟的课程对孩子们也是保有比较开放的态度，很少用纪律或强压的方式来方便管理，孩子的不确定性也就比整齐划一的教学中更高。学习者和环境本身两方面的不确定性在一起，这不是一个相加的关系，而是相乘的关系。一名优秀的自然教育导师，需要面对、接纳这一切，同时也要利用这样的不确定性创造更多更有魅力的学习体验。

成为一名优秀的自然教育导师需要面临的hard还有很多。以上几个层面，是我们对于闻啼鸟想做的自然教育梳理出的几层比较重要的对导师的要求，而这样的要求也成为了我们遇见匹配的导师的几层hard。可能很多人也并不认为做一名自然教育导师需要有这么多层面的要求，那可能是因为对“自然教育”这件事情本身有着不同的理解。


闻啼鸟已经快4年没对外招聘了，我们断断续续在做一些人才培养的事，在这个过程中成长起来并确认自己想要从事这个行业的志愿者和学员陆续加入到闻啼鸟。但作为一名自然教育导师的成长之路依然是漫长的，闻啼鸟也不可能完全依靠内部培养作为导师的唯一来源。

今天也是一个特别的日子，到今天，闻啼鸟正好走过了五年，在这一天我们再次开启主动地对外招聘。

闻啼鸟致力于陪伴、引导孩子/家庭在大自然中健康成长，探索并认识世界的美好和多样，建立人与自然的情感连接，在这个过程中，培养独立自由且具有环境素养的地球公民。

如果你认可这件事，同时也认为自己具备上文所提到的对于一名自然教育导师的能力要求，并愿意在这几个层面与我们一起持续学习、共同成长，那么欢迎你加入闻啼鸟，与孩子们一起，在大自然中自由奔跑！我们的主要工作地点在安徽合肥，节假日可能会各地奔走。

# 行业发声：关于夜间观察活动的行业倡议和经验分享
又一年夏季到来，自然界里的各位小虫子开始越来越活跃，青蛙、壁虎等生物也都复苏，开始了它们新一年的生活，尤其是在夜间。在夏夜里走进自然，寻找、发现、观察这些小生物是非常有趣、非常吸引人的一种亲近自然的方式，也是我们了解黑夜大自然、换一个角度认识黑夜的一种方式。

在夜间观察的黑夜探秘小勇士们  大羊摄

闻啼鸟自然教育工作室是合肥最早开设这类夜间观察课程的机构之一。早在2016年，闻啼鸟正式创立前，创始人大羊老师便在大蜀山开设“夜探大蜀山”，这一个以夜间生物观察为主要内容的课程，并在2017年闻啼鸟创立后成为闻啼鸟的重要品牌活动。

近年来，越来越多的家庭接触到自然教育，乐意带孩子了解大自然、在自然中成长的家庭也越来越多，随着这个需求的增加，合肥做自然教育的团队和个人也逐渐增加。夜间观察非常受孩子欢迎，也有越来越多的机构团体或个人开始开设这一活动。

早些年闻啼鸟在大蜀山开设活动时，只有我们一个小队伍在一个少有人关注的路线做观察，而现在却经常遇见其他组织。

有更多人带领孩子了解大自然、感受夜间探索的乐趣是值得欣喜的事，虽然与其他团队“相撞”的情况确实给我们自身开展活动带来了更多挑战，但我们会尽量避免与其他夜观队伍的相遇，有时也会询问对方的时间和线路安排，提前打好招呼、尽量减少相互干扰。

我们也偶尔听说一些其他团队做的夜观活动中，有些理念与闻啼鸟提倡的不相符的部分，本着做好自己认可的事就能影响越来越多人的态度，伙伴们之间主要都是私下交流。

但近两年，我们越来越高频地亲眼目睹个别老师带领孩子夜间观察的行为和言语，实在是有悖自然教育行业的基础共识（“尊重自然。进而减少对野生生物和生态环境的影响，不以自然教育的名义，伤害野生生物，破坏生态环境。”——摘自《自然教育行业自律公约》），令闻啼鸟的伙伴们痛心、愤怒。

说两个案例吧：

案例一
对话1：老师：“有没有小朋友想养蜗牛的？”小朋友：“我！我！我……”纷纷举手。老师：“不要挤，大蜀山有很多，每个人都能带回去养！”

对话2：老师：“这是XXX的蛹，大蜀山这个季节有很多蝴蝶和蛾子的蛹，大家想养的可以找了带回去啊。”

这位老师带的活动我们这两年常遇到，基本都是人很多的一个队伍，到了时间老师就直接带领孩子打着手电开始找东西观察讲解，期间老师的嗓门很大，孩子一挤一乱老师就会怒斥孩子，有一次我们老师在现场还听到有家长在活动中骨折了。

案例二
案例二去年7月，我们在大蜀山遇到一位老师，也是认识的物种很多、讲解声音很大（也许与案例一是同一位），期间也听到了很多他对孩子言语上的否定或呵斥，不过更让我们惊讶的还是他发现一条赤练蛇后的行为。

这位老师发现蛇的时候，蛇应该已经在逃走了，但老师反应很快，立刻用专门捕蛇的镊子抓住了蛇，可此时这条赤练蛇应该是钻到了某个石头下面，这位老师花了点时间硬拽也没拽出来，后来他好像是调整了姿势还是用另一只手搬开了石头，才终于把这只不大的赤练蛇抓住了。

我们隐约听到该老师说“蛇都受伤流血了……”但他的语气感觉依然是洋洋得意、很有成就感的样子。他用镊子夹着那条蛇的尾部，举在学生和家长面前，开始讲解赤练蛇的一些相关知识，讲着讲着，他突然把蛇往学生面前一扔，吓得一些学生和家长尖叫连连。随后他又继续讲了些啥，最后把蛇扔回了树林中。

在这些年的夜观中，我们遇见几次让孩子捕捉生物带回家的老师，我们听到或看到的带回去的生物就包含各种昆虫（包括各类甲虫、毛毛虫、萤火虫等）、蜈蚣（明确说可以做中药卖）、宁波滑蜥、赤练蛇（有孩子捉了说回去喂自己养的眼镜王蛇）等，而从各方听说的，有些机构在活动过程中对生物和环境造成比较大干扰的事也不少。

闻啼鸟是合肥最早开设夜观、并促成夜间观察这类活动在合肥被市场认可的组织之一，在看到了多次行业中对自然、对孩子不负责任的行为后，我们有必要发出声音、提出建议，以及在此分享一些已有的经验。

我们期待合肥以及全国的自然教育行业都能良性发展，减少因个别从业者不负责任的行为，造成家庭、学校和整个社会对自然教育行业的误解、影响对自然教育的认可和信心。


关于对野生动物的捕捉带走
01、闻啼鸟建议活动中不对野生动物捕捉带走，或慎重捕捉带走这类行为的发生。

02、我们认可在科研活动或了解一个物种的过程中，可能需要通过捕捉、养殖观察、或制作标本等方式进行，但我们需要思考：在一个面向普通大众、尤其是孩子的短时间活动而言，是否有必要通过这样的方式来了解自然？

03、我们认为在野外的观察，更重要的是感受自然界真实的样子、观察生物在自然界的真实状态，这些都无需通过捕捉带走就可实现。对更多普通大众、尤其是孩子而言，认识自然的多样和精彩、学会尊重生命是更重要的，有了友善的自然观、生态观，即便未来在探索学习的过程中有捕捉带走的需要，也会自发地克制，控制捕捉带走的量。

04、如果在活动目标设定中，需要通过捕捉带走行为来实现，请评估这个行为带给受众的影响，是否做到了尽量少地带走？还是让受众觉得自然界的动物是可以任意带走的？此处与闻啼鸟此前发布的《给喜欢摸鱼捉虾捞蝌蚪的大小朋友们的几条建议》类似，需要我们评估每个行为背后的意义，如何做到尽量少地对自然的干扰。

05、如果仅仅需要捕捉而非带走就可以实现活动目标（例如观察一些跑得比较快、不利于观察的虫子、或观察一些水生生物等），请注意在捕捉过程中减少对它们的伤害，以及在观察完、或活动结束后放归自然界，并足够重视这个动作，作为活动设计中必有的一部分，让参与者不仅仅看到了捕捉、也看到了放归。

作为组织方，我们对待自然的每个小的行为，都可能对参与者起到示范作用，我们需要慎重对待自己有可能引起误导的行为，作为老师的角色面向学生时，这一点会更重要。

活动中考量捕捉或带走行为对受众的影响需要带领者的精心设计和丰富经验。而对于初次夜观、或通过类似方式探索大自然的受众而言，对一些低年龄的孩子而言，在短短的一次活动中，通常都很难传递那么那么深的观念和思考，在这种情况下，活动中的捕捉行为，就更需要慎重评估。

07、对于在活动中没有任何铺垫、提醒，甚至直接鼓励孩子们捕捉野生动物带回家的行为，闻啼鸟明确反对。甚至有些捕捉的动物属于法律保护的野生动物，这样的行为可能直接触犯了法律，这样的教学或解说方式，对受众和大自然都是极其不负责任的。

关于使用灯光01、灯光对于夜行性生物确实会产生影响，但想要对夜间生物进行观察大都避免不了灯光，我们可以选择根据不同的情况，尽量减少灯光的干扰。

02、探索和发现本身就是自然观察的乐趣之一，自主发现的过程也是自然观察带给孩子的重要成长方式，发现一个小动物能给孩子带来成就感、提高孩子对自然的兴趣，对于成人其实也是如此。在鼓励自主发现的夜间活动中，可以给每位孩子发手电筒，鼓励自主寻找发现。

03、在集中讲解的环节，可以让孩子关闭手电筒，一方面可以避免多个手电照射同一生物带来的强光打扰，另一方面也有助于孩子的注意力集中在老师照射的物体上，专注老师解说的内容。

04、不同的生物对光线的敏感程度不同，例如部分昆虫，光线的照射可能对它没有什么打扰，我们可以更长时间的观察，可对于某些蛙类，光线的照射会有较大影响，我们便需要更加留意光线照射的强度和照射的时间，而且这些内容本身就可以成为教学或解说的一部分。

05、不同环境和寻找的对象不同，对于光线强弱的需求可能也不尽相同，例如我们过去在婆罗洲热带雨林里夜观时，在偌大的雨林中，会携带光线更强、覆盖范围更广的手电，而在城市公园的观察，并不需要这么强光的手电筒。

06、人的眼睛对于灯光的直射也不适，我们也需避免灯光对人的影响。强光或长时间照射甚至可能导致伤害，需要提醒孩子注意自己的灯光对身边人的影响，不仅仅需要考量小动物，也同样需要考量身边的人。

07、在黑夜进入自然并非一定要有人造的灯光，夜间生物观察只是了解夜间大自然的方式之一，这种方式可能无法避免人造光的出现；但黑暗本身才是自然界自然的样子，有些黑夜里的活动或活动的部分环节，需要关闭所有灯光。当关闭人造光时，月光、星空、萤火虫、虫鸣、猫头鹰的声音……可能会更容易让孩子觉察到，这都是黑夜自然的一部分，包括黑暗本身。这些对于我们认识大自然、建立与自然的连接都很重要。

关于人数01、自然教育课程通常都会保证较高的师生比，总人数也不会很多，这样的选择是户外环境条件限制与自然教育的具体目标决定的。

02、以闻啼鸟为例，夜间观察大部分情况一场限制8个孩子，配2位老师，如果再增加孩子 ，通常会再增加老师，并分成两队。今年为了避免与其他团队“撞车”增加活动现场的难度，我们会尽量把人数控制在6个孩子。不同的机构对人数的控制不一样，但对于夜间观察类的活动，我们建议组织方更加谨慎控制参与人数。

03、夜间盲区较多，而孩子在户外、在夜间通常都会更兴奋，人数多会给现场带来更多的安全隐患。

04、夜行性生物本就有相对更高的敏感性，人流量带来的声音、光线等，都会给生物带来更大干扰。

05、人数太多也会影响教学本身的质量，夜间观察以学员观察和老师的解说为主，而很多小生物是无法保证多个孩子一起观察的，同时老师还需要控制在夜间的音量，这些因素都限制了一场活动能够承载的人数。

06、依旧要回到我们为何要带孩子在夜间走进大自然，为何要做夜间观察？如何降低对自然的干扰？我们很难想象浩浩荡荡一个大队伍，几十个人跟着一位老师，吵吵闹闹的状态下，如何安心观察生物，如何体现我们对大自然的尊重？

关于解说和观察的内容方式01、夜间观察离不开对物种的基础了解，物种相关的知识传递确实是受众非常感兴趣的内容，但相较于知识本身，走进真实的夜间大自然，重新认识黑夜，打破对黑夜的恐惧，发现黑夜下的精彩世界，收获对生命的敬畏和对自然的尊重，并在这个过程中，保有孩子的好奇心、锻炼孩子的专注力、观察力等等，都是比知识更重要的目标。

知识是重要的手段，而非目的，对不同年龄段的孩子，知识传递的比重也应该有所不同。

02、基于以上目标的共识下，在安全（人和物种双向安全）的前提下，我们鼓励多跟自然界的小生命有直接的接触，碰一碰平时觉得恐怖的虫子，摸一摸平时感觉很恶心的虫子，闻一闻虫子留在手上的味道，让小虫子在手上爬一爬看是啥感觉等，都是观察的重要方式，直接的感受对孩子而言是无可替代的，这样的真实连接能够有效地打消对自然的恐惧和偏见，比客观的知识告知显得更重要。

03、观察和解说的方式，直接体现了我们对待自然的态度，即便是接触，也不是暴力的抓捕，更不是直接带走，越是在这个过程留意减少伤害，越能通过行为本身达到理念传递的目标，对自然既非盲目恐惧，也非高高在上的自大，而是建立在真实感知和观察基础上的情感连接和科学认知。

走进自然，就避免不了对自然带来打扰，但这件事和这世间的很多事一样，并不是非黑即白的。
人类是大自然的一部分，避免不了与自然的互动，但当我们认清自己就是自然的一部分而非万物的主宰时，当我们愿意谦卑地认识和了解大自然时，智慧的我们也一定能有更好的办法降低自己对自然的打扰，更友好地与自然相处。

如何做，并无标准答案，而是评估我们的目标和能力，在当下的环境可以做到哪个程度。例如有些同行提出选择用有红色光的手电筒做夜间观察对生物的影响更小，而闻啼鸟认为观察到生物原本的颜色也是观察中很重要的一部分，所以没有选择这种方式，但对于有些活动，这样做也是合适的选择。

自然教育之所以称之为教育，是有对于教育目标、对于想要培养什么样的人有基本目标的。即便你觉得自己并非在做教育，但一场面向孩子的活动，带领者的行为，也会对孩子产生直接的影响。教育这件事在生活中时时刻刻都在发生。

抛除生态影响，当我们遇见自己喜欢的小生物就决定要占为己有时，认为只有属于自己才是喜欢的方式时，鼓励还是克制这样的行为，也许也会影响一个孩子如何对待自己喜欢的人，是控制、占有，还是相互独立、自由？当我们遇见弱小的虫子、青蛙，就觉得自己可以随意左右它们的生存，也许会影响一个孩子如何对待身边的弱者，是欺软怕硬，还是给予更多的善意和尊重？

同样，老师是高高在上的绝对权威做知识输出的角色，还是带领孩子探索自然的引导者角色，都会对孩子产生影响。

更多的关于教育的认识，例如我们在活动中用怎样的方式与孩子对话，规则要如何设定，遇见不知道的内容如何应对等等，我们在此文中不做更多阐述。在夜间观察活动中，认识很多物种、懂很多知识、博学的老师往往是很受欢迎的，但如果只是如此，那带给参与者的最大印象也可能会仅仅停留在“这个老师什么都知道、好厉害呀！”的层面，我们带领孩子走进自然的目标，到底传递了多少呢？而从长期来看，一位老师能获得尊重，也并非只依靠博学的知识。

在当下人与自然缺失连接、甚至人与人之间的信任都越来越难建立的时代，传递本文所倡导的理念并非易事，但我们相信这是这个社会以及人们内心真实的需要，而每次坚守原则都恰恰是最好示范、教学时刻，也是真正赢得认可和尊重的时刻。也唯有如此，我们才能真正促进整个行业更加良性可持续。

# 我们应该跟野生动物保持怎样的距离？

学校陆续放假了，许多小朋友们也开始了暑期的自然探索。虽然大家的生活早已步入了正轨，但新冠疫情的影响还在持续，当然，有些影响是好的，例如让人们的生活节奏慢一些，更加关注日常的点滴；例如开始有更多人反思吃野味带来的危害。

在出门的同时，大家却对自然有了些胆怯，认为大自然是极其危险的，野生动物都是恐怖的。新闻里报道，有人看到家附近有蝙蝠，就会非常害怕，打119报警，类似的事情不断发生。
很显然，人们因为疫情也对野生动物产生了很大的误解。我们到底应该跟野生动物保持怎样的距离？这里简单给出一些我自己的建议。

丁香家园对外的信息显示，新冠病毒“最初传染源为野生动物，可能是中华菊头蝠”，而这种蝙蝠在我国分布广泛。图片翻拍《中国哺乳动物多样性及地理分布》。

我们身边其实就有蝙蝠，即便是城市里也有，尤其到了夏季，你在傍晚经常能够看到蝙蝠在低空飞行，他们主要是捉小昆虫吃的，你讨厌的蚊子都是它们的食物。

蝙蝠虽然携带有很多的病毒，但这些病毒直接传染给人的可能性非常低，SARS病毒的源头虽然是蝙蝠，但实际上是通过果子狸传播给人类的，这个病毒从蝙蝠到果子狸，果子狸再被长途运输，这期间病毒经过很多代的变异，才最终成为可传染给人的病毒，如果我们不吃野味，没有这么亲密的接触，就不会发生这样的悲剧。

第二，蝙蝠没有那么可怕不代表可以乱摸

没有跟蝙蝠直接接触，基本上没有任何风险。我不建议随便乱摸，毕竟如果跟它们直接接触，那么传播的风险就会增加很多。当然也不用因此恐慌，研究蝙蝠的科学经常在蝙蝠洞里做研究，身边全是蝙蝠，通常也不会做夸张的防护，如果你跟它们有过直接的接触，事后记得洗手消毒就好了。

第三，萌萌哒的哺乳动物需要留意病毒传播风险

病毒跨种族传播的可能性极低，亲缘关系越近，越有可能相互传染病毒，近年来的流行疾病都与哺乳动物有关。例如鼠疫是啮齿类动物身上的病毒，MERS(中东呼吸综合征)是从骆驼传染给人的，SARS的源头是蝙蝠，这些动物都是哺乳动物。人类也属于哺乳动物，所以我们与哺乳动物的接触需要留意。

我们身边其实就有蝙蝠，即便是城市里也有.

去年有游客在青海与土拨鼠”亲密接触“（有鼠疫的风险），某自然教育机构带领孩子在马达加斯加徒手接触蝙蝠，都被业内人士评价非常危险，不值得提倡。

第四，禽流感在鸟类中非常普遍，偶尔出现变异可传染给人类，我们很难避免。

在升金湖自然保护区核心区里的家养大白鹅，它不该出现在这样的地方.

近年来禽流感也给人们带来很大恐慌。但禽流感有很多很多种，并且是在时刻变异的。大部分可以传染给人类的禽流感主要是由野鸟传染给家禽，再在家禽中变异，最终变成了可以传染给人的疾病。

所以解决禽流感，更重要的可能是我们的家禽养殖管理，需要注意尽量避免家禽跟野鸟的接触。尤其是很多养鸡养鸭的家禽养殖场，跟野生雁鸭类的栖息地过于接近，就会导致风险的增加。

第五，其他物种向人类传播病毒的可能性几乎为零
爬行类、两栖类、鱼类，以及一些节肢动物、软体动物等，因为与人类的亲缘关系太远了，几乎不可能出现能够同时感染它们也感染我们的病毒，如果哪一天真的出现了这种病毒，那可真是名副其实的超级病毒。

第六，大部分两爬类直接接触没有危险，需要留意毒蛇。

蟾蜍虽有毒，但跟它亲密接触也不会有什么危险.

蛇、蜥蜴、青蛙、蟾蜍，这些动物是很多人害怕的物种，但其实它们并没有那么危险。自然教育课程中，有些环节就会鼓励孩子摸一摸青蛙，这样的触感是非常有利于一个人建立与自然的连接，消除对自然的恐惧的，甚至是有毒的蟾蜍，我也同样会触摸，因为它也很温顺，只有在生命受到威胁的时刻，才会流出毒液，这个过程我反而学会了如何减少对它的打扰。

短尾蝮，合肥市区就能发现的毒蛇.

蛇类我们确实需要注意，但我们遇到的大部分蛇都是无毒的，且都很怕人，见到人就会逃走。即便是毒蛇，你在不伤害它的情况下（例如抓它或者踩到它）它也不会攻击人。

眼镜王蛇有可能主动攻击人，但在野外遇到它的概率也几乎为0，我们想找都找不到。如果真的害怕蛇，那么“打草惊蛇”是很管用的方法，拿着棍子敲一敲，附近的蛇就全跑了。
另外我们需要留意两爬类的寄生虫，如果食用野生的两爬类，会增加寄生虫感染的风险。

第七，大部分昆虫直接接触没有危险，全世界杀人最多的昆虫是蚊子。

虫子也是很多人非常害怕的生物，但同两栖爬行类一样，它们反而没那么危险，很多昆虫我们都可以直接用手接触。我们反而需要留意的是，与它接触的时候，如何尽量减少对它的伤害。不过，这些不起眼的小虫子确实有一些值得我们注意。

蜂类危险是很多人都知道的，被叮一口可能不会有太大的危险，被一群叮咬那可就有很大的生命危险了，甚至会当场休克。如果本身是过敏体质，也许被叮一口都有生命危险。最好的防护是，别去招惹蜂类，尽管很多很多蜂都不危险，但毕竟大部分人无法分清，所以你尽量远离就好。

遇到这类小虫子的概率不高，如果遇到了，别抓它们，如果爬到了你身上，别像蚊子一样拍死它们。因为拍死了它们，你的皮肤也就遭殃了，轻轻把它弹走就好了。

刺蛾、毒蛾、枯叶蛾的幼虫，俗称洋辣子。

这类虫子有毒大家也是都知道的，不必我多说。不过大家会把大部分没有毒的毛毛虫也归为这一类，所以都很害怕。其实大部分的毛毛虫都跟蚕宝宝一样人畜无害、含高蛋白，就算有的颜色鲜艳、张牙舞爪也改变不了这个事实。

蚊子才是最危险的，但我们又能怎么办呢？

其实蚊子是全世界杀人最多的昆虫，甚至是杀人最多的动物，就是因为它们在吸血的时候会传播病菌。登革热、疟疾这些恐怖的病全都跟蚊子有关，即便它们没这么危险，估计也没人喜欢它。可就算是这么危险这么讨人厌的生物，我们还是只能与它共存。这么想想，你也许就不怕别的动物了。（插一句题外话，有喜欢蚊子的人么？）

第八，所有的蜘蛛都有毒，不过中国的蜘蛛的毒性对人类没什么危险。

所有蜘蛛都有毒，这是它们捕食所需，但它们的毒量微小，可以麻痹或致死一个小昆虫，却对人类没什么影响。更何况，人家知道你不是它的食物，也不会咬你。所以，就算是络新妇那样长相恐怖的人面蜘蛛，你拿到手上玩也不会有什么危险。当然，你也不用妄想，被蜘蛛咬一口就变成了蜘蛛侠。

不过另一种蛛形纲的生物需要我们留意，那就是蜱虫。这种虫子，几乎每年都会被报道，要我们警惕。有些哺乳动物待比较多的地方，蜱虫会比较多，因为它们也是吸血的，例如野猪常活动的地方等。如果被蜱虫咬了，处理会相对麻烦一下，千万不要自己拔，因为它的口器有可能残留在伤口。最好还是找医生处理，做好消毒。

第九，蜈蚣、蚰蜒可能会误伤人，但马陆没啥危险。

蜈蚣和蚰蜒这两类张牙舞爪的多足生物都有毒，别轻易碰它们。我听说的最多的被蜈蚣咬的案例是由于蜈蚣爬到了鞋子里，一穿鞋就被咬了。（我也不知道蜈蚣为啥会躲在你的鞋子里？）

另一种被人误以为是蜈蚣的很多条腿的虫子其实是马陆，它们就温顺很多，也不像蜈蚣是捕食的生物。不过有的马陆在收到危险的时候会释放出有异味的液体或气体，但对我们的皮肤不会有什么伤害，不喜欢的话，不去招惹就好了，也不用害怕。

第十，蚂蟥没那么危险。
蚂蟥吸血，而且会让你一段时间血流不止，看上去就像受了很大的伤害一样。但往往很多人被蚂蟥咬了，自己都不知道，至少我有几次被蚂蟥咬，我是不知道的，它吸完血自己就跑了。

所以你把它想象成蚊子就没这么可怕了，它还不会像蚊子一样传播那么多疾病，也不会让你痒得难受。如果发现蚂蟥正在咬你，你就不断地拍打它，它感觉到危险就自动脱落。

第十一，野外不熟悉的地方别单独去
如果没有专业人士的带领，别往未知的地方去冒险。恐惧源于未知，危险常常来自于瞎逞能。如果真的想有更多的自然探索，那就积极学习自然知识，了解的越多，你就越能安全地享受自然带来的美好。
不过我有时候也会瞎逞能，毕竟常在河边走，哪有不湿足？从来没有百分百的安全，就看自己的评估了。对于未知的生物，尽量保持距离但也无需恐慌，说不定你的大喊大叫反而带来麻烦。

第十二，如果你是过敏体质，那就有更多说不准的事，只能建议你多留意一些
有研究表明，孩子从小如果有猫狗陪伴的话，未来过敏的可能性会比较小。其实从小多在大自然里玩耍，对这些自然的过敏也会小很多。

而且，有很多过敏其实不是身体过敏，只是心理在过敏。

第十三，我们不需要害怕野生动物，而是需要反思我们与大自然的关系。

不用因为这个疫情就觉得野生动物是恐怖的，如果你身边真的没有了野生动物了，如果野生动物从这个世界消失了，那么也就没有人类了。

人类医学事业的发展和农业养殖驯化的发展，给我们的健康树立了越来越坚实的保护墙。但对于野味贪婪、对于原本野生动物生存空间的不断掠夺，也同时在不断瓦解着这个保护墙。至少可以判断的是，近几年的公共卫生安全事件都是人祸。

第十四，有人说“最好的就是野生动物过野生动物的，我们人类在城里待着就好，别去打扰他们”，我不认同这个说法。

没有了人类的打扰，这当然是一个对于野生动物而言的美好世界。但无奈的事，我们根本不可能做的到。如果我们把人类和野生动物的空间划分地太过清晰，按照现在人类的发展，最后只会导致这个属于人类的空间越来越大。

如果我们不让更多的人关注野生动物，了解生物多样性的重要，不学会敬畏自然，那么健康的人与自然的关系就永远无法建立。走进大自然，亲近大自然，是学会敬畏自然最重要的一步。

当然，这并不意味着我们就鼓励大群的人走进山野走马观花，也不意味着我在反对建立自然保护区或国家公园。如何发挥这些保护地的教育科普意义，引导公众真正学会欣赏自然、尊重自然？自然教育和保护部门还任重道远。

第十五，以上的内容仅仅是我根据自己的经验提出的建议，并非标准答案。
说不定还会有人出来质疑批评，欢迎探讨。但我们不要啥事都想着能有一个标准答案，也许最后的标准答案是“成为一个被隔离的人员才是最安全的”。
# 大羊：多样性，不仅仅在生物这个层面重要
1.  什么是生物多样性？
“生物多样性”是一个非常年轻的词语，毕竟整个生态学都是近几十年才有的一个年轻的概念。相关的定义比较多，都是在通过这个词语去描述生物界多样性的程度。

 我国《保护生物学》一书中，给生物多样性所下的定义为：“生物多样性是生物及其环境形成的生态复合体以及与此相关的各种生态过程的综合，包括动物、植物	、微生物和它们所拥有的基因以及它们与其生存环境形成的复杂的生态系统。”

现在，我们在提到生态多样性时，都会提到三个层面：**遗传多样性、物种多样性、生态系统多样性**。这三个层面相互关联，相互促进，互为因果。 

这个概念是随着人类社会对于环境保护的重视而提出的。对于我们现代社会而言，“生物多样性很重要”几乎跟环保一样，是个被普遍接受的、“政治正确”的观念。而在若你有一定的自然经验，认真观察过身边的大自然，就更会赞叹生物多样性的力量，和生物多样性的宝贵。

2. 大自然没有绝对的对错，且每个生命都在时刻演化。

 我曾在之前提过，是多样造就了大自然的健康以及上亿年的生生不息。而大自然之所以能够如此丰富多彩，有一个重要的原因，是在自然界这样的一个系统中，没有绝对的对错是非，每一种生命都可以以自己的方式选择适合自己的生存方式。若大自然的评判标准是单一的，生物变成了千篇一律，那便一定不会有这个健康富饶的地球。

从个体来看，在大自然中，每一个物种都有自己的生存智慧，都在演化过程中形成了自己的生存机制；从整体来看，物种虽然各不相同，但却是无处不在的相互配合，甚至产生**协同进化**，共同促进着这个生态系统越来越健康，抵抗力越来越强。 

在这样的一个系统中，生命朝着各种不可思议的方向不断演化，形成了现在我们看到的多样、富饶、充满生命力的大自然。

这背后有两个重要的特点：一是每个生命都是被平等对待的，大自然没有统一的标准限制生命这样是对的，那样是错的，或者说一定得要是什么样子；二是生命是时时刻刻都在演化，生命充满希望，永远能找到出路。

3. 文化多样性是社会生命力的表现。
 这样的思考给我带来很多启发。由生态多样性，我联想到文化多样性。我想到人类历史进程中的很多事情，都反应了文化多样性的重要。 

回望历史，是不是在文化思想单一评判的时代，社会的进步是缓慢的、停滞的，甚至是倒退的，毫无生命力？例如秦朝“焚书坑儒”带来的后果，明清思想管控带来的危机，以及离我们比较近的文革时期，都有类似的特点。

而若在一个开放的时代，社会各方面的发展是否就重新焕发了活力，整个社会的发展也富有朝气？例如西方的雅典城邦时期、启蒙运动时期，东方的春秋时期、唐宋时期、民国前期，以及改革开放后的中国，都很类似。 

再看当今人类社会的发展，我不禁会从文化多样性的角度感受到些许可惜。在全球化、城市化、消费主义的大趋势下，文化的多样性，实质在遭受着重大的打击。

有意思的是，近代社会文化多样性所受冲击的时间线，又恰巧与生物多样性所受冲击的时间线高度重合。仔细想象也不足为奇，因为现在生物多样性所遭受的冲击，也恰恰是与全球化、城市化、消费主义的发展密切相关。

在文化多样性的减少层面上，以中国为例。少数民族的文化相较于汉族的文化是弱势文化；乡村的文化相较于城市的文化是弱势文化。

在这样的背景下，很多少数民族文化不知不觉消失，乡村文化不知不觉成为老一辈人的回忆。我在这里这么说，不是在反对发展，而是我们真的确定，这强势的一种文化，就是好的文化么？如果如此，为何现在又有那么多人想要回归乡村？有那么多人开始重新重视老手艺？开始重新重视非物质文化？甚至有些都会用“抢救”这个词，就像抢救一种濒危野生动物一样。

如果这种消逝是自然而然的，确实也不必太遗憾。但往往有很多消逝，是因为强制推行的一些措施，即在社会层面上有个默认的定义：那个是落后的，要被先进的代替。

但我们细想想，一些少数民族的文化与当地自然环境共生了几百年上千年，也许那就是最符合当地的生活方式。而我们现代化的生活方式才刚刚出现几十年或刚到100年，还没有来得及经历时间的检验。

以草原的游牧文化为例，我们认为定居是更好的生活方式，但实际上，游牧民族已经与草原生态系统融为一体。强行改变，不仅仅带来文化上的可惜，更在生态上带来不小的恶果。

当下每个人可能都有一种感观：似乎到了每个城市都是相似的，与当地环境磨合多年的生活方式和文化正逐渐消逝。 我确实也说不定到底哪个好、哪个不好，我只是期待我们不要太高高在上，我们能够谦卑一些，给多元的文化和生活方式更多包容。

有一天它消失了，就像物种的消失一样，永远都无法挽回。而若一个个都消失，仅仅保留有一种所谓好的、先进的，那我们的社会也会如生态系统一样，脆弱不堪。

4. 教育的多样性是人能够成为独立生命体的重要条件。

由生态多样性，我联想到教育的多样性。我想到每个孩子都是独特的生命个体，如果我们给孩子提供千篇一律的教育，用单一的评价标准去衡量每一个孩子，那么将有多少孩子的才华被抹杀？为了能够让每个孩子都能成为他自己，他们有权选择不同的教育，我们的社会有义务创造不同的教育。

 现在的教育方式来自于100多年前的普鲁士，当时创造这样教育方式的目标，是为了培养分工明确的工人。尽管100多年了，人类的科技发展迅猛，但在教育的底层范式上，与100多年前完全相同。所以，我们会戏说：用19世纪的教育方式，教授20世纪的知识，让孩子面对21世纪的未来。 

我承认当下的教育有特别多的不得已，但并不代表我们无从努力，因为我们无需幻想在一瞬间就彻底改变教育环境。

你是否还是紧盯着孩子的分数不放？你是否因为孩子的中考高考而非常焦虑？你是否觉得孩子如果不考上名牌、985就毫无前途？这一切，恰恰正是在用一个统一的、非黑即白的评价方式在评价一个孩子。但孩子要面对的未来，一定不是这样的。

如何更全面地、个性化的评价一个孩子，现在创新教育领域也有很多尝试，例如故事评估，徽章评估等方式，虽然这依然是一个很难解决的点，但至少我们可以做到的是，尝试创造传统学校之外的更多学习环境。更重要的是，尝试在生活的点滴，创造教育的机会。 

说到教育的多样性，我想到Aha社会创新学院对教育公平的定义，而这样的教育公平定义，恰恰与我之前提到的生态多样性的前提一致：每个生命都是被平等对待的，大自然没有统一的标准限制生命这样是对的，那样是错的，或者说一定得要是什么样子。 

孩子就是一个独立的生命体，他们当然应该被这样对待，不仅仅是孩子，我们每一个人都是。 

但还有一个前提：生命是时时刻刻都在演化，生命充满希望，永远能找到出路。换做教育，那就是每一个孩子（人）都有内在成长的渴望，每个孩子（人）都充满希望，时刻在成长变化。

这一点我是绝对相信的，你呢？


5. 多样性的价值，大自然一直在告诉我们。

我们如果想让人类社会的未来更有希望、更加健康、更有活力，就一定要记得多样性的价值。我们如果想让自己的生命更加精彩、更不负此生，就一定要记得多样性的价值。因为真正理解了多样性的价值，能帮助我们获得真正的自信，因为你会因此相信自己作为一个独立生命体的价值。 

多样性，不仅仅在生物层面上重要，在人类社会领域，同样重要。大自然其实一直在告诉我们这个道理。
# 大羊：为什么大自然是培养自由独立个体的绝佳场所？
在创业的过程中，经常会有人（尤其是关注教育的同行和家长）问我：闻啼鸟到底想要给孩子带来什么？你们想要培养的是一个什么样的人？

 这个问题我一度非常模糊，说不清楚，但心里似乎又有着一个明确的方向。随着这几年对于教育的深度思考，以及回望我自己的成长经历，我现在可以试着做一个回答。 

闻啼鸟在去年确定了我们新的slogan：让孩子在大自然中自由奔跑！

我觉得这句话也同样可以回答上面的问题，这句话换一个说法就是：我们要**培养一个懂得尊重自然的独立且自由的个体**。 这就是闻啼鸟要给孩子带来的，这就是闻啼鸟要培养的人。
（点击“让孩子在大自然里自由奔跑”，阅读我之前写的对于这一教育目标的详细解读。）

这也与Aha社会创新学院近几年提出的教育新范式，提出“好的教育是通向自由的教育”所一致。（点击“教育的新范式”，阅读详文。） 

在这里，有必要把对于“自由”的理解，再做一下阐释，我们对自由的理解有三层含义：

**自由是一种动力，一种促使人主动学习成长的内驱力**；

**自由是一种权利，一种“我自己可以选择什么值得做、什么可以做、什么不该做，以及如何去做”的权利**；

**自由是一种能力，一种分辨、思考、探索、行动的能力**。

在以上观点的基础上，我在这里提出：**大自然是培养自由个体的绝佳场所**。为什么这么说呢？

大自然保护（或唤醒）了我们的好奇心。
在一次婆罗洲热带雨林探索旅行结束后，一位爸爸感慨到：“我已经多年没有像这几天一样，对身边的事物充满好奇了。这是我很小的时候，才会有的状态。”而有类似感慨的成人，不止这位爸爸一人，我本人也有着深刻的感受。 

所有人都知道，孩子对这个世界是充满好奇的，他们不停地问为什么，这种好奇促使着他们不断尝试、玩耍、观察，于是学习就发生了。所以在我看来，好奇心=自主学习的动力，有好奇心的人会一刻不止地学习，而出于自主动力完成的学习，其效果和效率也远高于被逼迫的学习。

孩子既然天生有好奇心，所以我会称“**保护孩子的好奇心**”。遗憾的是，我们长大的过程，通常也是一个好奇心逐渐衰退的过程，我们的好奇无法在成人世界或在这个社会中得到回应，甚至还会被打压，而不感兴趣的学习，却不断压在我们身上，于是好奇心没有了，主动学习的热情也没有了。

在自然教育中，我们鼓励孩子们去发现、寻找大自然里的生物，在这个过程中，他们探索世界的好奇被鼓励，这种正向的反馈，会有助于好奇心的保留。 

而成人在自然观察的过程中，他们也会发现，原来这个世界还有这么多我不了解却这么有趣的东西，于是好奇心也开始促进他们主动寻找、认真观察。所以我会称，成人的好奇心，是需要被唤醒的，越是心里保留的好奇心成分多，越是容易被唤醒。不排除有些人的好奇心，可能一辈子都不会再被唤醒。

为什么大自然容易唤醒人们的好奇心，很重要的原因是因为它与我们日常生活所关注的东西差别太大了。**你若是身处在一个真实的大自然中，身体的感官也会被全方位调起**，这是一副摄影作品或一个纪录片无法替代的感受。 

所以我说，我们可以在室内或线上做自然教育的辅助教学，但绝不可依赖，因为自然教育的核心舞台，一定在真实的大自然中。

好奇心是激发我们探索一个事物的源动力，但若想持续探索，并在这个过程中有更多收获，就需要掌握方法，否则新鲜感一过，很可能就又觉得无趣了。

 所以，在自然教学中，我们会引导孩子如何去做观察。例如观察植物的方法，不仅仅可以看颜色和形状，还可以通过触摸叶子感受不同的质感，可以揉碎了闻一闻味道。在孩子提出问题的时候，也要引导他们通过自己的思考寻找答案，并学会在观察和阅读中验证自己的答案。

 这其实就是最基本的科学素养。一旦掌握了这个基本的方法，他们便有可能将一时的新鲜感，转变称长期的兴趣，这其实也是好奇心保留的一个过程。 

我们不用担心孩子提不出问题，尤其是在大自然这个丰富的场所中。这是什么东西在叫？为什么这个鸟的头上是白色的？这两只虫子在干嘛？ 

我们同样也不用担心自己回答不上来孩子的问题，或者这个问题找不到答案怎么办。我们我在寻找答案的过程，基于问题而对自然的认真观察才是最重要的，这个过程我们会收获很多远超过答案本身的成长。 

在大自然中提出的很多问题，通过观察自然就可解决。事实上，在真正的科研领域，很多宏观动物行为方面的研究依然在进行，而这些研究的基础数据，正是需要自然观察才能获得的。 

所以在自然教育中，我们会提出一个词，叫“**公民科学家**”。因为相比于很多在实验室里才能做的研究，自然观察是随时可以做的研究，对普通人来说，实在是门槛最低的一项科研活动了，孩子们也同样可参加。例如，我们现在民间参与很广的某某鸟类同步调查等等，都是普通人的自然观察给科研带来的贡献。

  3
面对不确定的勇气和坦然的态度。

这几年，VUCA这个词开始被越来越多的领域里使用，它是四个英文单词的缩写，这四个词分别为：Volatility（波动性）、Uncertainty（不确定性）、Complexity（复杂性）、Ambiguity（模糊性）。我们说新的时代是一个VUCA的时代，未来什么都有可能发生。 

很多人喜欢的未来是可控的，这可以带来强烈的安全感。这也是为什么公务员、事业单位、大公司会更受人欢迎的原因，但现在作出以上选择的人可能越来越少了，这恰恰也是这个时代越来越VUCA的体现。 

我们若是经常自然观察，就会发现大自然其实一直在告诉我们这个世界是一个VUCA的世界，而我们也会在探索大自然中，学会坦然地接受不确定，以及对未来的不确定拥有足够的勇气面对。 

我们提倡自然观察，是鼓励在自然中观察自由的野生动物。这与在动物园看被圈起来的动物有什么区别呢？一个很大的区别就是，野外的不确定性更大。 

例如，你准备明天早晨去某个公园观察翠鸟，你不可能提前给翠鸟打个电话约时间，请它明天早晨6:30在湖边的某个木桩上等你。你明天早晨去，有可能遇见、也有可能遇不见，但这都是自然观察中很自然的现象。而如果出现了，你一定会很开心，但如果它不出现，难道这一趟出行就没有意义了么？说不定你会遇见一群白头鹎在水边洗澡，这又会成为一个惊喜。 

当我们坦然接受了不确定是正常的，我们就会更享受这个寻找和等待的过程。 

这是自然教育魅力，却也是自然教学的难点。比起室内教学，户外的不确定性增强了多倍。老师需要灵活应对自然中随时可能发生的状况，需要抓住自然中随时可能出现的教学点。

 若是这场出行看到的动物少了，可能会有家长会因此觉得，这个自然课没啥用。因为在他的眼里，只有确定的知识才是有用的，而恰恰忽略了在大自然中真正重要的东西。 

4
**在大自然中学习对多样性的包容**，并获得真正的自信。

大自然是个丰富多彩的世界。直至今日，人类依然有可能发现某个全新的物种。大自然是充满多样性的，从生态学的角度，这个多样性包含生态系统的多样性、物种的多样性、基因的多样性。 

是多样造就了大自然的健康以及上亿年的生生不息。若大自然的评判标准是单一的，生物变成了千篇一律，那便一定不会有这个健康富饶的地球。 

从个体来看，在大自然中，每一个物种都有自己的生存智慧，都在演化过程中形成了自己的生存机制；从整体来看，物种虽然各不相同，但却是无处不在的相互配合，甚至产生协同进化，共同促进着这个生态系统越来越健康，抵抗力越来越强。 

当你真的认识到了生物多样性的价值，你也就会反思我们自己的生活。

 例如我们的教育，如果我们给孩子提供千篇一律的教育，用单一的评价标准去衡量每个孩子，那么将有多少孩子的才华被抹杀？再回望我们的历史，是不是在文化思想单一评判的时代，社会的进步是缓慢的、停滞的，甚至是倒退的？而若在一个开放的时代，社会各方面的发展是否就重新焕发了活力？

 我们每个人都有自己的价值，“我是独一无二的”。当你认识到了多样性的价值，那你也一定会因此收获真正的自信。  

5在学习对大自然的尊重和谦卑中，学会尊重弱者。

自然观察是一个在野外观察野生状态下的动植物的行为，正如我上文所说，我们能否遇见一只翠鸟，不是我能够决定的。这不像是我们把一只翠鸟抓起来，关在笼子去观察。这需要我们根据翠鸟的行为习性，在不打扰它、不惊吓它的前提下，静静守候、耐心观察。

 在大自然中，我常教孩子要学会安静，安静就是走进自然时对带自然尊重的第一原则。安静代表着我们把自己当客人，我们不希望打扰到这里“原住民”的生活。 

我们若是懂得谦卑和尊重，大自然也就会对我们更加照顾，你也就一定能发现更多意思的场景。在这个过程中，我们不再是高高在上的万物之灵，而是跟万物平等的生命。

 有人会说，人类确实高于其他生物，我们是唯一有文明的物种。若真从这个角度来说，我会觉得，“真正的文明”恰恰要体现在我们是否能够尊重弱者这个问题上。 

大孩子欺负小孩子，这不是文明；
大人用权威压制孩子，这不是文明；
富人因金钱鄙视穷人，这不是文明；
健全人不考虑社会里残障人士的生存权利，这不是文明；
城市人瞧不起乡村，这不是文明；
经济发展较快的民族不尊重少数民族的文化，这不是文明…… 

在人类有能力破坏地球上任何一个角落时，我们依然尊重、考量野生动物的生存状态，这才是真正的文明。

 若一个孩子，了解大自然，懂得尊重野生动物，他也一定不会是个欺软怕硬的人。


以上的五个观点，正是在回应之前提到的自由的三个维度。

“好奇心”是我们学习和成长的内驱力，是一种动力；
“独立思考和面对未知的勇气”是一种当下社会以及未来重要的能力；
“获取真正的自信”是认识到自己权利的体现；

而“认识到多样性的价值，以及学会尊重弱者”恰恰是将此权利推及给更多的人和生命，是一个从个体的自由上升到社会的自由、世界的自由的认识。

▼
几个维度说起来可能有些深，但若从教育的视角简单来看，就是“在探索大自然的过程中，孩子能够清晰地知道自己为何要来探索（学习），以及如何探索（学习）。”

当然，需特别强调的是，并非进入到大自然就能学习到以上这一切。甚至在大自然中出现极其自大、伤害他人、伤害自然的人也很多。所以重要的是，我们的老师，我们的家长，能否时刻带着觉察的视角，去引导我们的孩子？ 

更重要的是，我们是否会带着这样觉察的视角，审视我们自己的生活，并学会在大自然中终身学习？

# 这是我夏季主要的夜生活方式
南艳湖是我经常开课的一个公园，这里不及大蜀山、逍遥津、滨湖湿地公园等这么出名，但在生物多样性上其实毫不逊色，是众多自然爱好者们都非常喜欢的地方。

这里是我们观鸟课的主要开课点，因为这里既有湿地，又有树林，所以水鸟、林鸟皆可观察。再加上视野开阔，所以在物种种类上虽不及大蜀山，但观测条件却比大蜀山好很多，尤其适合观鸟的初学者。我们在这儿见证着不同鸟儿的迁徙往返，以及众多新生命的诞生。

到了夏季，这儿又会是另外一番景象，尤其是在夜晚，各类生物在这片少有干扰的湿地环境中，上演着精彩的生命故事。

水边最容易发现的昆虫要数蜻蜓这一类了。暑假期间，正是它们最活跃的时间，白天去水边，你会看到大量的蜻蜓在水边或水面上飞行盘旋，它们有的是在捕食，有的是在争抢领地获取交配权。如果运气好，还可能看到你追我赶的水边“空战”，或者**蜻蜓点水的产卵场面**。

但三伏天的白天确实太热了，所以我们常常晚上去寻找它们。夜晚虽然看不到精彩的“战斗”场面，但近距离观察却会变得容易很多，因为大部分蜻蜓晚上都休息了，所以它们大多都趴在某个植物上静静地呆着，我们得以有时间耐心观察和拍摄。当然有时也会遇见清醒的它们，可能会因趋光性直接飞到你的身上，这时不用惊慌哦。

除了**蜻蜓目昆虫**，南艳湖最常见的就是**直翅目**这一类别了。在南艳湖各个大大小小的湖边**禾本科**的植物上，都会发现各类蝗虫的身影。蝗虫本身不是夜间活动的生物，但南艳湖的这些水中禾本科植物可能太符合他们的胃口了吧，所以即便是夜晚，也能看到它们进食的场面。仔细留意，你会发现，处处都有它们啃食过的痕迹。

实际上，夜晚比蝗虫更为活跃的是另一些直翅目昆虫，在中国的民间，他们都是非常出名的。那就是我们俗称的“蝈蝈”和“蛐蛐”。现在这个时候，大部分的蛐蛐都还没有成熟，所以我们听不到太多蛐蛐悠扬的歌声，反倒是你往水边的草丛一走，就会发现一只只虫子从你脚边跳开，这就是它们了。而蝈蝈呢，此时已经到了最“兴奋”的状态，开始高唱起来。他们的歌声会从傍晚持续到天亮之前，会从7月下旬持续到9月下旬。

蛐蛐和蝈蝈的触角很长，这样方便它们在夜晚的时候感知周围的世界，但蝗虫就不同了，他们主要依靠视力在白天活动。这也是区分这两大类直翅目的一个简单方法，仔细思考生物为什么会长这样，会发现越来越多有趣的知识。

若说南艳湖小朋友可能最喜欢的昆虫，一定要说**鞘翅目的天牛**啦。其实只要你认真观察一棵园子里的树，就会发现很多和天牛相关的东西。很多的柳树上都有一些圆圆的洞，他们就是天牛留下的痕迹。难怪说很多做园林的人不喜欢天牛，在大蜀山留下了很多当年治理天牛留下的器材。

但这不过是大自然生态链的一部分，现在没有专门治理天牛，没有专门治理蝗虫，南艳湖的生态却也在一步步变得多样。只要人少去干扰就好了，**大自然自会有他的办法。**

天牛的“长辫子”是小朋友喜欢它的重要原因，但我更喜欢他们霸气的眼睛，像极了黑社会老大。而它极具咬合力的口器（嘴巴），也确实给了它做“黑社会”的资本。

但你只要胆子大一点，避开它的嘴，还是很容易捉住它们的，不过它的力量确实很大，当你把它拿离树干的时候，一定小心不要用力太猛，不然它死死勾住树皮的脚，就有可能被扯断了。你把它放在手上，它并不会咬你，也许会停留一会，然后扇起翅膀就飞走了。**天牛的飞行一定要看，可美了！**

南艳湖的树上，还有另一位更令孩子兴奋的昆虫，那就是同属鞘翅目的锹甲。我真的很奇怪，这家伙前面长了这么大个钳子，头不会重么？有人说锹甲的钳子力量很大，可能会夹出血来，我没有拿自己的手指头去尝试过，只是用枯草去挑逗过，可它并没有传说中那么好战，让我有些扫兴。不过，就只是这个威武的外形，已经能让大家兴奋很久啦！

除了昆虫，蛙类应该是这片湿地更有代表性的物种了。只要不是人流众多的路面，我们都有可能看到一个大大肥肥的家伙傻傻地呆在路面上，他们就是中华蟾蜍。很多人觉得这家伙危险、恶心，但其实我接触的5岁以下的孩子几乎没有讨厌它的。

而所谓的人一碰就会中毒，或者皮肤变得跟它一样，也都是谣传而已。蟾蜍虽有毒，但毒液不会轻易流出来，人正常的接触只要不伤害它都是没问题的。尝试碰一碰它吧，会让你不再对它恐惧，恐惧只是源于我们的想象。

泽陆蛙也是在稍远离水面也可以经常发现的两栖动物，它们的身体纤细，纹路色泽多变，是南艳湖的几种蛙中，最不依赖于水的蛙。

水边最常见的是我们民间俗称的“田鸡”：黑斑侧褶蛙。这类蛙要比泽陆蛙大很多，后腿的力量也很大，弹跳力极强。它们常常在夜晚，在水边蹲守捕食猎物。如果你走在水边，听到“扑通”“扑通”的落水声，那基本上都是它们跳入水的声音。但它一跳入水中，你就几乎很难发现它们的身影了。

另一类跟黑斑侧褶蛙长得很像的是金线侧褶蛙，它们大多更显绿一些，皮肤也更光滑一些。这类蛙我们更常在水里发现它们，可能露个头，也可能蹲在莲叶上面。

黑斑侧褶蛙和金线侧褶蛙在夜晚都是鸣叫高手，声音洪亮有力，也不愧对于它们自己的大个子。但另一类声音洪亮的蛙却有着较小的身躯（大拇指甲盖大小），它就是饰纹姬蛙。饰纹姬蛙也常常会到水边的陆地上，但由于个头太小，所以我常是听见声音，却找不见它。它也有着极强的弹跳力，在遇见危险时大步一跳便“消失”了。

在这么丰富的湿地中，蛇类的身影当然必不可少。但毕竟属于食物链上层的生物，数量并不会太多，且蛇类都是独自悄悄地行动，大多时候躲在某个角落静静地守候猎物，并不容易遇见。由于它们的主要食物青蛙在雨天会更加活跃，所以我们在雨后发现蛇类的可能性会更大。

我在南艳湖遇见过两种蛇。一类是大蜀山最常见的赤链蛇，它本质上是一种无毒蛇，但我曾遇见过被一只大的赤链蛇咬过鼻子的小狗，小狗的鼻子都肿了个大包。

见到我之后，从水中逃走的赤链蛇。

所以有人说赤链蛇会积累它吃掉的蟾蜍体内的毒素，也许是真的。它们的胆子很小，遇到动静就会快速逃走，如果旁边有水，它们通常会选择逃入水中。

另一类蛇就是真正的毒蛇了，那便是短尾腹。听说大蜀山也有它，可我却从没见过，反倒在南艳湖遇到过好几次。有毒的蛇通常都会“淡定”很多，盘在某个地方，等待猎物经过时突然袭击。即便人靠近，它们也并不一定会动，因为移动常常是被发现的原因，当一直处于不动的状态时，无论是猎物还是天敌，可能都发现不了它们。

虽说南艳湖有蛇，但大家也不用担心。南艳湖每日来来往往那么多人，知道这里有蛇的却没有几个。毕竟，这些小生物对比人类而言，都是极其弱小的，它们躲人类还来不及呢，又怎么会来找我们的麻烦呢？

除了以上介绍的生物，南艳湖还有无数有趣的小生命们。
水中不仅有鱼虾游弋，更有鳖蝽、水虿、孑孓等在里面生存；水面上不仅有荷花、莲花、菖蒲，更有狸藻这类有趣的食肉植物在水里伺机捕食；草丛中不仅有蜻蜓、蝗虫，还有各类毛毛虫和蜘蛛在为自己的生存各自努力……

尽管这只是一个都市中的小湿地，尽管夏日的每一个夜晚都有数不清的人在这个公园里散步、游玩、运动，但这其实是一个我们不曾留意过的精彩世界。

我们在这儿生活，也同样有无数可爱的生命，在这里繁衍生息，来来往往，它们比我们更需要这片小小的大自然。

在夜晚走进自然，夜探大蜀山，探秘湿地夜生活，做一名黑夜探秘小勇士。
# 合肥河道惊现巨蟒！它究竟是什么？
近日，媒体一则《重要提醒！合肥一河道惊险巨蟒！（视频）》的消息传遍了合肥人的朋友圈。
滨湖新区塘西河河道内发现一条长达6m的惊天巨蟒，一下子让本就受暴雨困恼的合肥人出门更加困扰！

那么这究竟是什么蛇？为什么合肥地区有蟒蛇？接下来就由闻啼鸟著名侦探鹡鸰酱和大家一起了解一下这一巨大的物种。

这是什么蛇？
图中这条蟒蛇，身体粗壮，身体黄色，背部具有许多大块状的棕色云状斑纹，虽然视频中的拍摄者说体长可能达6-7m，但根据视频目测可能仅有3-4m之长。

通过外形，可以看出这是一条缅甸蟒（Python bivittatus），无毒，主要分布在东南亚热带雨林地区，在我国华南地区也有边缘分布记录，但数量稀少。

缅甸蟒还是亚洲第二大蟒蛇，体长仅次于网纹蟒。虽然是第二大蟒蛇，但缅甸蟒体长大多在5m以内，但由于其毕生会不断生长，极少数个体能达到7m以上。
缅甸蟒主要以鸟类和中小型哺乳动物为食，擅长爬树和游泳，十八般武艺样样精通。

为什么合肥会有蟒蛇？
野生的缅甸蟒是我国的一级重点保护野生动物，同样也是IUCN濒危物种红色名录易危（VU）物种，濒危动植物种国际贸易公约附录Ⅱ物种，野外种群急需保护。
但作为一个典型的南方物种，其主要生活环境是热带雨林，怎么会在合肥地区的河道中被发现呢？

虽然是一种大型蟒蛇，但缅甸蟒性格比较温和，对人的攻击性不强，因此深受国内外爬宠爱好者喜爱，甚至是许多蟒蛇养殖爱好者的入门宠物之一，也培育出许多变种，如黄金蟒。

而合肥地区作为亚热带丘陵平原地区，是不可能有野生蟒蛇分布的，因此这条惊天巨蟒必定是一条被人放生或逃逸的宠物蟒。



原本缅甸蟒作为人工繁育非常成熟的宠物蟒蛇，虽然是国家一级保护的，但执法部门通常睁一只眼闭一只眼。然而今年由于疫情原因，我国加大了对野生动物饲养的管控，继续饲养缅甸蟒属于严重的违法行为，因此这只塘西河的缅甸蟒很有可能是一条被放生的蟒蛇。


这个蟒蛇会攻击人吗？

一般情况下，人工饲养的缅甸蟒性情都非常温顺，不会主动攻击人类，遇到人类，可能它们会更加害怕，并且由于蟒蛇体形超大，可以更容易发现它的存在，因此在合肥遇到它，不需要害怕，只需要避开就行了。


真正需要注意的是，合肥地区野外是有毒蛇分布的，比如短尾蝮，毒性足以致命。

可虽然短尾蝮的毒性大，但它们也并不主动攻击人，而且还很害怕人。通常发现它们时，它们会保持不动，继续伪装。

所以，只要不随意捕捉，在野外注意脚下安全不要踩到它们，都不会有危险。

暴雨后，蛇类常会出现在高地避险，因此在野外行走一定要注意水边草丛、灌丛等环境，大家都是暴雨受害者，切勿捕捉和伤害蛇类。
# 穿山甲被调整为国家一级保护动物
020年6月3日，国家林业和草原局发函，正式将中国所有穿山甲属物种调整为国家一级保护野生动物。这也是自1989年《野生动物保护法》规定国家保护动物以来，第二次对保护名录进行调整。（第一次是2003年将麝类提升为国家一级保护野生动物）

说到穿山甲，基本上所有人都认识这种背披鳞甲、擅长打洞的的动物。传说中，穿山甲因为擅长打洞，便被认为具有“通乳”作用（事实上没有科学依据）。据统计，上世纪60-80年代，中国穿山甲年消费量达160000头，穿山甲也从过去的常见物种变为了现在的极度濒危物种。

本次保护等级调整，预示着中国穿山甲保护进入新的阶段，为打击穿山甲捕猎、交易、运输等黑色产业链提供了更强大的法律支持。

全世界共有3属8种穿山甲，其中有1属3种分布在中国，也就是穿山甲属的中华穿山甲、马来穿山甲和印度穿山甲。

中华穿山甲
世界自然保护联名（IUCN）极度濒危（CR）物种，作为中国以前最常见的穿山甲，遍布于中国南方的山区丘陵，如今已濒临灭绝。
安徽皖南地区近年来依旧有多次中华穿山甲的救助记录，可能是安徽最后的种群之一。

马来穿山甲
世界自然保护联名（IUCN）极度濒危（CR）物种，主要分布于泰国、马来西亚等东南亚国家，在中国仅边缘分布于云南南部。
由于中华穿山甲被大规模捕杀后，马来穿山甲被作为中华穿山甲的替代品也被大规模捕杀。

印度穿山甲
世界自然保护联名（IUCN）濒危（EN）物种，主要分布在印度、巴基斯坦等南亚国家，在中国仅边缘分布于藏南地区。
# 枸骨
常绿灌木。叶片厚革质，二型，四角状长圆形或卵形，长4-9厘米，宽2-4厘米，先端具3枚尖硬刺齿，有时全缘（此情况常出现在卵形叶），叶面深绿色，具光泽；花淡黄色，4基数。果球形，直径8-10毫米，成熟时鲜红色。
常见于山坡、丘陵等的灌丛中、疏林中以及路边、溪旁和村舍附近，城市园林绿化等也有栽培。
# 火棘
常绿灌木，高达3米；侧枝短，先端成刺状。叶片倒卵形或倒卵状长圆形，先端圆钝或微凹，叶柄短。花集成复伞房花序，花瓣白色。果实近球形，桔红色或深红色。花期3-5月，果期8-11月。
**火棘树形优美**，夏有繁花，秋有红果，果实存留枝头甚久，在庭院中做绿篱以及园林造景材料，在路边可以用作绿篱，美化、绿化环境。
# 腊梅
落叶灌木，高达4米；幼枝四方形，老枝近圆柱形，灰褐色。叶纸质至近革质，卵圆形，先花后叶，花期11月至翌年3月。
花香酷似梅花，花瓣如同用黄蜡捏成，故称蜡梅。果托近木质化，坛状或倒卵状椭圆形，口部收缩，并具有钻状披针形的被毛附生物。
野生于我国多数省份山地林中。合肥常见园林树木。蜡梅不是梅花哦！
# 迎春花
合肥常见园林物种。落叶灌木，直立或匍匐，枝条下垂。枝稍扭曲，光滑无毛，小枝四棱形。叶对生，三出复叶；花单生于去年生小枝的叶腋，稀生于小枝顶端。花萼绿色，花冠黄色，裂片5-6枚。花期2-3月，果期4-9月。

别名迎春、黄素馨、金腰带，落叶灌木丛生，株高30到500厘米。小枝细长直立或拱形下垂，呈纷披状。3小叶复叶交互对生，叶卵形至矩圆形。花单生在去年生的纸条上，先于叶开放，有清香，金黄色，外染红晕，花期2到4月。

因其在白花之中开花最早，花后即迎来百花齐放的春天而得名，**与梅花、水仙、和山茶花统称为‘雪中四友’**
# 老鸦瓣
又称二叶郁金香。鳞茎皮纸质，内面密被长柔毛。茎长10-25厘米，通常不分枝，无毛。叶2枚，长条形，无毛，远比花长。花单朵顶生，靠近花的基部具2枚对生（较少3枚轮生）的苞片，苞片狭长形。花被片狭椭圆状披针形，白色，背面有紫红色纵条纹；雄蕊3长3短，蒴果近球形，有长喙。花期2-3月，果期3-4月。

有鳞茎。花被片狭白色。生于山坡草地及路旁。大蜀山西扩公园易见。
# 结香
合肥常见园林物种。花期冬末春初，果期春夏间。灌木，小枝粗壮，褐色，常作三叉分枝；皮极坚韧，叶痕大，叶在花前凋落，披针形，两面均被银灰色绢状毛；头状花序顶生，多朵花聚成绒球状，花芳香，黄色。果椭圆形，绿色。
# 阿拉伯婆婆纳
黄色。又名波斯婆婆纳。合肥各个公园、小区草地常见。花期2-5月，果期4-6月。铺散多分枝草本，边缘具钝齿，两面疏生柔毛。总状花序很长，花冠蓝色，蒴果肾形。原产于亚洲西部及欧洲，现已归化为荒野杂草。

茎密生两列多细胞柔毛。总状花序很长；花冠蓝紫色，长4到6毫米，裂片卵形至圆形，喉部疏被毛；雄蕊短于花冠。蒴shuo果肾形，被腺毛，成熟后几乎无毛，种子背面具深的横纹，长约1.6毫米。

常见于路边、宅旁、旱地夏熟作物田。
# 婆婆纳
中国本土的婆婆纳。合肥各个公园、小区草地常见。花果期3-5月，果期4-7月。铺散多分枝草本，被长柔毛，叶片心形，边缘有深刻的钝齿，两面被白色长柔毛。总状花序很长；花冠淡紫色至白色，蒴果近于肾形。
# 直立婆婆纳
合肥各个公园、小区草地常见。小草本，花期3-6月，果期4-7月。茎直立，多不分枝分枝，叶边缘具钝齿，两面被硬毛。总状花序长而多花，花冠蓝紫色。
# 繁缕
合肥各个公园、小区草地常见。一年生草本，花期3-6月，果期3-6月。叶片宽卵形，基生叶具长柄，上部叶常无柄。疏聚伞花序顶生；花瓣白色，深2裂达基部，花柱3，蒴果卵形，种子卵圆形。

一年或二年生草本；茎一侧具有短柔毛；叶对生，卵形；聚伞花序，花白色，花瓣5，深2裂达基部，看似10瓣，雄蕊常3到5枚，柱头3裂；蒴果卵圆形。

繁缕生长极为繁茂，往往长成一大丛。《本草纲目》记载‘茎蔓甚繁，中有一缕，故名’。**这‘一缕’是指茎折断后，维管束依然相连**。
# 珠序卷耳
合肥各个公园、小区草地常见。一年生草本，花期3-5月，果期5-6月。茎密被长柔毛，茎下部叶片匙形；茎上部叶片倒卵状。聚伞花序簇生；花瓣5，白色，顶端2浅裂。周围有10枚黄色雄蕊，花柱5裂。蒴果长圆柱形。

珠序卷耳的‘珠序’是指它的花序在未开放时呈较明显的球状。球序卷耳的腺毛具有粘性，质地细腻，手感柔软，又称‘粘毛卷耳’。
# 望春玉兰
合肥各公园偶有栽培。花期3月，果熟期9月。落叶乔木，树皮淡灰色。叶椭圆状披针形，上面暗绿色，下面浅绿色。花先叶开放，芳香；花被9，外轮3片紫红色，近狭倒卵状条形，长约1厘米，中内两轮近匙形，白色，外面基部常紫红色。聚合果圆柱形，常因部分不育而扭曲；种子心形，外种皮鲜红色。
# 玉兰
合肥常见园林物种。花期2-3月，果期8-9月。落叶乔木，高达25米；树皮深灰色，粗糙开裂；冬芽及花梗密被淡灰黄色长绢毛。叶纸质，倒卵形、宽倒卵形或、倒卵状椭圆形；花先叶开放，芳香，直径10-16厘米；密被淡黄色长绢毛；花被片9片，白色，基部常带粉红色，聚合果圆柱形，蓇葖厚木质。
# 二乔玉兰
乔木，高6-10米。叶纸质，倒卵形，花先叶开放，浅红色至深红色，花被片6-9，外轮3片花被片常较短约为内轮长的2/3；蓇葖卵圆形或倒卵圆形，熟时黑色；种子深褐色。本种是玉兰与辛夷的杂交种，本种的花被片大小形状不等，紫色或有时近白色，芳香或无芳香。在园艺栽培约有20栽培种。
# 紫玉兰
又称辛夷。落叶灌木，高达3米，常丛生。叶椭圆状倒卵形或倒卵形，花叶同时开放，瓶形，稍有香气；花被片9-12，外轮3片萼片状，紫绿色，常早落，内两轮肉质，外面紫色或紫红色，内面带白色，花瓣状；雄蕊紫红色，雌蕊淡紫色；成熟蓇葖近圆球形，顶端具短喙。
# 野迎春
又名云南黄素馨。合肥常见园林物种。花期11-8月，果期3月-5月。常绿直立亚灌木，高0.5-5米，枝条下垂。小枝四棱形，具沟，光滑无毛。叶对生，三出复叶或小枝基部具单叶；花通常单生于叶腋，稀双生或单生于小枝顶端；花冠黄色，漏斗状，栽培时出现重瓣。
# 金钟花
合肥常见园林物种。花期3-4月，果期8-11月。落叶灌木，高可达3米。枝棕褐色或红棕色，直立，小枝绿色或黄绿色，呈四棱形，皮孔明显，具片状髓。叶片长椭圆形至披针形，或倒卵状长椭圆形，花1-4朵着生于叶腋，先于叶开放；花冠深黄色，4裂。相似种连翘多生于北方，茎中空。
# 蒲公英
合肥各个公园、小区草地常见。多年生草本。花期3-9月，果期3-10月。叶倒卵状披针形、倒披针形或长圆状披针形，先端钝或急尖，边缘有时具波状齿或羽状深裂，有时倒向羽状深裂或大头羽状深裂。头状花序直径约30-40毫米，舌状花黄色，边缘花舌片背面具紫红色条纹，花药和柱头暗绿色。瘦果倒卵状披针形，暗褐色，冠毛白色。

别名黄花地丁、婆婆丁。叶子长4到20厘米，宽1到5厘米。舌片长约8毫米，宽约1.5毫米。种子上有白色冠毛结成的绒球，花开后随风飘到新的地方孕育新生命。
# 芥菜
一年或二年生草本。基生叶丛生呈莲座状，茎生叶窄披针形或披针形，长5到6.5毫米，宽2到15毫米，基部箭形，抱茎，边缘有缺刻或锯齿。总状花序顶生及腋生；花瓣白色，短角果倒心状三角形。

说到芥菜，大家并不陌生，我们最常在味觉上与其打交道。将芥菜连根拔起，做成芥菜饺子或其他吃食，口齿回甘有清香，让人不得不爱。
# 碎米芥
一年生小草本，高15-35厘米，茎直立或斜升，分枝或不分枝，下部有时淡紫色，被较密柔毛。基生叶具叶柄，有小叶2-5对，顶生小叶肾形或肾圆形。
总状花序生于枝顶，花小，直径约3毫米，花瓣4，白色，倒卵形。长角果线形，稍扁，无毛，长达30毫米；种子椭圆形，顶端有的具明显的翅。
# 菥（xī）蓂（mì）
一年生草本，花期3-4月，果期5-6月。茎直立，具棱。基生叶倒卵状，边缘具疏齿；总状花序顶生，花白色。短角果倒卵形，扁平，边缘有翅。
# 附地菜
一年生草本。茎通常多条丛生，基部多分枝。基生叶呈莲座状，有叶柄，叶片匙形。叶揉碎后有黄瓜味，这也是识别附地菜的一个小技巧。

花序生茎顶，花冠淡蓝色或粉色，5裂，正中央有一圈黄色结构称为“喉部”。早春开花，花期甚长。
# 点地梅
一年生或二年生草本。主根不明显，具多数须根。叶全部基生，叶近圆形或卵圆形。花葶通常数枚自叶丛中抽出，高4-15厘米，被白色短柔毛。伞形花序，花冠白色，喉部黄色，蒴果近球形，果皮白色。

**点地梅的花初开是白色的，喉部的黄色很容易让昆虫找到它储有花蜜和花粉的花管；柱头受精后花朵会变成粉红色，使昆虫很容易区分。**
# 紫花地丁
多年生草本，生于田间、荒地、山坡草地、林缘或灌丛中。无地上茎，高4-14厘米，果期高可达20余厘米。叶多数，基生，莲座状；叶片长圆形、狭卵状披针形或长圆状卵形，基部截形或楔形，边缘具较平的圆齿。花中等大，紫堇色或淡紫色，稀呈白色，喉部色较淡并带有紫色条纹。蒴果长圆形，种子卵球形，淡黄色。花果期3月中旬至9月。
# 猪殃殃
多枝、蔓生或攀援状草本，花期3-7月，果期4-9月。茎四棱，棱上、叶缘及叶下面中脉上均有倒生小刺毛。叶4-8片轮生，近无柄，叶片条状倒披针形，顶端有凸尖头。聚伞花序腋生或顶生，单生或2~3个簇生，有黄绿色小花数朵，花瓣4枚。果干燥，密被钩毛，每一果室有1颗平凸的种子。
# 红花檵（ji四）木
合肥各个公园、小区常见。灌木，多分枝，小枝有星毛。叶革质，卵形；花簇生，比新叶先开放，花瓣4片，带状，花期3-4月。野外原生种檵木，花瓣为白色。
# 石楠
合肥常见园林物种。花期4-5月，果期10月。常绿小乔木，叶片革质，长椭圆形；复伞房花序顶生，花白色密生，花药带紫色；果实红色，后成褐紫色。花朵具有强烈气味，容易吸引大量昆虫帮助其传粉。
# 猪殃殃
多枝、蔓生或攀援状草本，花期3-7月，果期4-9月。茎四棱，棱上、叶缘及叶下面中脉上均有倒生小刺毛。叶4-8片轮生，近无柄，叶片条状倒披针形，顶端有凸尖头。聚伞花序腋生或顶生，单生或2~3个簇生，有黄绿色小花数朵，花瓣4枚。果干燥，密被钩毛，每一果室有1颗平凸的种子。
# 通泉草
一年生草本，本种体态上变化幅度很大。总状花序生于茎、枝顶端；花冠白色或蓝紫色。蒴果球形；种子小而多数，黄色。通泉草生长在较为潮湿的地方，潮湿的地方往往靠近水源，这也就解释了通泉草名字的由来。通泉草的花完全开后，像一只滑翔的小鸟，所以在日本还有“紫鷺苔”的名字。
# 救荒野豌豆
别名大巢菜，花期3-6月，果期4-7月。一年生或二年生草本，茎斜升或攀援，单一或多分枝，具棱，被微柔毛。偶数羽状复叶，叶轴顶端有卷须，托叶戟形，小叶2-7对，长椭圆形或近心形。花腋生，无梗；花冠紫红或红色。荚果线长圆形，成熟时背腹开裂，果瓣扭曲。
# 四籽野豌豆
一年生缠绕草本。茎纤细柔软有棱，多分支。偶数羽状复叶，顶端为卷须。总状花序，花甚小，1-2朵着生于花序轴先端；花冠淡蓝色至紫白色，荚果长圆形，具网纹。花期3-6月，果期6-8月。
# 广布野豌豆
别名苕草，花期3-6月，果期4-7月。多年生蔓生草本。茎攀援、有棱，被柔毛。偶数羽状复叶，叶轴顶端有分叉的卷须，小叶互生。总状花序，花多数，密集一面向着生于花序轴上部；花冠紫色至紫红色。荚果长圆形有4-6粒种子。
# 泽漆
别名五朵云。一年生草本，花期3-5月，果期4-6月。根纤细，茎直立，单一或自基部多分枝，分枝斜展向上，光滑无毛。叶互生，倒卵形或匙形，先端具牙齿。总苞叶5枚，倒卵状长圆形，先端具牙齿。花序单生，有柄或近无柄，雄花数枚，雌花1枚。蒴果三棱状阔圆形，光滑无毛。种子卵状，暗褐色。
# 斑种草
一年生草本，高15-30厘米。茎细弱，丛生，直立或平卧，多分枝，被糙伏毛。叶椭圆形或狭椭圆形，上下两面被向上贴伏的糙伏毛或短硬毛。花序柔弱，细长，长10-20厘米；花冠蓝色或淡蓝色。小坚果肾形。
# 黄鹌an菜
一年生草本，花期3-8月，果期5-10月，高10-100厘米。茎直立，顶端伞房花序状分枝或下部有长分枝，下部被稀疏的皱波状长或短毛。基生叶全形倒披针形、椭圆形、长椭圆形或宽线行，大头羽状深裂，边缘有锯齿或几全缘。头花花序，舌状小花黄色，花冠管外面有短柔毛。瘦果纺锤形，压扁，褐色或红褐色。
# 刺儿菜
别名小蓟ji，多年生草本，花期4-7月，果期5-9月。茎直立，高30-80厘米。基生叶和中部茎叶椭圆形、长椭圆形或椭圆状倒披针形，叶缘有细密的针刺，针刺紧贴叶缘。
头状花序单生茎端，或植株含少数或多数头状花序在茎枝顶端排成伞房花序。小花紫红色或白色。瘦果淡黄色，椭圆形或偏斜椭圆，冠毛刚毛长羽毛状。
# 泥胡菜
一年生草本，高30-100厘米。花期3-8月，果期6-9月。茎单生，很少簇生，上部长分枝，少有不分枝的。基生叶长椭圆形或倒披针形，花期通常枯萎，中下部茎叶与基生叶同形，全部叶大头羽状深裂或几全裂。头状花序在茎枝顶端排成疏松伞房花序，少有植株仅含一个头状花序而单生茎顶的。小花紫色或红色。
# 车前
二年生草本，花期4-8月，果期6-9月。叶基生呈莲座状；叶片纸质，宽卵形至宽椭圆形，边缘波状、全缘，两面疏生短柔毛。花序3-10个，直立或弓曲上升；花序梗有纵条纹，疏生白色短柔毛；穗状花序细圆柱状，花冠白色。
# 北美车前
别名毛车前，一年生或二年生草本，花期4-5月，果期5-6月。叶基生呈莲座状，平卧至直立；叶片倒披针形至倒卵状披针形，两面及叶柄散生白色柔毛。花序1至多数；花序梗直立或弓曲上升，有纵条纹，密被开展的白色柔毛；穗状花序细圆柱状，下部常间断；花冠淡黄色，花两型。蒴果卵球形。
# 楝
别名苦楝，花期4-5月，果期10-12月。落叶乔木，高10余米。叶为2-3回奇数羽状复叶，小叶对生，卵形、椭圆形至披针形，顶生一片通常略大。圆锥花序约与叶等长，无毛或幼时被鳞片状短柔毛；花芳香，花瓣淡紫色，匙形。核果球形至椭圆形，种子椭圆形，微毒。
# 海桐
合肥常见园林植物，花期3-5月，果熟期9-10月。常绿灌木叶聚生于枝顶，革质，倒卵形，发亮。伞形花顶生，花白色，有芳香，后变黄色；雄蕊2型，蒴果圆球形，呈三角形。
# 野老鹳草
一年生草本，花期4-7月，果期5-9月。高20-60厘米；基生叶早枯，茎生叶互生或最上部对生；叶片圆肾形，长2-3厘米，宽4-6厘米。花序腋生和顶生，长于叶，每总花梗具2花，顶生总花梗常数个集生，花序呈伞形状；花瓣淡紫红色，倒卵形。成熟蒴果易爆裂，将种子弹出。
# 忍冬
**忍冬别名金银花**，半常绿藤本，花期4-7月，果期8-11月；幼枝红褐色，密被黄褐色糙毛、腺毛、短柔毛。叶纸质，卵形。花冠白色，后变黄色，果实圆形，熟时蓝黑色。
# 刺果毛茛gen
一年生草本，花果期4-6月。须根扭转伸长。茎高10-30厘米，自基部多分枝。基生叶和茎生叶均有长柄；叶片近圆形，长及宽为2-5厘米，3中裂至3深裂，边缘有缺刻状浅裂或粗齿，通常无毛；花瓣5，狭倒卵形，长5-10毫米，瘦果扁平，多枚刺。
# 毛茛
多年生草本，花期4-8月，果期6-9月。茎直立，高30-70厘米，中空，有槽，生开展或贴伏的柔毛。基生叶多数；叶片圆心形或五角形，长及宽为3-10厘米，3浅裂，边缘有粗齿或缺刻，两面贴生柔毛；聚伞花序有多数花；花直径1.5-2.2厘米；花瓣5，倒卵状圆形。聚合果近球形。
# 桑
乔木或为灌木，高3-10米或更高。叶卵形或广卵形，表面鲜绿色。桑树的栽培历史悠久，桑叶可喂蚕。
穗状花序单性，雌雄异序，花与叶同时生出，淡绿色。聚花果卵状椭圆形，是一种甜美多汁的水果，一般称作“桑葚”或“桑椹”，成熟时红色或暗紫色。花期4-5月，果期5-8月。
# 石榴
落叶灌木或乔木，高通常3-5米。叶通常对生，纸质，上面光亮；叶柄短。花大，1-5朵生枝顶；花瓣通常大，红色、黄色或白色。浆果近球形，通常为淡黄褐色或淡黄绿色，有时白色，稀暗紫色。种子多数，钝角形，红色至乳白色，肉质的外种皮供食用。
# 杜鹃
又名映山红、山石榴。相传，古有杜鹃鸟，日夜哀鸣而咯血，染红遍山的花朵，因而得名。落叶灌木，高2(-5)米；分枝多而纤细，密被亮棕褐色扁平糙伏毛。叶革质，常集生枝端。
一般春季开花，每簇花2-6朵，花冠漏斗形，有红、淡红、杏红、雪青、白色等，花色繁茂艳丽。
# 粉红绣线菊
又称日本绣线菊，花期6-7月，果期8-9月。直立灌木，枝条细长。叶片卵形至卵状椭圆形。复伞房花序生于当年新枝顶端，花朵密集，密被短柔毛，花瓣卵形，粉红色；雄蕊远长于花瓣。合肥各个公园、小区常见栽培植物。
# 夹竹桃
常绿直立大灌木，全株有毒。高达5米，枝条灰绿色，含水液。叶3-4枚轮生，叶面深绿，叶背浅绿色。
聚伞花序顶生，着花数朵；花芳香；花冠深红色或粉红色，栽培演变有白色或黄色。花期几乎全年，夏秋为最盛；果期一般在冬春季，栽培很少结果。
# 广玉兰
又称荷花玉兰，常绿乔木，合肥市树。花期5-6月，果期9-10月。叶厚革质，椭圆形，叶面深绿色，有光泽。
花白色，有芳香；花被片9-12，厚肉质，倒卵形，长6-10厘米，宽5-7厘米。聚合果圆柱状长圆形或卵圆形，密被褐色或淡灰黄色绒毛；种子近卵圆形，外种皮红色。
# 窃衣
一年生草本，植株像小芹菜。花期4-8月，果期6-11月。叶片长卵形，羽状分裂，复伞形花序顶生，花瓣白色至蓝紫色，果实圆卵形，有内弯或呈钩状的皮刺，易粘在衣物上。
窃衣这名字好听，不是牛郎织女那个窃衣。窃是附着、粘上，是被窃的意思。窃衣就是粘着衣服不能解，一不小心被衣给窃走了。
# 打碗花
小时候有一种花是不敢摘的，因为大人们告诫说摘了这种花，吃饭的时候会打碎碗，是什么花这么神奇呢？
打碗花是一年生草本，花期5-9月，果期7-10月。全体不被毛，植株通常矮小。基部叶片长圆形，基部戟形；花腋生，1朵；花冠淡紫色钟状。蒴果卵球形，种子黑褐色。
# 金丝桃
灌木，高0.5-1.3米。花期5-8月，果期8-9月。茎红色，叶对生，叶片倒披针形或椭圆形至长圆形，上面绿色，下面淡绿但不呈灰白色。
花瓣金黄色至柠檬黄色，花朵五瓣，雄蕊五簇，每簇都有20-30根不等，上百根雄蕊围绕着雌蕊，如众星捧月。蕊似金丝形似桃，是金丝桃名字的由来。种子深红褐色，十分可爱动人。

# 合肥外来入侵水生物——合肥常见水生外来入侵物种
https://mp.weixin.qq.com/s?__biz=MzU4MTU0Mjk2NQ==&mid=2247547262&idx=1&sn=2eeae4c9a373d833218baf4509cbf47f&scene=21#wechat_redirect

阳春三月，春暖花开，正是外出踏青游玩的好时机。清明节假期，各个公园池塘边来了很多左手持网，右手拎桶的大朋友小朋友们，想要探究一下池塘里的神奇生物。这确实是探索自然的一种方式，但具体操作时，却反而有可能给我们喜爱的这些小生物带来伤害，破坏了给我们带来欢乐的大自然。

之前，大羊的一篇《给喜欢摸鱼捉虾捞蝌蚪的大小朋友们的几条建议》引来了很多的朋友的转发关注，文中不建议我们随意将自然界中的动物带回家，但同时也提到了一些入侵物种特例。
今天，我们针对合肥池塘里常见的几种入侵物种做更详细的介绍，如果你们捞到了以下种类的小动物，那么建议你带走它们，不要再留在自然界中。

何为外来物种？何为入侵物种？外来物种是指在一个物种出现在了它的自然分布范围以外的情况，比如在自然状态下生活在美洲的物种来到了中国繁衍生息。当外来物种能够在当地生态系统中适应生存并不会给当地生态平衡造成冲击和负面影响时，外来物种其实可以成为本地生态系统中正常的一部分。实际上，所谓的“外来”，是自然界中自然而然且不可避免的一个现象。

但其中有些外来物种生存能力、繁殖能力都很强、在当地无天敌，这就可能严重影响本土物种的繁衍（由于竞争或捕食等），给当地的生物多样性造成很大的负面影响，那么这些物种就被称之为入侵物种。而且这些入侵物种，大部分是由于人类的行为造成的。

合肥常见水生外来入侵物种
一、克氏原螯虾（节肢动物门原鳌虾属）

这应该是我们最熟悉的外来入侵物种了，克氏原螯虾，也就是我们都熟悉的小龙虾。不少人的童年有它相伴的快乐回忆，也是夏季餐桌上的饕餮美味，很多人都以为这是本土养殖产品，并不知道它原来是外来入侵物种。它们原产于墨西哥和美国，来到中国后可谓是遍地开花，它们喜欢吃腐烂的动植物尸体，生存适应性强，给当地生态带来了很大威胁。同时它还擅长打洞，农田的田埂被破坏了不说，还会危及堤坝的稳固。

二、福寿螺（腹足纲瓶螺科）

福寿螺原产南美，最初是作为餐桌上的美食引入中国的，但是因为它的口感不佳被抛弃到野外，然后它凭借自己食性杂、繁殖快等特点很快就泛滥成灾。它们一般夜晚活动，喜欢啃食植物的茎叶，在稻田和池塘里都能很好地适应生存。

我们经常可以在池塘边的水草上能见到福寿螺的卵，粉粉嫩嫩的还有些可爱。但如果遇到了，建议尽量将其敲落至水中，福寿螺的卵在水内无法成活。

值得注意的是，福寿螺与合肥本土的田螺很像，常常会被混淆，福寿螺的体型比一般田螺大，螺身的尖口较平。我们可以通过下图来分辨。

福寿螺是广州管圆线虫、卷棘口吸虫等寄生虫的中间宿主，所以如果想要食用它，一定记得要彻底煮熟。在水族市场上它还有一个名称叫苹果螺，在人工培育下，小巧可爱，价格便宜，比较受顾客欢迎，快去看看你家的鱼缸里有没有它吧。

三、红耳龟

红耳龟原产地是密西西比区域，它们头部类似耳朵的位置有两块红斑，这也是它们名字的由来。龟在中国传统文化中是吉祥长寿的象征，这也成了很多人进行放生活动的首选，而在市场上便宜、好养的红耳龟自然备受青睐。

但是，它之所以能够在中国野外蔓延，主要是宠物弃养和野外放生。红耳龟食性杂且凶猛、繁殖能力和适应力也很强。放生后的红耳龟不仅可能捕食本土生物，还挤占了本土龟类的生存空间，给当地生态环境造成严重危害。

如果你经常去池塘边，那你一定见过正在翘着头晒太阳的红耳龟，因为它在合肥的池塘中真的是太普遍了，而这事想想就可怕，它们的存在给原本池塘里生活的鱼、虾、蛙类甚至鸟类都造成了直接威胁。

四、鳄龟

鳄龟来自美国南部，从它的名字就可以知道其战斗力，它粗壮的身体、强大的咬合力使其一旦来到野外自然环境，几乎所向披靡。鳄龟的成年个体背甲可达50-70厘米，巨大的进食量和快速的生长速度也使得它可以很快称霸一片池塘。

鳄龟引进主要是作为宠物市场的观赏用途，但是人们饲养一段时间就会发现除了需要提供它惊人的进食量之外，养它的玻璃缸也需要特别加固，最后很多人选择弃养放生，这也直接导致了它对本地生态环境的破坏。

五、牛蛙
同样是来自美国的一种凶猛捕食者，牛蛙的体型比我们常见的青蛙（黑斑侧褶蛙）要大得多，秉承着“大嘴吃四方”的信仰，在池塘中横行霸道，鱼呀、蛇呀、甚至是自己的同类，只要它的嘴巴能吃得下，它都会去捕食。它的引入当然主要是为了满足人们的口欲，食用之前也一定要彻底煮熟。

外来物种不仅在其自身的生存上会给周围生态环境造成影响，它们身上可能携带的病原微生物也同样对其他生物的生存甚至对人类的健康构成威胁。所以如果在野外捞到了这些物种，请尽量带走，不要再放生回去。

这些入侵物种确实给本土生态带来严重负面影响，但我们也无需对物种本身多么憎恨，通过以上物种的了解，我们会发现，这些物种之所以会在中国的野外生存繁殖，大部分是由于人类的贪婪和不负责任，甚至是“爱心”。

其实，不论是外来物种还是入侵物种，都是一个相对的概念，当我们追溯物种的历史分布就会发现，很多本土的物种，都曾在某个历史阶段是“外来”的，所以“外来”本身是个中性词，这是生态系统演变、生物多样性维系自然而然的现象。

当一个物种融入本地的生态平衡之中，外来自然也就成了本土，但当一个物种打破本地的生态平衡，而且愈演愈烈，它们就形成了入侵。融入本土还是成为入侵，生物本身并不会主观选择，但有一种生物不同，那就是人类。

从人类的历史来看，人类足迹的扩张，也是我们不断地成为某个地方“外来物种”的过程。但不同的是，人类有智慧，人类可以做主观选择，我们可以主观选择自己的行为，让我们成为一个促进当地生物多样性蓬勃发展的“本土物种”，还是打破当地生物多样性的“入侵物种”。

至少如何对待身边的小池塘，就可以成为我们的选择。
