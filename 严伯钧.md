# ‘不动’的水流 不会动的水流，不动的其实是速度分布，而不是水
最近一条阿尔卑斯山静止水流的视频火了，这是为啥呢？时间静止了吗？这其实**在流体力学里面叫层流**，什么是层流呢？当流体的速度不是很快时，就处在层流状态。我们**研究流体，研究的不是具体每个流体分子的运动，而是对目标流体放上一个时空坐标，然后研究流体在每个位置的流速，也就是每个位置的流速才是我们关心的**。

你写**流体方程也是流速作为时空坐标的函数**。而**当目标流体的速度比较慢，就会是层流**。层流中，不同位置的流速基本不随时间变化，也就是每个点的流速基本上恒定。这样的情况下，流体中每个点流过的流体分子在流过以后，后面的分子流过来也是完全一样的速度。这样的话，流体看上去就静止不动了。

因为你平常看到的会流动的水，每个点的流体速度都是随时间改变的速度有变化，才会有水流动的感觉。**与层流相对的叫湍流**，湍流就是流体的一种杂乱无章的运动，既然都杂乱无章了，流体的每个点的流速肯定会有变化，且比较随机。我们大部分情况下看到的水流都伴随湍流现象，这就导致我们习惯于看到会动的水。

其实呢，你把水龙头关到特别小的时候，仔细看看，能看出来水在流动吗？其实你也看不出来，就是因为水龙头关小了，流速慢了就形成了层流。其实层流的实验也很好做。你弄个大气球里面装满水，用胶带贴出一个小方框，然后呢刺破小方框，出来的水就是层流。记住啊，小方框的位置最好要靠气球上方，如果开在下方，水压太大出来的水流速度会比较快。😊水流速度快了，那就会有湍流，就没有静止不动的层流的感觉了。

# 自然数真的自然吗？ 自从0变成了自然数，一堆奥数题目就要重出了
自然数为啥叫自然数natural number，是因为它自然原生态吗？哎，还真是。首先自然数啊其实就是零以及所有正整数，并且啊在2000年之前啊，自然数不包含零，我还记得上小学时，有一天老师宣布一件事，说是从现在开始啊，零算是自然数了。这下就happy了。我还记得有很多小学奥数题啊，说已知N是一个自然数，然后如何如何。但如果自然数包含零的话呢，这些题就变得非常的好做。所以呢我印象特别深刻。

但是为何自然数叫自然数呢？这里的原因是**只有自然数是自然中天然存在的数字，这里的天然存在怎么理解啊？哎，应该理解为是直观存在**。远古人啊其实就会数数了，会用结在绳子上的结来数数，因为一个苹果，两个苹果，三个苹果，对吧，这些数是天然能够直观感受到的。但是呢自然数以外的数字其实都是人造数，都来源于人的规定，并非一种直观。

我们一个个看啊，这自然数以外是什么呢？首先是负数，对吧？-1，-2，-3，那有人觉得这也是直观的呀，我欠你1块钱2块钱的，不就可以写成-1，-2吗？不对，这个里面直观的还是欠的1块2块，这是直观的。但是欠这个动作用负号来表示是人为规定的，只不过是用负号代替了减法这个操作而已。再往下呢那就是小数了，1块5，5块1块，可以写成1.5元和5.1元，这难道不是直观的吗？

哎，其实也不是，这只是人为规定了进制，因为1点5元，其实就是15个1毛，5.1元，就是51个1毛，直观的还是自然数。那有人又要说了，1.5个苹果不就是一个半苹果吗？哎，不也是直观吗？问题来了，请问你一刀切下去，你感觉你切了半个苹果，但是你能保证你切下的两个半苹果，真的是精确的体积相等吗？这不能保证吧

也就是半个苹果2分之1啊，这个概念是只存在于你的脑海当中的。你有半个苹果就是体积精确等于原来一半的苹果这个概念，所以呢这样看啊，小树这个概念也是人类思想当中的一个抽象概念，它并不天然的存在，并非直观。再往下有理数，也就是可以写成两个互质数M比N形式，例如3分之2，那天然存在吗？它也不天然啊，理由跟上面一样，3分之2等于0.6666666无限循环。

我给你一块蛋糕，你给我切出3分之2块出来，你能够保证这3分之2精确等于原来蛋糕面积的3分之2吗？就算你用再精确的激光去进行切割也一定有误差，不存在精确的3分之2。再往下无理数，比方圆周率π，π等于圆的周长比上它的直径，但是π天然存在吗？也很难说，因为现实中根本没法画出一个精确无比的圆，一定有误差。所以π也是存在于人的理念世界当中的概念。

再往下虚数，根号下-1。这个从名字上就可以看出来，虚的很，对吧？imaginary number，这虚数没有物理意义，没有任何一个物理量对应于一个虚数。再往下什么拉格朗日4元数，它就更加没有现实对应物了。所以你看盘了一圈啊，只有自然数直观存在。其他的数都是人造的，都是数学概念都是抽象、理想世界才存在的概念

所以啊我们说**数学不是科学，就是因为数学研究的，其实是人的理念世界**。所以现在知道为啥自然数叫自然数了吧，就是因为只有自然数是自然界真实存在的。但如果其实再钻研的深一点啊，你也说不好自然数是否自然存在。因为一个苹果两个苹果，这是因为人的大脑认知了苹果这个东西，但对于自然来说，它care这是个苹果嘛？它大概不care，无非就是一坨特定种类的原子聚在一块而已。也就是你去分一个两个，个的概念也是来自于人的认知。

那一个电子两个电子这种基本粒子总是自然存在的吧，哈哈也不一定，两个电子放在一起一模一样，我把两个电子交换一下位置，你根本无法判断我有没有交换它们的位置。既然电子之间无法区分彼此，再去分一个两个，还有啥意义吗？并且一个电子它的大小，它的边界在什么地方呢？如果看一个电子，我们可以粗略的用它的库仑力的影响范围来设定它的边界。

# π包含一切？就目前看来，π应该可以包含一切，但这没有被证明
之前我有一条视频，说是用π弹出来的音乐非常好听是骗人的，不光π好听，所有无理数弹出来的音乐都可以好听。关键在和声以及节奏，但是呢那条视频里面我有一个点说的不够严谨，讲太快了。我发现很多粉丝已经犀利的指出了这一点，这个我得给你们点个赞，无限不循环小数并不一定包含所有可能的信息。而我说π可以包含所有一切数字的排列组合，所以原则上所有人类写过的音乐甚至一切文学作品，一切信息都可以在π里面找到

就目前看来呢，这个结论很有可能是对的，但是还没被完全证明，**能够包含所有信息的数叫合取数disjunctive**。比方说啊我们可以人造一个数字，就是0.1234567891011121314516……，以此类推，把所有数字都排在后面，一直排下去，这明显无限不循环吧，也明显所有可能的数字组合排列，都可以在里面找到吧，这就是一个合取数

但π是不是合取数没有被证明，但就目前已经算出来π的位数看来，π有极大可能是合取数，为啥呢？合取数必然满足的一个标准，应该是合取数里面0到9这10个数字出现的概率完全一样，是平均分布的。这是为何呢？哎，我们换成二进制就比较好理解了。我们现在所有互联网上表达的信息都是二进制的，只有0和1。

如果要表达所有信息则0和1的数量必然相等。因为0和1只是两个符号，你也可以人为规定，1代表0,0代表1，根据对称性，0跟1没啥不同。所以如果表达所有信息，0跟1的数量必然相等。那0到9这10个数字其实是人为规定出来的十进制，10个数字在表达信息的问题上，各自应该是平等的，没啥不同。所以要表达所有信息，0到9这10个数字在合取数里面出现的概率就应该相同。

但是这里我还是跳了步，是不是合取数，其实在很多情况下跟进制也有关。有一些数字用二进制合取数，用十进制就不一定是合取数。有一些用二进制合取数，用六进制就不是合取数等等。但是呢get精神就好。而根据目前对于π的计算来说，看到的0到9这10个数字的分布非常平均，而π又是无限不循环的，所以如果π满足无限不循环，又满足每个数字出现的概率一样，那么我们就可以说这是一个合取数。

只不过我们现在并没有用演绎法去证明π里的每个数字出现的概率一样，只是通过归纳法发现现有的已经算出来的π，里面的数字出现概率相同。因此推断π应该是一个合取数。这样的话，确实就能在π里面找到所有可能的信息。

# 高考出分：去哪里？在你不确定读啥专业的情况下：城市＞学校＞专业
。高考成绩已经陆续出来了啊，祝愿大家都能考出自己想要的分数，虽然我知道这是不可能的啊，但既然高考出分了，就要面临报志愿的问题。今天我就来讲讲应该怎么选学校和专业啊。我去学习过的大学还是挺多的。我高中搞奥赛的时候，就在中科大、吉大、武大、华南师大、湖南师大参加过培训，入选国家集训队之后呢，又在南京大学培训过3个月。

上大学前因为保送的是清华，所以也在清华听过课。本科在港科大，本科期间去巴黎大学和休斯顿的Rice大学交换过，博士阶段在布朗大学。所以呢我的大学样本量还是比较大的。我有一个体会啊，就是当你对于想上的专业不笃定时，应该怎么选学校呢？除非你非常明确，比方你就想读医就想读法，就想当学者搞科研，否则我觉得优先级应该是城市大于学校大于专业。

为何这么说呢？因为企业招聘啊，很多时候有就近原则。一个企业去影响它所在城市的高校，所谓近水楼台先得月啊也更方便。所以城市的经济水平是一个重要的指标，一个城市的经济好，就代表就业机会多。我高中在深圳中学，当时呢班上有这样的情况，上不了清华北大就去深大，深中的小孩还亲切称呼深大为高四。为啥呢？因为深圳经济发达，就业机会多，去一个三线城市的211在深圳找工作，可能还没有深大方便。

并且要知道，大学四年也是不短的时间啊，并非所有人都要专业搞学术研究的。所以在大学期间能接触发达的城市，生活也会更加丰富多彩，也更加有机会。举个美国的例子啊，达特茅斯学院是长春藤名校，但是所在城市Hanover汉诺威，那叫一个荒啊，大学四年除了hand over宿醉，啥也干不了啊，那还不如去纽约大学。所以城市第一，其次就是学校了，公司招人时，尤其是商务类岗位，其实不一定会太看重你的专业。

我招一个客户经理，我看中的是表达能力，综合素质，跟你学的啥专业其实没必然联系啊。你看我一个学物里的，我这表达能力当个金牌销售没啥问题吧。那么公司在招聘初步筛选时，毕业学校肯定是一个好的背书。比方江南大学跟东南大学那都是211的好学校，江南大学的食品科学专业是全国第一，但是名气上呢总感觉东南大学还要响亮些，在应聘的时候呢，感觉也要占一些优势。

那么城市跟学校学好了就是专业了，这就复杂了。这里我只能代表物理专业来劝大家报考一下，那为何要学物理专业呢？我觉得就一个原因，就是读物理专业的延展性强，就算以后不做物理研究，那可以干的事太多了。但是非物理专业的以后要从事物理工作基本就很难了。学物理也可以去当程序员啊，因为现在的物理专业不编程基本很难做研究了。

学物理的也可以去做金融，金融公司做模型的就需要强大的建模能力；学物理的可以去学数学，学物理的可以去当老师，学物理的可以去搞电子工程，学物理的可以去搞企业管理。学物理可以去做土木工程，学物理的可以去读法学。总之，本科学物理读研时，基本所有理工科专业都可以读。所以学物理不仅锻炼思维能力，未来的路也很宽，来读物理系吧。😊
