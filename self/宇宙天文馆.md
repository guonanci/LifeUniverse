# 残忍现实，三十年身高持续倒退，日本人为何越来越矮了？

全中国人都在长高，怎么日本人却越来越矮了？说个残忍的真相，日本19岁男性的平均身高，比我国整整矮了3厘米。170.9就是他们的极限身高了。更悲催的是，日本专家预测，未来日本人的身高还会持续缩水。

所以日本小年轻干脆不追求长高，转头对女性身高提出严苛要求。150到155是樱花妹里的香饽饽，超过160就迈入丑女行列了。至于170更不用想，这是被全社会淘汰的女性身高。

但不管日本怎么自我说服，都改变不了全民变矮的趋势。实际上，在漫长的历史长河中，矮才是日本的常态。江户时代的日本男性，平均身高只有148到151厘米。日本‘参天巨人’织田信长，身高也不过一米六九。

因为日本有1200年的素食史，只有穷人才要顿顿吃肉。但日本也有一段快速长高期，在1896到1996年间，日本靠着恶补营养，以及跟美军结合改善基因，成功让身高拔高到170厘米。

结果日本专家半路开香槟，让日本孕妇保持整个孕程低体重，孕妇孕期体重不得超过5公斤。导致出生的孩子大多低于2500克，而且日本的卡路里摄入远低于韩国，先天不足营养又跟不上，达不到理想标准也实属正常。

如果日本继续坚持白幼瘦审美，恐怕又要倒退回倭国时代了。

# 美基因VS丑基因，哪个更强大？如何才能生个好看的孩子？

为何丑基因更容易遗传？大鼻子、肿眼泡、凸嘴，这些相对不符合审美的特征，为何没有被优胜劣汰掉？说一个扎心的事实，大部分孩子都是挑着父母颜值缺点长的。前英国王夫菲利普亲王堪称神颜，唯一的缺点是秃顶。英国女王伊丽莎白二世有点凸嘴，但五官长得明艳大气。明明爹妈都是帅哥美女，但秃顶、凸嘴的基因一结合，后代的颜值就大打折扣。

更可怕的是，这两个基因还会延续很久，就比如令妃和溥（pu三）仪的对比图。凸嘴基因一进入家族血脉，几乎就是代代相传了。对人类来说，大鼻头也属于不容商量的显性遗传，就比如成龙大哥和他的两个儿子，不管两位妈妈长得有多好看，后代都逃不过大鼻子的命运。难道丑基因比美基因更强大？

其实基因不分美丑，只看谁更有利于生存和繁衍，有的显性基因看上去比较夸张，实际上是生命力顽强的体现，但美是多元的凸嘴这些特征在脸上未必会丑，反而会成为极具辨识度的特点。而且生活条件和环境也很重要，很多人都追求白皮肤，但对于热带地区的人来说，新生浅色皮肤往往意味着疾病。

而对于温带和寒带的人类，天生黑色素更少，如果要强行把皮肤晒成古铜色，又会带来紫外线不耐受的烦恼，这样一想，你现在的模样也许就是最佳状态。

# 脱光提臀晒太阳，日本流行的日光浴，究竟是啥？

脱掉裤子，抬起屁股，将两瓣屁股对准太阳，这就是让日本年轻人痴迷不已的日肛浴。把屁股当成向日葵，只需在太阳下暴晒30秒，就能让太阳的能量，从肛门输送到全身经脉，让人精神饱满充满干劲。

发明日肛浴的是美国人，但流传到日本后就被玩出了各种花样。比如专门到山顶晒屁股，或者在阳光灿烂时，约上三五个好朋友一起脱掉裤子晒太阳。让太阳晒屁股变得具象化，但屁股平时和阳光可不熟悉，这么毫无防备的接受暴晒，晒伤几率蹭蹭蹭往上涨。

有的勇士就因为太过沉迷，最后受伤的是倒霉的屁股。不过话说回来，平时适当晒晒太阳有很多好处，但一定要注意日晒时长。如果长时间暴露在紫外线下，也会给皮肤带来很多烦恼。

# 专门针对男同的新型骗局来了，什么是‘0号胶囊’？男生更要警惕

专门针对男同的新型骗局来了，被基友圈滥用的‘0号胶囊’到底有多可怕？当一个0被塞入一颗0号胶囊，不到1分钟就会变得神志不清，失去判断和自控力，任由摆布。有的人甚至把它当成‘幸福催化剂’。

但0号胶囊的真实身份，是苯丙胺的衍生物——二甲基色胺，危害性远比冰毒更强。如果一个男人说它无害，这个男人一定是渣男无疑，因为0号胶囊本质上，就是种活性精神物质。市面上流通的0号胶囊，大多是化学合成，纯度更高、更直接。

让人忽视安全措施，为艾滋病传播提供了土壤。而为了追求极致快乐，男同们往往会超剂量使用，6mg就能产生幻觉和幻听，30mg基本没有回头路了。上一条是天堂，下一秒可能是死亡。

除了0号胶囊外，一种气体版的Rush Poppers也是男同群体常用的物质。大家可能很熟悉它的成分，就是食品问题中常听到的亚硝酸盐。亚硝酸盐变成一种气体，能在无形中，让人放松全身的平滑肌，心率增加、血压上升等，被当成缓解痛楚的工具。

但它的影响积少成多，怀着侥幸心理一次次尝试，最终可能会付出生命代价。

# 人脑有多精密？人类开发不到10%，全部开发会怎样？

这是一颗完整的大脑标本，它的重量仅占全身2%，却储存了66亿TB数据，究竟是咋做到的？人脑中有860亿个神经元，每个神经元有多达10000个突触，形成了庞大的互联网络。思想每小时游走300多里，平均每24小时产生4千种思想，是世界上最精密最灵敏的器官，蕴藏着无数待开发的资源。

而人类对大脑的开发只有10%，如果大脑潜能全部开发，又会发生啥呢？其实有人说过，爱因斯坦那么聪明，都是因为大脑比普通人多开发了，所以死后有人偷偷切下他的大脑。但研究发现他的大脑与常人无异，只不过某些区域比普通人更活跃，如果大脑全部开发，或许人类会拥有‘超能力’。

但那是永远不可能的事，因为大脑的血氧承受能力有限，无法同时支撑处理很多数据，用脑过度时总感觉很疲惫，就是这个道理。当然，保持终身学习、勤奋思考，会让大脑变得更灵活。
